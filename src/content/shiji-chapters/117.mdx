<h2>史記卷一百一十七</h2><h2>司馬相如列傳第五十七</h2><p>　　司馬相如者，蜀郡成都人也，字長卿。少時好讀書，學擊劍<span class="inlay1"><span class="number">一</span></span>，故其親名之曰犬子。<span class="inlay1"><span class="number">二</span></span>相如旣學，<span class="inlay1"><span class="number">三</span></span>慕藺相如之為人，更名相如。以貲為郎，事孝景帝，為武騎常侍，<span class="inlay1"><span class="number">四</span></span>非其好也。會景帝不好辭賦，是時梁孝王來朝，從游說之士齊人鄒陽、淮陰枚乘、吳莊忌夫子<span class="inlay1"><span class="number">五</span></span>之徒，相如見而說之，因病免，客游梁。梁孝王令與諸生同舍，相如得與諸生游士居數歲，乃著《子虛之賦》。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：《呂氏春秋》劍伎云「持短入長，倏忽縱橫之術也」。魏文典論云「余好擊劍，善以短乘長」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：孟康云：「愛而字之也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：案：秦密云「文翁遣相如受七經」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：張揖曰：「秩六百石，常侍從格猛獸。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「名忌，字夫子。」　《索隱》：徐廣、郭璞皆云名忌字夫子。案：鄒陽傳云枚先生、嚴夫子，此則夫子是美稱，時人以為號。《漢書》作「嚴忌」者，案忌本性莊，避明帝諱改姓嚴也。</span></p><p>　　會梁孝王卒，相如歸，而家貧，無以自業。素與臨邛令王吉相善，吉曰：「長卿久宦遊不遂，而來過我。」於是相如往，舍都亭<span class="inlay1"><span class="number">一</span></span>。臨邛令繆為恭敬，日往朝相如。相如初尚見之，後稱病，使從者謝吉，吉愈益謹肅。臨邛中多富人，而卓王孫家僮八百人，程鄭亦數百人，二人乃相謂曰：「令有貴客，為具召之。」幷召令。令旣至，卓氏客以百數。至日中，謁司馬長卿，長卿謝病不能往，臨邛令不敢嘗食，自往迎相如。相如不得已，彊往，一坐盡傾。酒酣，臨邛令前奏琴曰：「竊聞長卿好之，願以自娛。」相如辭謝，為鼓一再行<span class="inlay1"><span class="number">二</span></span>。是時卓王孫有女文君新寡，好音，故相如繆與令相重，而以琴心挑之。<span class="inlay1"><span class="number">三</span></span>相如之臨邛，從車騎，雍容閒雅甚都；<span class="inlay1"><span class="number">四</span></span>及飲卓氏，弄琴，文君竊從戶窺之，心悅而好之，恐不得當也。旣罷，相如乃使人重賜文君侍者通殷勤。文君夜亡奔相如，<span class="inlay1"><span class="number">五</span></span>相如乃與馳歸成都。家居徒四壁立。<span class="inlay1"><span class="number">六</span></span>卓王孫大怒曰：「女至不材，我不忍殺，不分一錢也。」人或謂王孫，王孫終不聽。文君久之不樂，曰：「長卿第俱如臨邛，<span class="inlay1"><span class="number">七</span></span>從昆弟假貸猶足為生，何至自苦如此！」相如與俱之臨邛，盡賣其車騎，買一酒舍酤酒，而令文君當鑪。<span class="inlay1"><span class="number">八</span></span>相如身自著犢鼻褌，<span class="inlay1"><span class="number">九</span></span>與保庸雜作，<span class="inlay1"><span class="number">一〇</span></span>滌器於市中。<span class="inlay1"><span class="number">一一</span></span>卓王孫聞而恥之，為杜門不出。昆弟諸公<span class="inlay1"><span class="number">一二</span></span>更謂王孫曰：「有一男兩女，所不足者非財也。今文君已失身於司馬長卿，長卿故倦游，<span class="inlay1"><span class="number">一三</span></span>雖貧，其人材足依也，且又令客，獨柰何相辱如此！」卓王孫不得已，分予文君僮百人，錢百萬，及其嫁時衣被財物。文君乃與相如歸成都，買田宅，為富人。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：臨邛郭下之亭也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：案：樂府長歌行、短歌行，行者曲也。此言「鼓一再行」，謂一兩曲。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：郭璞曰：「以琴中音挑動之。」　《索隱》：張揖云：「挑，嬈也。以琴中嬈之。」挑音徒了反。嬈音奴了反。其《詩》曰：「鳳兮鳳兮歸故鄕，遊遨四海求其皇，有一豔女在此堂，室邇人遐毒我腸，何由交接為鴛鴦」也。又曰「鳳兮鳳兮從皇栖，得託子尾永為妃。交情通體必和諧，中夜相從別有誰」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：韋昭曰：「閒，讀曰「閑』，甚得都邑之容也。」郭璞曰：「都猶姣也。詩曰『恂美且都』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：郭璞云：「婚不以禮為亡也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：郭璞曰：「言貧窮也。」　《索隱》：案：孔文祥云「徒，空也。家空無資儲，但有四壁而已，云就此中以安立也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：弟如臨邛。文穎云：「弟，且也。」郭璞云：「弟，語辭。如，往也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：韋昭曰：「鑪，酒肆也。以土為墮，邊高似鑪。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《集解》：韋昭曰：「今三尺布作形如犢鼻矣。稱此者，言其無恥也。今銅印言犢紐，此其類矣。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：《方言》曰：「保庸謂之甬，奴婢賤稱也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《集解》：韋昭曰：「瓦器也。每食必滌溉者。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《集解》：郭璞曰：「諸公，父行也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《集解》：郭璞曰：「厭游宦也。」</span></p><p>　　居久之，蜀人楊得意為狗監，<span class="inlay1"><span class="number">一</span></span>侍上。上讀子虛賦而善之，曰：「朕獨不得與此人同時哉！」得意曰：「臣邑人司馬相如自言為此賦。」上驚，乃召問相如。相如曰：「有是。然此乃諸侯之事，未足觀也。請為天子游獵賦，賦成奏之。」上許，令《尚書》給筆札。相如以「子虛」，虛言也，為楚稱；<span class="inlay1"><span class="number">二</span></span>「烏有先生」者，<span class="inlay1"><span class="number">三</span></span>烏有此事也，為齊難；<span class="inlay1"><span class="number">四</span></span>「無是公」者，無是人也，明天子之義<span class="inlay1"><span class="number">五</span></span>。故空藉<span class="inlay1"><span class="number">六</span></span>此三人為辭，以推天子諸侯之苑囿。其卒章歸之於節儉，因以風諫。奏之天子，天子大說。其辭曰：</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：郭璞曰：「主獵犬也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：郭璞曰：「稱說楚之美。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「烏，一作惡。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：郭璞曰：「詰難楚事也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：郭璞曰：「以為折中之談也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：音假借，與積同音。</span></p><p>　　　　楚使子虛使於齊，齊王悉發境內之士，備車騎之衆，與使者出田。田罷，子虛過詑<span class="inlay1"><span class="number">一</span></span>烏有先生，而無是公在焉。坐定，烏有先生問曰：「今日田樂乎？」子虛曰：「樂。」「獲多乎？」曰：「少。」「然則何樂？」曰：「僕樂齊王之欲夸僕以車騎之衆，而僕對以雲夢之事也。」曰：「可得聞乎？」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：郭璞曰：「詫，誇也。音託夏反。」　《索隱》：上音戈，下音敕亞反。誇詫是也。</span></p><p>　　　　子虛曰：「可。王駕車千乘，選徒萬騎，田於海濱。列卒滿澤，罘罔彌山，<span class="inlay1"><span class="number">一</span></span>揜兔轔鹿，射麋脚麟。騖於鹽浦，割鮮染輪。射中獲多，矜而自功。顧謂僕曰：『楚亦有平原廣澤游獵之地饒樂若此者乎？<span class="inlay1">孫守真按：「揜兔」下原《中華文化網》所載《瀚典》本掉一行。</span>楚王之獵何與寡人？』<span class="inlay1"><span class="number">四</span></span>僕下車對曰：『臣，楚國之鄙人也，幸得宿衞十有餘年，時從出游，游於後園，覽於有無，然猶未能遍睹也，又惡足以言其外澤者乎！』齊王曰：『雖然，略以子之所聞見而言之。』</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：郭璞曰：「罘，罝也。音浮。」　《正義》：《說文》云「罘，兔罟也」。今幡車罟也。彌，竟也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「轔音吝。」駰案：郭璞曰「脚，掎足。轔，車轢」。　《索隱》：脚麟，韋昭云「謂持其一脚也」。司馬彪曰「脚，掎也」。《說文》云「掎，偏引一脚也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：郭璞曰：「鹽浦，海邊地多鹽鹵。鮮，生肉也。染，擩也。音而沿反，又音而悅反。擩之於輪，鹽而食之。鶩，馳也。音務。」　《索隱》：李奇云：「鮮，生肉也。染，濡也。切生肉濡鹽而食之。」染或為「淬」，與下文「脟割輪淬」意同也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：郭璞曰：「與猶如也。」</span></p><p>　　　　「僕對曰：『唯唯。臣聞楚有七澤，嘗見其一，未睹其餘也。臣之所見，蓋特其小小者耳，<span class="inlay1"><span class="number">一</span></span>名曰雲夢。<span class="inlay1"><span class="number">二</span></span>雲夢者，方九百里，其中有山焉。其山則盤紆岪鬱，隆崇嵂崒；岑巖參差，日月蔽虧；<span class="inlay1"><span class="number">三</span></span>交錯糾紛，上干靑雲；罷池陂陀，下屬江河。其土則丹靑赭堊，<span class="inlay1"><span class="number">四</span></span>雌黃<span class="inlay1"><span class="number">五</span></span>白坿，<span class="inlay1"><span class="number">六</span></span>錫碧<span class="inlay1"><span class="number">七</span></span>金銀，衆色炫燿，照爛龍鱗。<span class="inlay1"><span class="number">八</span></span>其石則赤玉玫瑰，<span class="inlay1"><span class="number">九</span></span>琳誩琨珸，<span class="inlay1"><span class="number">一〇</span></span>瑊䘏玄厲，<span class="inlay1"><span class="number">一一</span></span>瓀石武夫。<span class="inlay1"><span class="number">一二</span></span>其東則有蕙圃<span class="inlay1"><span class="number">一三</span></span>衡蘭，芷若<span class="inlay1"><span class="number">一四</span></span>射干，<span class="inlay1"><span class="number">一五</span></span>穹窮<span class="inlay1"><span class="number">一六</span></span>昌蒲，江離麋蕪，諸蔗猼且。<span class="inlay1"><span class="number">一七</span></span>其南則有平原廣澤，登降陀靡，<span class="inlay1"><span class="number">一八</span></span>案衍壇曼<span class="inlay1"><span class="number">一九</span></span>，緣以大江，限以巫山。<span class="inlay1"><span class="number">二〇</span></span>其高燥則生葴艻苞荔，<span class="inlay1"><span class="number">二一</span></span>薛莎靑薠。<span class="inlay1"><span class="number">二二</span></span>其卑溼<span class="inlay1"><span class="number">二三</span></span>則生藏莨蒹葭，東薔<span class="inlay1"><span class="number">二四</span></span>雕胡，<span class="inlay1"><span class="number">二五</span></span>蓮藕菰蘆，<span class="inlay1"><span class="number">二六</span></span>菴琷軒芋，<span class="inlay1"><span class="number">二七</span></span>衆物居之，不可勝圖。<span class="inlay1"><span class="number">二八</span></span>其西則有湧泉淸池，激水推移；外發芙蓉蔆華，內隱鉅石白沙。其中則有神龜蛟鼉，<span class="inlay1"><span class="number">二九</span></span>玳瑁<span class="inlay1"><span class="number">三三</span></span>木蘭，<span class="inlay1"><span class="number">三四</span></span>蘗離朱楊，<span class="inlay1"><span class="number">三五</span></span>樝犭梬栗，<span class="inlay1"><span class="number">三六</span></span>橘柚芬芳。<span class="inlay1"><span class="number">三七</span></span>其上則有赤猿蠷蝚，<span class="inlay1"><span class="number">三八</span></span>鵷雛孔鸞，騰遠射干。<span class="inlay1"><span class="number">三九</span></span>其下則有白虎玄豹，蟃蜒貙豻，<span class="inlay1"><span class="number">四〇</span></span>兕象野犀，<span class="inlay1"><span class="number">四一</span></span>窮奇獌狿。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：郭璞云：「特，獨也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：赭詮音亡棟反，又音莫風反。裴駰云「孫叔敖激沮水作此澤」。張揖云「楚藪也，在南郡華容縣」。郭璞曰「江夏安陸有雲夢城，南郡枝江亦有雲夢城。華容縣又有巴丘湖，俗云卽古雲夢澤也」。則張揖云在華容者，指巴湖也。今安陸東見有雲夢城、雲夢縣，而枝江亦有者，蓋縣名取此澤，故有城也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：《漢書音義》曰：「高山壅蔽，日月虧缺半見。」　《索隱》：案：《漢書注》此卷多不題注者姓名，解者云是張揖，亦兼有餘人也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「一作『瑕』。」　《索隱》：張揖云：「赭，赤土，出少室山。堊，白堊，，《本草》云一名白墡也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《正義》：藥對曰：「雌黃出武都山谷，與雄黃同山。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「音符。」駰案：《漢書音義》曰「白坿，白石英也」。　《索隱》：張揖曰：「白石英也，出魯陽山。」蘇林音附，郭璞音符也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《正義》：顏云：「錫，靑金也。碧謂玉之靑白色者也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：郭璞曰：「如龍之鱗采。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《集解》：郭璞曰：「赤玉，赤瑾也。見《楚辭》。玫瑰，石珠也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：《漢書音義》曰：「琳，球也。珉，石次玉者。琨珸，山名也，出善金，《尸子》曰『昆吾之金』者。」　《索隱》：琨珸，司馬彪云「石之次玉者」。按：《河圖》云「流州多積石，名昆吾石，鍊之成鐵，以作劍，光明昭如水精」。案：字或作「昆吾」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《集解》：徐廣曰：「瑊音古咸反，䘏音勒，皆次玉者。」駰案：《漢書音義》曰「玄厲，黑石可用磨者」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《集解》：徐廣曰：「石似玉。」駰案：《漢書音義》曰「瓀石出鴈門，武夫出長沙也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《索隱》：司馬彪云：「蕙，香草也。」《本草》云：「蕙草一名蕙。」《廣志》云：「薰草綠葉紫莖，魏武帝以此燒香，今東下田有此草，莖葉似麻，其華正紫也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《集解》：《漢書音義》曰：「衡，杜衡也。其狀若葵，其臭如稺蕪。芷，白芷。若，杜若。」　《索隱》：張揖云「衡，杜衡，生下田山」。案：《山海經》云「天帝之山有草，葉如葵，臭如蘪蕪，可以走馬」。《博物志》云「一名土杏，其根一似細辛，葉似葵」。故《藥對》亦為似細辛是也。蘭，張揖云「秋蘭」。芷若，張揖云「若，杜若；芷，白芷也」。《本草》云「一名茞」。《埤蒼》云「齊曰茞，晉曰虈」。《字林》曰「茞音昌亥反，又音昌里反。虈音火嬌反」。《本草》又曰「杜若，一名杜衡。」今杜若葉似薑而有文理，莖葉皆有長毛。古今名號不同，故其所呼別也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《索隱》：《廣雅》云「烏蓬，射干」。《本草》名烏扇。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一六</span></span>《索隱》：芎藭。司馬彪云：「芎藭似藁本。」郭璞云：「今歷陽呼為江離。」《淮南子》云：「夫亂人者，若芎藭之與槁本。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一七</span></span>《集解》：徐廣曰：「猼音匹沃反。」駰案：《漢書音義》曰「江離，香草，蘪蕪，蘄芷也，似蛇床而香。諸蔗，甘柘也。猼且，蘘荷也」。　《索隱》：《吳錄》曰「臨海縣海水中生江離，正靑似亂髮，卽《離騷》所云者是也」。《廣志》云「赤葉紅華」，則與張勃所說又別。案：今芎藭苗曰江離，綠葉白華，又不同。孟康云「麋蕪，蘄芷也，似蛇床而香」。樊光曰「藁本一名麋蕪，根名蘄芷」。又《藥對》以為麋蕪一名江離，芎藭苗也。則芎藭、藁本、江離、麋蕪並相似，非是一物也。諸柘，張揖云「諸柘，甘柘也」。搏且，上音並卜反，下音子余反。《漢書》作「巴且」，文穎云「巴蕉也」。郭璞云「搏且，蘘荷屬」。未知孰是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一八</span></span>《集解》：音移糜。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一九</span></span>《索隱》：司馬彪云：「案衍，窳下；壇曼，平博也。」衍音弋戰反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二〇</span></span>《集解》：郭璞曰：「巫山今在建平巫縣也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二一</span></span>《集解》：徐廣曰：「葴音針，馬藍也。艻，或曰草，生水中，華可食。荔音力詣反。草，似蒲。」駰案：《漢書音義》曰「苞，藨也」。　《索隱》：葴析。音針斯二音。孟康曰「葴，馬藍也」。郭璞曰「葴，酸漿，江東名烏葴」。析，《漢書》作「斯」，孟康云「斯，禾，似燕麥」。《埤蒼》又云「生水中，華可食」。《廣志》云「涼州地生析草，皆如中國燕麥」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二二</span></span>《集解》：徐廣曰：「薛音先結反。」駰案：《漢書音義》曰「薛，賴蒿也。莎，鎬侯也。靑薠，似莎而大也。音煩」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二三</span></span>《索隱》：其庳溼。庳音婢。庳，下也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二四</span></span>《集解》：徐廣曰：「烏桓國有薔，似蓬草，實如葵子，十月熟。」駰案：《漢書音義》曰「藏，似薍而葉大。莨，莨尾草也。蒹，薕也。葭，蘆也」。　《索隱》：藏莨，郭璞云「狼尾，似茅」。蒹葭音兼加。孟康云「蒹葭似蘆也」。郭璞云「蒹，𧁱也。似雚而細小，高數尺，江東人呼為蒹蒿」。又云「葭，蘆也。似葦而細小，江東人呼為烏蓲」。薍音五患反。𧁱音敵。東薔，案《續漢書》云「東薔似蓬草，實如葵子，十一月熟」。《廣志》云「子色靑黑，河西語云『貸我東薔，償我白粱』也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二五</span></span>《索隱》：彫胡。案謂菰米。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二六</span></span>《集解》：徐廣曰：「生水中。」　《索隱》：郭璞云：「菰，蔣也。蘆，葦也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二七</span></span>《集解》：《漢書音義》曰：「奄閭，蒿也。軒芋，蕕草也。」　《索隱》：郭璞云：「菴閭，蒿，子可療病也。軒芋生水中，今楊州有也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二八</span></span>《集解》：郭璞曰：「圖，畫也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二九</span></span>《正義》：郭注《山海經》云：「蛟，似蛇而四脚，小頭細頸，有白嬰，大者數十圍，卵生，子如一二斛瓮，吞人。鼉，似蜥蜴而大，身有甲，皮可以冒鼓。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三〇</span></span>《正義》：似觜觿，甲有文，出南海，可飾器物也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三一</span></span>《集解》：郭璞曰：「林在山北陰地。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三二</span></span>《集解》：郭璞曰：「楩，杞也，似梓。柟，葉似桑。豫章，大木也，生七年乃可知也。」　《正義》：案：（溫）活人云「豫，今之枕木也。章，今之樟木也。二木生至七年，枕樟乃可分別」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三三</span></span>《正義》：郭璞云：「桂，似枇杷葉而大，白花，花而不著子，藂生巖嶺閒，無雜木，冬夏常靑。」案：今諸寺有桂樹，葉若枇杷而小，光靜，冬夏常靑，其皮不中食，蓋二色桂樹。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三四</span></span>《集解》：駰案：郭璞曰「木蘭，樹，皮辛香可食」。　《正義》：《廣雅》云：「似桂，皮辛可食，葉冬夏榮，常以冬華，其實如小（甘）柿，辛美，南人以為梅也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三五</span></span>《集解》：徐廣曰：「蘗音扶戾反。」《漢書音義》曰：「離，山梨。朱楊，赤楊也。」　《索隱》：朱楊，郭璞云「赤莖柳，生水邊」，《爾雅》云檉河柳是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三六</span></span>《集解》：徐廣曰：「梬音郢。」駰案：《漢書音義》曰「梬，梬棗也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三七</span></span>《正義》：小曰橘，大曰柚。樹有刺，冬不凋，葉靑，花白，子黃赤。二樹相似。非橙也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三八</span></span>《集解》：徐廣曰：「音劬柔。」　《正義》：蠷音劬，蝚音柔，皆猿猴類。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三九</span></span>《集解》：郭璞曰：「鵷雛，鳳屬也。孔，孔雀；鸞，鸞鳥也。」《漢書音義》曰：「騰遠，鳥名。射干，似狐，能緣木。」　《索隱》：孟康云「騰遠，鳥名」，非也。司馬彪云：「騰遠，蛇也。」郭璞云：「騰蛇，龍屬，能興雲霧。」張揖云：「射干，似狐，能緣木。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四〇</span></span>《集解》：郭璞曰：「蟃蜒，大獸，長百尋。貙，似貍而大。」《漢書音義》曰：「豻，胡地野犬，似狐而小也。」　《索隱》：郭璞云：「蟃蜒，大獸，長百尋。」張揖云：「貙，似貍而大。豻，胡地野犬，似狐而小，黑喙。」應劭音顏，韋昭一音岸。鄒誕生音苦姦反，協音，是。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四一</span></span>《正義》：兕，狀如水牛。象，大獸，長鼻，牙長一丈，俗呼為江猿。犀，頭似猿，一角在額。《漢書》無此一句。</span></p><p>　　　　「『於是乃使專諸之倫，手格此獸。楚王乃駕馴駁之駟<span class="inlay1"><span class="number">七</span></span>案節未舒，<span class="inlay1"><span class="number">八</span></span>卽陵狡獸，轔邛邛，蹴距虛，<span class="inlay1"><span class="number">九</span></span>軼野馬而𨎥騊駼，<span class="inlay1"><span class="number">一〇</span></span>乘遺風而射游騏；<span class="inlay1"><span class="number">一一</span></span>儵眒淒浰，<span class="inlay1"><span class="number">一二</span></span>雷動熛至，星流霆擊，弓不虛發，中必決眥，<span class="inlay1"><span class="number">一三</span></span>洞胸達腋，絕乎心繫，獲若雨獸，揜草蔽地。於是楚王乃弭節裴回，<span class="inlay1"><span class="number">一四</span></span>翱翔容與，<span class="inlay1"><span class="number">一五</span></span>覽乎陰林，觀壯士之暴怒，與猛獸之恐懼，徼谻受詘，<span class="inlay1"><span class="number">一六</span></span>殫睹衆物之變態。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《漢書音義》曰：「馴，擾也。駁，如馬，白身，黑尾，一角，鋸牙，食虎豹。擾而駕之，以當駟馬。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：郭璞曰：「以海魚須為旒旌，言橈弱也。通帛為旃也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：《漢書音義》曰：「以明月珠綴飾旗。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：《漢書音義》曰：「干將，韓王劍師。雄戟，胡中有茚，干將所造也。」　《索隱》：應劭曰：「干將，吳善冶者姓。」如淳曰：「干將，鐵所出。」晉灼曰：「闔閭鑄干將劍。」應劭說是。《方言》云：「戟中小孑刺者，所謂雄戟也。」周處《風土記》云：「戟為五兵雄也。」茚音巨。案：《周禮》「冶氏為戈，胡三之」。注云「胡其孑」也。又禮圖謂「戟支曲下為胡」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「韋昭云夏，夏羿也。矢室名曰服。」呂靜曰：「步叉謂之服也。」　《索隱》：案：夏羿，善射者。又服，箭室之名，故云「夏服」。又夏后氏有良弓名「繁弱」，其矢亦良，卽「繁弱箭服」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《集解》：《漢書音義》曰：「陽子，僊人陵陽子。纖阿，月御也。」韋昭曰：「陽子，古賢也。」　《索隱》：服虔云：「陽子，仙人陵陽子也。」張揖云：「陽子，伯樂也。孫陽字伯樂，秦繆公臣，善御者也。」服虔云：「纖阿為月御。或曰美女姣好貌。」又樂產曰：「纖阿，山名，有女子處其巖，月歷巖度，躍入月中，因名月御也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：郭璞曰：「言頓轡也。」司馬彪云「案轡徐行得節，故曰案節，馬足未展，故曰未舒之也」，亦（曰未）\〔為〕得也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《集解》：郭璞曰：「邛邛，似馬而色靑。距虛卽邛邛，變文互言之。《穆天子傳》曰『邛邛距虛，日走五百里』也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：徐廣曰：「𨎥音銳。」駰案：郭璞曰「野馬，如馬而小。騊駼，似馬。𨎥，車軸頭」。　《索隱》：䡺騊駼。上音衞。䡺，車軸頭也。謂車軸衝殺之。騊駼，野馬。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="inlay1"><span class="number">一</span></span>拘文牽俗，循誦習傳，當世取說云爾哉！必將崇論閎議，創業垂統，為萬世規。故馳騖乎兼容幷包，而勤思乎參天貳地。<span class="inlay1"><span class="number">二</span></span>且《詩》不云乎：『普天之下，莫非王土；率土之濱，莫非王臣。』<span class="inlay1"><span class="number">五</span></span>況乎上聖，又惡能已？故北出師以討彊胡，南馳使以誚勁越。四面風德，二方之君<span class="inlay1"><span class="number">六</span></span>鱗集仰流，願得受號者以億計。故乃關沬、若，<span class="inlay1"><span
