<h2>史記卷一百一十三</h2><h2>南越列傳第五十三</h2><p>　　南越王<span class="inlay1"><span class="number">一</span></span>尉佗者，<span class="inlay1"><span class="number">二</span></span>眞定人也，<span class="inlay1"><span class="number">三</span></span>姓趙氏。秦時已幷天下，略定楊越，<span class="inlay1"><span class="number">四</span></span>置桂林、<span class="inlay1"><span class="number">五</span></span>南海、象郡，<span class="inlay1"><span class="number">六</span></span>以謫<span class="inlay1"><span class="number">七</span></span>徙民，與越雜處十三歲。<span class="inlay1"><span class="number">八</span></span>佗，秦時用為南海龍川令<span class="inlay1"><span class="number">九</span></span>。至二世時，南海尉<span class="inlay1"><span class="number">一〇</span></span>任囂<span class="inlay1"><span class="number">一一</span></span>病且死，召龍川令趙佗語曰：「聞陳勝等作亂，秦為無道，天下苦之，項羽、劉季、陳勝、吳廣等州郡各共興軍聚衆，虎爭天下，中國擾亂，未知所安，豪傑畔秦相立。南海僻遠，吾恐盜兵侵地至此，吾欲興兵絕新道，<span class="inlay1"><span class="number">一二</span></span>自備，待諸侯變，會病甚。且番禺負山險，阻南海，東西數千里，頗有中國人相輔，此亦一州之主也，可以立國。郡中長吏無足與言者，故召公告之。」卽被佗書，<span class="inlay1"><span class="number">一三</span></span>行南海尉事。<span class="inlay1"><span class="number">一四</span></span>囂死，佗卽移檄告橫浦、<span class="inlay1"><span class="number">一五</span></span>陽山、<span class="inlay1"><span class="number">一六</span></span>湟谿<span class="inlay1"><span class="number">一七</span></span>關曰：「盜兵且至，急絕道聚兵自守！」因稍以法誅秦所置長吏，以其黨為假守。<span class="inlay1"><span class="number">一八</span></span>秦已破滅，佗卽擊幷桂林、象郡，自立為南越武王。<span class="inlay1"><span class="number">一九</span></span>高帝已定天下，為中國勞苦，故釋佗弗誅。漢十一年，遣陸賈因立佗為南越王，與剖符通使，和集百越，毋為南邊患害，與長沙接境。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：都廣州南海縣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：尉他。尉，官也；他，名也；姓趙。他音徒河反。又《十三州記》云「大郡曰守，小郡曰尉」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：韋昭曰：「故郡名，後更為縣，在常山。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：張晏曰：「楊州之南越也。」　《索隱》：案：《戰國策》云吳起為楚收楊越。　《正義》：夏禹九州本屬楊州，故云楊越。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：按：《地理志》武帝更名鬱林。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：案：本紀始皇三十三年略陸梁地，以為南海、桂林、象郡。《地理志》云「武帝更名日南」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：音直革反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：徐廣曰：「秦幷天下，至二世元年十三年。幷天下八歲，乃平越地，至二世元年六年耳。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：《地理志》縣名，屬南海也。　《正義》：顏師古云：「龍川南海縣也，卽今之循州也。」裴氏廣州記云：「本博羅縣之東鄕，有龍穿地而出，卽穴流泉，因以為號也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：徐廣曰：「爾時未言都尉也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：五刀反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《索隱》：案：蘇林云「秦所通越道」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《集解》：韋昭曰：「被之以書。音『光被』之『被』。」　《索隱》：韋昭云「被之以書」，音皮義反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《索隱》：服虔云：「囂詐作詔書，使為南海尉。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《索隱》：案：南康記云「南野縣大庾嶺三十里至橫浦，有秦時關，其下謂為『塞上』」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一六</span></span>《索隱》：姚氏案：《地理志》云揭陽有陽山縣。今此縣上流百餘里有騎田嶺，當是陽山關。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一七</span></span>《集解》：徐廣曰：「在桂陽，通四會也。」　《索隱》：涅谿。鄒氏、劉氏本並作「涅」，音年結反。《漢書》作「湟谿」，音皇。又\〔衞靑傳〕\〔《南粵傳》〕云「出桂陽，下湟水」是也。而姚察云《史記》作「涅」，今本作「湟」，涅及湟不同，良由隨聞則輒改故也。《水經》云含匯縣南有匯浦關，未知孰是。然鄒誕作「涅」，《漢書》作「湟」，蓋近於古。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一八</span></span>《索隱》：案：謂他立其所親黨為郡縣之職或假守。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一九</span></span>《集解》：韋昭曰：「生以『武』為號，不稽於古也。」</span></p><p>　　高后時，有司請禁南越關市鐵器。佗曰：「高帝立我，通使物，今高后聽讒臣，別異蠻夷，隔絕器物，此必長沙王計也，欲倚中國，擊滅南越而幷王之，自為功也。」於是佗乃自尊號為南越武帝，發兵攻長沙邊邑，敗數縣而去焉。高后遣將軍隆慮侯灶<span class="inlay1"><span class="number">一</span></span>往擊之。會暑溼，士卒大疫，兵不能踰嶺。<span class="inlay1"><span class="number">二</span></span>歲餘，高后崩，卽罷兵。佗因此以兵威邊，財物賂遺閩越、西甌、駱，役屬焉，<span class="inlay1"><span class="number">三</span></span>東西萬餘里。迺乘黃屋左纛，稱制，與中國侔。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：韋昭云：「姓周。隆慮，縣名，屬河內。音林閭二音。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：案：此嶺卽陽山嶺。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：《漢書音義》曰：「駱越也。」　《索隱》：鄒氏云「又有駱越」。姚氏案：廣州記云「交趾有駱田，仰潮水上下，人食其田，名為『駱人』。有駱王、駱侯。諸縣自名為『駱將』，銅印靑綬，卽今之令長也。後蜀王子將兵討駱侯，自稱為安陽王，治封溪縣。後南越王尉他攻破安陽王，令二使典主交阯、九眞二郡人」。尋此駱卽甌駱也。</span></p><p>　　及孝文帝元年，初鎭撫天下，使告諸侯四夷從代來卽位意，喻盛德焉。乃為佗親冢在眞定，置守邑，歲時奉祀。召其從昆弟，尊官厚賜寵之。詔丞相陳平等舉可使南越者，平言好畤陸賈，先帝時習使南越。迺召賈以為太中大夫，往使。因讓佗自立為帝，曾無一介之使報者。陸賈至南越，王甚恐，為書謝，稱曰：「蠻夷大長老夫臣佗，前日高后隔異南越，竊疑長沙王讒臣，又遙聞高后盡誅佗宗族，掘燒先人冢，以故自棄，犯長沙邊境。且南方卑溼，蠻夷中閒，其東閩越千人衆號稱王，其西甌駱裸國<span class="inlay1"><span class="number">一</span></span>亦稱王。老臣妄竊帝號，聊以自娛，豈敢以聞天王哉！」乃頓首謝，願長為藩臣，奉貢職。於是乃下令國中曰：「吾聞兩雄不俱立，兩賢不並世。皇帝，賢天子也。自今以後，去帝制黃屋左纛。」陸賈還報，孝文帝大說。遂至孝景時，稱臣，使人朝請。然南越其居國竊如故號名，其使天子，稱王朝命如諸侯。至建元四年卒。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：躶國。音和寡反。躶，露形也。</span></p><p><br/>　　佗孫胡為南越王。<span class="inlay1"><span class="number">一</span></span>此時閩越王郢興兵擊南越邊邑，胡使人上《書》曰：：「兩越俱為藩臣，毋得擅興兵相攻擊。今閩越興兵侵臣，臣不敢興兵，唯天子詔之。」於是天子多南越義，守職約，為興師，遣兩將軍<span class="inlay1"><span class="number">二</span></span>往討閩越。兵未踰嶺，閩越王弟餘善殺郢以降，於是罷兵。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「皇甫謐曰越王趙佗以建元四年卒，爾時漢興七十年，佗蓋百歲矣。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：王恢、韓安國。</span></p><p>　　天子使莊助往諭意南越王，胡頓首曰：「天子乃為臣興兵討閩越，死無以報德！」遣太子嬰齊入宿衞。謂助曰：「國新被寇，使者行矣。胡方日夜裝入見天子。」助去後，其大臣諫胡曰：「漢興兵誅郢，亦行以驚動南越。<span class="inlay1">孫守真按：行此震懾之事也。找對主詞對象。</span>且先王昔言，事天子期無失禮，要之不可以說好語入見。<span class="inlay1"><span class="number">一</span>孫守真按：1.說好語猶說定、言定，約諾也；猶今之「說好、說定」也。找對主詞：此好語乃越王語也。2.說=悅，字形結構換部首。找對主詞受詞：悅漢帝之好語，而被說動入覲也。感恩感恩南無阿彌陀佛</span>入見則不得復歸，亡國之勢也。」於是胡稱病，竟不入見。後十餘歲，胡實病甚，太子嬰齊請歸。胡薨，謚為文王。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：悅好語入見。悅，《漢書》作「怵」。韋昭云「誘怵好語」。</span></p><p><br/>　　嬰齊代立，卽藏其先武帝璽。<span class="inlay1"><span class="number">一</span></span>嬰齊其入宿衞在長安時，取邯鄲樛氏女，<span class="inlay1"><span class="number">二</span></span>生子興。<span class="inlay1"><span class="number">三</span></span>及卽位，上書請立樛氏女為后，興為嗣。漢數使使者風諭嬰齊，嬰齊尚樂擅殺生自恣，懼入見要用漢法，比內諸侯，固稱病，遂不入見。遣子次公入宿衞。嬰齊薨，謚為明王。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：李奇云「去其僭號」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：摎氏女。摎，紀虯反。摎姓出邯鄲。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「一作『典』。」</span></p><p><br/>　　太子興代立，其母為太后。太后自未為嬰齊姬時，嘗與霸陵人安國少季<span class="inlay1"><span class="number">一</span></span>通。及嬰齊薨後，元鼎四年，漢使安國少季往諭王、王太后以入朝，比內諸侯；令辯士諫大夫終軍等宣其辭，勇士魏臣等輔其缺，<span class="inlay1"><span class="number">二</span>孫守真按：輔=補，缺=決，字形結構換部首。</span>衞尉路博德將兵屯桂陽，待使者。王年少，太后中國人也，嘗與安國少季通，其使復私焉。國人頗知之，多不附太后。太后恐亂起，亦欲倚漢威，數勸王及羣臣求內屬。卽因使者上書，請比內諸侯，三歲一朝，除邊關。於是天子許之，賜其丞相呂嘉銀印，及內史、中尉、太傅印，餘得自置。除其故黥劓刑，<span class="inlay1">孫守真按：上下文、前後文：前文「嬰齊尚樂擅殺生自恣」。</span>用漢法，比內諸侯。使者皆留塡撫之。王、王太后飭治行裝重齎，為入朝具。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：安國，姓也；少季名也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「一作『決』。」</span></p><p>　　其相呂嘉年長矣，相三王，宗族官仕為長吏者七十餘人，男盡尚王女，女盡嫁王子兄弟宗室，及蒼梧秦王有連。<span class="inlay1"><span class="number">一</span>孫守真按：及=興；有=又。</span>其居國中甚重，越人信之，多為耳目者，得衆心愈於王。王之上書，數諫止王，王弗聽。有畔心，數稱病不見漢使者。使者皆注意嘉，勢未能誅。王、王太后亦恐嘉等先事發，乃置酒，介漢使者權，<span class="inlay1"><span class="number">二</span></span>謀誅嘉等。使者皆東鄕，太后南鄕，王北鄕，相嘉、大臣皆西鄕，侍坐飲。嘉弟為將，將卒居宮外。酒行，太后謂嘉曰：「南越內屬，國之利也，而相君苦不便者，何也？」以激怒使者。使者狐疑相杖，遂莫敢發。嘉見耳目非是，卽起而出。太后怒，欲鏦嘉<span class="inlay1"><span class="number">三</span></span>以矛，王止太后。嘉遂出，分其弟兵就舍，<span class="inlay1"><span class="number">四</span></span>稱病，不肯見王及使者。乃陰與大臣作亂。王素無意誅嘉，嘉知之，以故數月不發。太后有淫行，國人不附，欲獨誅嘉等，力又不能。<span class="inlay2">孫守真按：找對主詞：誰欲、孰獨？前文「太后」也。獨力、獨自也，單字想複詞。</span>}{</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《漢書音義》曰：「蒼梧越中王自名為秦王，連親婚也。」　《索隱》：案：蒼梧越中王自名為秦王，卽下趙光是也，故云「有連」。連者，連姻也。趙與秦同姓，故稱秦王。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：韋昭曰：「恃使者為介冑也。」　《索隱》：韋昭曰「恃使者為介冑」，《志林》云「介者因也，欲因使者權誅呂嘉」，然二家之說皆通。韋昭以介為恃。介者閒也，以言閒恃漢使者之權，意卽得；云恃為介冑，則非也。虞喜以介為因，亦有所由。案：介者，賓主所由也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：韋昭曰：「鏦，撞也。」　《索隱》：韋昭云：「鏦，撞也。」案：《字林》七凶反。又吳王濞傳「鏦殺吳王」，與此同。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：分弟兵就舍。案：謂分取其兵也。《漢書》作「介」。介，被也，恃也。</span></p><p>　　天子聞嘉不聽王，王、王太后弱孤不能制，使者怯無決。又以為王、王太后已附漢，獨呂嘉為亂，不足以興兵，欲使莊參以二千人往使。參曰：「以好往，數人足矣；以武往，二千人無足以為也。」辭不可，天子罷參也。郟<span class="inlay1"><span class="number">一</span></span>壯士故濟北相韓千秋奮曰：<span class="inlay1">孫守真按：單字想複詞：自告奮勇。不見題目只見關鍵字：壯、奮。</span>「以區區之越，又有王、太后應，獨相呂嘉為害，願得勇士二百人，必斬嘉以報。」於是天子遣千秋<span class="inlay1"><span class="number">四</span></span>好為謾辭謝罪，發兵守要害處。於是天子曰：「韓千秋雖無成功，亦軍鋒之冠。」封其子延年為成安侯。<span class="inlay1"><span class="number">五</span></span>樛樂，其姊為王太后，首願屬漢，封其子廣德為龍亢侯<span class="inlay1"><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「為校尉。」</span></p><p><span class="inlay2">　<span class="inlay1"><span
