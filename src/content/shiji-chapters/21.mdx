<h2>史記卷十五</h2><h2>六國年表第三</h2><p><span class="inlay2">　　《索隱》：六國，魏、韓、趙、楚、燕、齊，幷秦凡七國，號曰「七雄」。</span></p><p>　　太史公讀秦記，<span class="inlay1"><span class="number">一</span></span>至犬戎敗幽王，周東徙洛邑，秦襄公始封為諸侯，作西畤用事上帝，僭端見矣。禮曰：「天子祭天地，諸侯祭其域內名山大川。」今秦雜戎翟之俗，先暴戾，後仁義，位在藩臣而臚於郊祀，<span class="inlay1"><span class="number">二</span></span>君子懼焉。及文公踰隴，攘夷狄，尊陳寶，營岐雍之閒，而穆公脩政，東竟至河，則與齊桓、晉文中國侯伯侔矣。是後陪臣執政，大夫世祿，六卿擅晉權，征伐會盟，威重於諸侯。及田常殺簡公而相齊國，諸侯晏然弗討，海內爭於戰功矣。三國終之卒分晉，田和亦滅齊而有之，六國之盛自此始。務在彊兵幷敵，謀詐用而從衡短長之說起。矯稱蜂出，誓盟不信，雖置質剖符猶不能約束也。秦始小國僻遠，諸夏賓之，比於戎翟，至獻公之後常雄諸侯。論秦之德義不如魯衞之暴戾者，量秦之兵不如三晉之彊也，然卒幷天下，非必險固便形埶利也，蓋若天所助焉。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：卽秦國之史記也，故下云「秦燒詩書，諸侯史記尤甚。獨有秦記，又不載日月」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：案：臚字訓陳也，出《爾雅》文。以言秦是諸侯而陳天子郊祀，實僭也，猶季氏旅於泰山然。　《正義》：\〔臚作〕「臚」，音旅，祭名。又旅，陳也。</span></p><p>　　或曰「東方物所始生，西方物之成孰」。夫作事者必於東南，收功實者常於西北。故禹興於西羌，<span class="inlay1"><span class="number">一</span></span>湯起於亳，<span class="inlay1"><span class="number">二</span></span>周之王也以豐鎬伐殷，秦之帝用雍州興，漢之興自蜀漢。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：皇甫謐曰：「《孟子》稱禹生石紐，西夷人也。傳曰『禹生自西羌』是也。」　《正義》：禹生於茂州汶川縣，本冄駹國，皆西羌。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「京兆杜縣有亳亭。」</span></p><p>　　秦旣得意，燒天下詩書，諸侯史記尤甚，為其有所刺譏也。詩書所以復見者，多藏人家，而史記獨藏周室，以故滅。惜哉，惜哉！獨有秦記，又不載日月，其文略不具。然戰國之權變亦有可頗采者，何必上古。秦取天下多暴，然世異變，成功大。<span class="inlay1"><span class="number">一</span></span>傳曰「法後王」，何也？以其近己而俗變相類，議卑而易行也。<span class="inlay1"><span class="number">二</span></span>學者牽於所聞，見秦在帝位日淺，不察其終始，因舉而笑之，<span class="inlay1"><span class="number">三</span></span>不敢道，此與以耳食無異。<span class="inlay1"><span class="number">四</span></span>悲夫！</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：以言人君制法，當隨時代之異而變易其政，則其成功大。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：易，以豉反。後王，近代之王。法與己連接世俗之變及相類也，故議卑淺而易識行耳。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：舉猶皆也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：案：言俗學淺識，舉而笑秦，此猶耳食不能知味也。</span></p><p>　　余於是因秦記，踵《春秋》之後，起周元王，<span class="inlay1"><span
