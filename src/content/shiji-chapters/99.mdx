<h2>史記卷九十九</h2><h2>劉敬叔孫通列傳第三十九</h2><p>　　劉敬<span class="inlay1"><span class="number">一</span></span>者，齊人也。漢五年，戍隴西，過洛陽，高帝在焉。婁敬脫挽輅，<span class="inlay1"><span class="number">二</span>孫守真按：脫=稅，稅駕的稅。</span>衣其羊裘，見齊人虞將軍曰：「臣願見上言便事。」虞將軍欲與之鮮衣，<span class="inlay1"><span class="number">三</span>孫守真按：單字想複詞：鮮美、衣服，同義複詞∴鮮衣=美服。</span>婁敬曰：「臣衣帛，衣帛見；衣褐，衣褐見：終不敢易衣。」於是虞將軍入言上。上召入見，賜食。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：敬本姓婁，《漢書》作「婁敬」。高祖曰「婁卽劉也」，因姓劉耳。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：蘇林曰：「一木橫鹿車前，一人推之。」孟康曰：「輅音胡格反。挽音晚。」《索隱》：挽者，牽也。音晚。輅者，鹿車前橫木，二人前挽，一人後推之。音胡格反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：上音仙。鮮衣，美服也。</span></p><p>　　已而問婁敬，婁敬說曰：「陛下都洛陽，豈欲與周室比隆哉？」上曰：「然。」婁敬曰：「陛下取天下與周室異。周之先自后稷，堯封之邰，<span class="inlay1"><span class="number">一</span></span>積德累善十有餘世。公劉避桀居豳。太王以狄伐故，去豳，杖馬箠居岐，<span class="inlay1"><span class="number">二</span></span>國人爭隨之。及文王為西伯，斷虞芮之訟，始受命，呂望、伯夷自海濱來歸之。<span class="inlay1"><span class="number">三</span></span>武王伐紂，不期而會孟津之上八百諸侯，皆曰紂可伐矣，遂滅殷。成王卽位，周公之屬傅相焉，迺營成周洛邑，<span class="inlay1"><span class="number">四</span></span>以此為天下之中也，諸侯四方納貢職，道里均矣，有德則易以王，無德則易以亡。凡居此者，欲令周務以德致人，不欲依阻險，令後世驕奢以虐民也。及周之盛時，天下和洽，四夷鄕風，慕義懷德，附離<span class="inlay1"><span class="number">五</span>孫守真按：附離=附麗→附屬之換句話說耳。離=罹=麗，字形結構換部首部件+字形結構換聲符耳。</span>而並事天子，不屯一卒，不戰一士，八夷大國之民莫不賓服，效其貢職。及周之衰也，分而為兩，<span class="inlay1"><span class="number">六</span></span>天下莫朝，周不能制也。非其德薄也，而形勢弱也。今陛下起豐沛，收卒三千人，以之徑往而卷蜀漢，定三秦，與項羽戰滎陽，爭成皋之口，大戰七十，小戰四十，使天下之民肝腦塗地，父子暴骨中野，<span class="inlay1">孫守真按：倒序重組：野中。</span>不可勝數，哭泣之聲未絕，傷痍者未起，而欲比隆於成康之時，臣竊以為不侔也。且夫秦地被山帶河，四塞以為固，卒然有急，百萬之衆可具也。<span class="inlay1">孫守真按：具備，同義複詞。</span>因秦之故，資甚美膏腴之地，此所謂天府<span class="inlay1"><span class="number">七</span>孫守真按：資本，同義複詞。單字想複詞：資=本→根本→根→根據，同義複詞。∴「資甚美膏腴之地=據甚美膏腴之地」。</span>者也。陛下入關而都之，山東雖亂，秦之故地可全而有也。夫與人鬭，不搤其亢，<span class="inlay1"><span class="number">八</span></span>拊其背，<span class="inlay1">孫守真按：拊=扑=撲，字形結構換聲符耳·同音通假。或：拊=搏=捕。對錯問題,回到二作。或「拊掌」之拊=抵，「拊掌（ㄈㄨˇ ㄓㄤˇ）」即「抵掌（ㄓˇ ㄓㄤˇ）」，決非「輕拍」也。搏拊（ㄅㄛˊ ㄈㄨˇ）之名，即取於茲。則搏、拊均有打、擊之義也。今猶有搏擊（ㄅㄛˊ ㄐ丨ˊ）一詞，即此之謂也。則「拊」即「搏」即「擊、打擊」也。《周禮》：「大祭祀登歌擊拊。」「拊」即「擊」也。拊，《漢語大字典》「搏取」義引《淮南子·齊俗》：「穿窬拊揵、抽箕踰備之姦」劉文典《集解》。○搤=隘，字形結構換部首，隘為隘口，則搤則掐死也，如隘口之狹隘也。形聲聲旁兼義之字。亢=吭，字形結構換部首，單字想複詞：引吭高歌，則喉嚨、咽喉也。</span>未能全其勝也。今陛下入關而都，案秦之故地，<span class="inlay1">孫守真按：案=按，字形結構換部首→據=按照，單字想複詞：依據=依照，同義複詞。</span>此亦搤天下之亢而拊其背也。」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：邰音胎。雍州武功縣西南二十三里故斄城是也。《說文》云：「邰，炎帝之後，姜姓所封國，棄外家也。」毛萇云：「邰，姜嫄國，堯見天因邰而生后稷，故因封於邰也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：張晏曰：「言馬棰，示約。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：呂望宅及廟在蘇州海鹽縣西也。伯夷孤竹國在平州。皆濱東海也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：《括地志》云：「故王城一名河南城，本郟鄏，周公所築，在洛州河南縣北九里苑中東北隅。《帝王紀》云武王伐紂，營洛邑而定鼎焉。」按此卽營都城也。書云「乃營成周」。《括地志》云：「洛陽故城在洛州洛陽城東二十六里，周公所築，卽成周城也。《尚書》\〔序〕曰『成周旣成，遷殷頑民』。《帝王世紀》云『居謄之衆』。」按：劉敬說周之美，豈言居頑民之所？以此而論，（《漢書》）\〔書序〕非也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：《莊子》曰「附離不以膠漆」也。《索隱》：案：謂使離者相附也。義見《莊子》。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：《公羊傳》云：「東周者何？成周也。西周者何？王城也。」按：周自平王東遷，以下十二王皆都王城，至敬王乃遷都成周，王赧又居王城也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：案：《戰國策》蘇秦說惠王曰「大王之國，地勢形便，此所謂天府」。高誘注云「府，聚也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：張晏曰：「亢，喉嚨也。」《索隱》：扼音厄。亢音胡朗反，一音胡剛反。蘇林以為亢，頸大脈，俗所謂「胡脈」也。</span></p><p>　　高帝問羣臣，羣臣皆山東人，爭言周王數百年，<span class="inlay1">孫守真按：先抓動詞：王，亦字形結構換部首：旺也。一語雙關。單字想複詞：王天下、興旺。</span>秦二世卽亡，不如都周。上疑未能決。及留侯明言入關便，卽日車駕西都關中。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：謂卽日西都之計定也。</span></p><p>　　於是上曰：「本言都秦地者婁敬，『婁』者乃『劉』也。」賜姓劉氏，拜為郎中，號為奉春君。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：張晏云「春為歲之始，以其首謀都關中，故號奉春君。」。</span></p><p><br/>　　漢七年，韓王信反，高帝自往擊之。至晉陽，聞信與匈奴欲共擊漢，上大怒，使人使匈奴。匈奴匿其壯士肥牛馬，但見老弱及羸畜。<span class="inlay1"><span class="number">一</span></span>使者十輩來，皆言匈奴可擊。上使劉敬復往使匈奴，還報曰：「兩國相擊，此宜誇矜見所長。<span class="inlay1"><span class="number">二</span></span>今臣往，徒見羸瘠<span class="inlay1"><span class="number">三</span></span>老弱，此必欲見短，伏奇兵以爭利。愚以為匈奴不可擊也。」是時漢兵已逾句注，<span class="inlay1"><span class="number">四</span></span>二十餘萬兵已業行。<span class="inlay1">孫守真按：業已，倒序重組。</span>上怒，罵劉敬曰：「齊虜！以口舌得官，今迺妄言沮吾軍。」<span class="inlay1"><span class="number">五</span></span>械繫敬廣武。<span class="inlay1"><span class="number">六</span></span>遂往，至平城，匈奴果出奇兵圍高帝白登，七日然後得解。高帝至廣武，赦敬，曰：「吾不用公言，以困平城。吾皆已斬前使十輩言可擊者矣。」迺封敬二千戶，為關內侯，號為建信侯。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：上力為反，下許又反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：韋昭曰：「誇，張；矜，大也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：上力為反。瘠音稷。瘠，瘦也。《漢書》作「胔」，音漬。胔，肉也，恐非。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：句注山在代州鴈門縣西北三十里。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：沮音才敘反。詩傳曰「沮，止也，壞也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：《地理志》縣名，屬鴈門。《正義》：廣武故縣在句注山南也。</span></p><p>　　高帝罷平城歸，韓王信亡入胡。當是時，冒頓為單于，兵彊，控弦三十萬，<span class="inlay1"><span class="number">一</span></span>數苦北邊。上患之，問劉敬。劉敬曰：「天下初定，士卒罷於兵，未可以武服也。冒頓殺父代立，妻羣母，以力為威，未可以仁義說也。獨可以計久遠子孫為臣耳，<span class="inlay1">孫守真按：獨，只。計謀，同義複詞。計久遠=貽謀。子孫即下文「生子必為太子」也。上下文、前後文。</span>然恐陛下不能為。」上曰：「誠可，何為不能！顧為柰何？」劉敬對曰：「陛下誠能以適長公主妻之，厚奉遺之，彼知漢適女送厚，蠻夷必慕以為閼氏，生子必為太子。代單于。何者？貪漢重幣。陛下以歲時漢所餘彼所鮮數問遺，因使辯士風諭以禮節。冒頓在，固為子婿；<span class="inlay1">孫守真按：子有兒子（男子）、女子，故子婿即女婿也。</span>死，則外孫為單于。豈嘗聞外孫敢與大父抗禮者哉？兵可無戰以漸臣也。若陛下不能遣長公主，而令宗室及後宮詐稱公主，彼亦知，不肯貴近，無益也。」高帝曰：「善。」欲遣長公主。呂后日夜泣，曰：「妾唯太子、一女，柰何弃之匈奴！」上竟不能遣長公主，而取家人子名為長公主，妻單于。使劉敬往結和親約。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：應劭曰：「控，引也。」</span></p><p><br/>　　劉敬從匈奴來，因言「匈奴河南白羊、樓煩王，<span class="inlay1"><span class="number">一</span></span>去長安近者七百里，輕騎一日一夜可以至秦中。秦中新破，少民，地肥饒，可益實。夫諸侯初起時，非齊諸田，楚昭、屈、景莫能興。今陛下雖都關中，實少人。北近胡寇，東有六國之族，宗彊，一日有變，陛下亦未得高枕而臥也。<span class="inlay1">孫守真按：高，七星大法·文學語言，豈必墊高之枕方安臥耶？對錯問題,回到二作。枕頭枕得高高的，猶云睡得香香的，睡與香味何干？皆以喻耳。</span>臣願陛下徙齊諸田，楚昭、屈、景，燕、趙、韓、魏後，及豪桀名家居關中。無事，可以備胡；諸侯有變，亦足率以東伐。此彊本弱末之術也」。上曰：「善。」迺使劉敬徙所言關中十餘萬口。<span class="inlay1"><span class="number">二</span>孫守真按：添字還原：迺使劉敬徙所言者至關中凡十餘萬口。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：張晏云：「白羊，匈奴國名。」《索隱》：案：張晏云白羊，國名。二者並在河南。河南者，案在朔方之河南，舊並匈奴地也，今亦謂之新秦中。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：案：小顏云「今高陵、櫟陽諸田，華陰、好畤諸景，及三輔諸屈諸懷尚多，皆此時所徙也」。</span></p><p><br/><br/>　　叔孫通者，<span class="inlay1"><span class="number">一</span></span>薛人也。<span class="inlay1"><span class="number">二</span></span>秦時以文學徵，待詔博士。數歲，陳勝起山東，使者以聞，二世召博士諸儒生問曰：「楚戍卒攻蘄入陳，於公如何？」博士諸生三十餘人前曰：「人臣無將，將卽反，罪死無赦。<span class="inlay1"><span class="number">三</span>孫守真按：《漢語多功能字庫》「將」字云『戰國文字又多借「𨒫」為「將」，《包山楚簡》簡236：「𨒫(將)楚邦之帀(師)徒」，表示率領楚國的軍隊。』而《漢語大字典》亦曰「𨒫」同「逆」，蓋「𨒫、逆」形近而誤？故「逆」誤為「𨒫」，又以「𨒫」訛成「將」矣。。若然，則此「人臣無將，將即反，罪死無赦」數句亦猶「順我者昌，逆我者亡」耳；亡則死無赦也。感恩感恩南無阿彌陀佛。</span>願陛下急發兵擊之。」二世怒，作色。<span class="inlay1">孫守真按：倒序重組：色作。</span>叔孫通前曰：「諸生言皆非也。夫天下合為一家，毀郡縣城，鑠其兵，示天下不復用。且明主在其上，法令具於下，使人人奉職，四方輻輳，安敢有反者！此特羣盜鼠竊狗盜耳，何足置之齒牙閒。郡守尉今捕論，<span class="inlay1">孫守真按：添字還原+單字想複詞：緝捕而論罪。</span>何足憂。」二世喜曰：「善。」盡問諸生，諸生或言反，或言盜。於是二世令御史案諸生言反者下吏，非所宜言。諸言盜者皆罷之。迺賜叔孫通帛二十匹，衣一襲，<span class="inlay1"><span class="number">四</span></span>拜為博士。叔孫通已出宮，反舍，諸生曰：「先生何言之諛也？」通曰：「公不知也，我幾不脫於虎口！」<span class="inlay1"><span class="number">五</span></span>迺亡去，之薛，薛已降楚矣。及項梁之薛，叔孫通從之。敗於定陶，從懷王。懷王為義帝，徙長沙，叔孫通留事項王。漢二年，漢王從五諸侯入彭城，叔孫通降漢王。漢王敗而西，因竟從漢。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：晉灼曰：「《楚漢春秋》名何。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：《楚漢春秋》雲名何。薛，縣名，屬魯國。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：瓚曰：「將謂逆亂也。《公羊傳》曰『君親無將，將而必誅』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span
