<h2>史記卷五十一</h2><h2>荆燕世家第二十一</h2><p>　　荆王劉賈者，<span class="inlay1"><span class="number">一</span></span>諸劉，不知其何屬<span class="inlay1"><span class="number">二</span></span>初起時。漢王元年，還定三秦，劉賈為將軍，定塞地，<span class="inlay1"><span class="number">三</span></span>從東擊項籍。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：年表云都吳也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：《漢書》賈，高帝從父兄。　《索隱》：按：注引《漢書》，云賈，高祖從父兄，則班固或別有所見也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：賈將兵定塞地，塞卽桃林之塞。</span></p><p>　　漢四年，漢王之敗成皋，北渡河，得張耳、韓信軍，軍脩武，深溝高壘，使劉賈將二萬人，騎數百，渡白馬津入楚地，<span class="inlay1"><span class="number">一</span></span>燒其積聚，以破其業，無以給項王軍食。已而楚兵擊劉賈，賈輒壁不肯與戰，而與彭越相保。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：《括地志》云：「黎陽，一名白馬津，在滑州白馬縣北三十里。」按：賈從此津南過入楚地也。</span></p><p>　　漢五年，漢王追項籍至固陵，<span class="inlay1"><span class="number">一</span></span>使劉賈南渡淮圍壽春。<span class="inlay1"><span class="number">二</span></span>還至，使人閒招楚大司馬周殷。周殷反楚，佐劉賈舉九江，迎武王黥布兵，皆會垓下，共擊項籍。漢王因使劉賈將九江兵，與太尉盧綰西南擊臨江王共尉。<span class="inlay1"><span class="number">三</span></span>共尉已死，以臨江為南郡。<span class="inlay1"><span class="number">四</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「在陽夏。」　《正義》：《括地志》云：「固陵，陵名。在陳州宛丘縣西北四十二里。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：今壽州壽春縣是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：共敖之子。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：今荆州也。</span></p><p>　　漢六年春，會諸侯於陳，<span class="inlay1"><span class="number">一</span></span>廢楚王信，囚之，分其地為二國。當是時也，高祖子幼，昆弟少，又不賢，欲王同姓以鎭天下，乃詔曰：「將軍劉賈有功，及擇子弟可以為王者。」羣臣皆曰：「立劉賈為荆王，王淮東五十二城；<span class="inlay1"><span class="number">二</span></span>高祖弟交為楚王，王淮西三十六城。」<span class="inlay1"><span class="number">三</span></span>因立子肥為齊王。始王昆弟劉氏也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：今陳州也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：表云劉賈都吳。又《漢書》以東陽郡封賈。東陽卽臨淮，故云淮東也。　《正義》：《括地志》云西北四十里，蓋此縣是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：淮以西徐、泗、濠等州也。</span></p><p>　　高祖十一年秋，淮南王黥布反，東擊荆。荆王賈與戰，不勝，走富陵，<span class="inlay1"><span class="number">一</span></span>為布軍所殺。高祖自擊破布。十二年，立沛侯劉濞為吳王，王故荆地。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：《地理志》縣名，屬臨淮。　《正義》：《括地志》云：「富陵故城在楚州盱眙縣東北六十里。」</span></p><p><br/>　　燕王劉澤者，諸劉遠屬也。<span class="inlay1"><span class="number">一</span></span>高帝三年，澤為郎中。高帝十一年，澤以將軍擊陳豨，得王黃，為營陵侯。<span class="inlay1"><span class="number">二</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《漢書》曰：「澤，高祖從祖昆弟。」　《索隱》：按：注引《漢書》云高祖從祖昆弟。又《楚漢春秋》田子春說張卿云「劉澤，宗家也」。按言「宗家」，似疏遠矣。然則班固言「從祖昆弟」，當別有所見矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：《地理志》縣名，在北海。　《正義》：《括地志》云：「營陵故城在靑州北海縣南三十里。」</span></p><p>　　高后時，齊人田生<span class="inlay1"><span class="number">一</span></span>游乏資，以畫干營陵侯澤。<span class="inlay1"><span class="number">二</span></span>澤大說之，用金二百斤為田生壽。田生已得金，卽歸齊。二年，澤使人謂田生曰：「弗與矣。」<span class="inlay1"><span class="number">三</span></span>田生如長安，不見澤，而假大宅，令其子求事呂后所幸大謁者張子卿。<span class="inlay1"><span class="number">四</span></span>居數月，田生子請張卿臨，親脩具。張卿許往。田生盛帷帳共具，譬如列侯。張卿驚。酒酣，乃屛人說張卿曰：「臣觀諸侯王邸弟百餘，皆高祖一切功臣。<span class="inlay1"><span class="number">五</span></span>今呂氏雅故本推轂高帝就天下，<span class="inlay1"><span class="number">六</span></span>功至大，又親戚太后之重。太后春秋長，諸呂弱，太后欲立呂產為（呂）王，王代。太后又重發之<span class="inlay1"><span class="number">七</span></span>，恐大臣不聽。今卿最幸，大臣所敬，何不風大臣以聞太后，太后必喜。諸呂已王，萬戶侯亦卿之有。<span class="inlay1"><span class="number">八</span></span>太后心欲之，而卿為內臣，不急發，恐禍及身矣。」張卿大然之，乃風大臣語太后。太后朝，因問大臣。大臣請立呂產為呂王。太后賜張卿千斤金，張卿以其半與田生。田生弗受，因說之曰：「呂產王也，諸大臣未大服。今營陵侯澤，諸劉，為大將軍，獨此尚觖望。<span class="inlay1"><span class="number">九</span></span>今卿言太后，列十餘縣王之，彼得王，喜去，諸呂王益固矣。」張卿入言，太后然之。乃以營陵侯劉澤為琅邪王。琅邪王乃與田生之國。田生勸澤急行，毋留。出關，太后果使人追止之，已出，卽還。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：晉灼曰：「《楚漢春秋》田子春。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：服虔曰：「以計畫干之也。」文穎曰：「以工畫得寵也。」　《索隱》：畫，一音「計畫」之「畫」，又音「圖畫」之「畫」，兩家義並通也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：孟康曰：「與，黨與。言不復與我為與也。」文穎曰：「不得與汝相知。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「名澤。」駰案：如淳曰閹人也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：按：此一切猶一例，同時也，非如他一切訓權時也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：如淳曰：「呂公知高祖相貴，以女妻之，推轂使為長者。」瓚曰：「謂諸呂共推轂高祖征伐成帝業。雅，正意也。」　《索隱》：按：雅訓素也。謂呂氏素心奉推高祖取天下，若人推轂欲前進塗然也，此略同臣瓚之意也。推音昌誰反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《集解》：文穎曰：「欲發之，恐大臣不聽。」鄧展曰：「重難發事。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《正義》：高后紀云封張卿為建陵侯。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：觖音決，又音企。</span></p><p>　　及太后崩，琅邪王澤乃曰：「帝少，諸呂用事，劉氏孤弱。」乃引兵與齊王合謀西，<span class="inlay1"><span class="number">一</span></span>欲誅諸呂。至梁，聞漢遣灌將軍屯滎陽，澤還兵備西界，遂跳驅至長安。<span class="inlay1"><span class="number">二</span></span>代王亦從代至。諸將相與琅邪王共立代王為天子。天子乃徙澤為燕王，乃復以琅邪予齊，復故地。<span class="inlay1"><span class="number">三</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《漢書音義》曰：「澤至齊，為齊王所劫，不得去。乃說王，求詣京師，齊具車送之。不為本與齊合謀也。」　《索隱》：按：《漢書·齊王傳》云使祝午劫琅邪王至齊，因留琅邪王不得反國。澤乃說求入關，齊乃送之。與此文不同者，劉氏以為燕、齊兩史各言其主立功之跡，太史公聞疑傳疑，遂各記之，則所謂實錄。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：《漢書音義》曰：「跳驅，馳至長安也。」　《索隱》：跳，他彫反，脫獨去也。又音條，謂疾去也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：李奇曰：「本齊地，分以王澤，今復與齊也。」</span></p><p>　　澤王燕二年，薨，謚為敬王。傳子嘉，為康王。</p><p>　　至孫定國，與父康王姬姦，生子男一人。奪弟妻為姬。與子女三人姦。定國有所欲誅殺臣肥如令郢人，<span class="inlay1"><span class="number">一</span></span>郢人等告定國，定國使謁者以他法劾捕格殺郢人以滅口。至元朔元年，郢人昆弟復上書具言定國陰事，以此發覺。詔下公卿，皆議曰：「定國禽獸行，亂人倫，逆天，當誅。」上許之。定國自殺，國除為郡。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：如淳曰：「定國自欲有所殺餘臣，肥如令郢人以告之。」　《索隱》：按：如淳意以肥如亦臣名，令郢人以告定國也。小顏以為定國欲有所誅殺餘臣，而肥如令郢人乃告定國也。然按《地理志》，肥如在遼西也。</span></p><p><br/>　　太史公曰：荆王王也，由漢初定，天下未集，故劉賈雖屬疏，然以策為王，塡江淮之閒。劉澤之王，權激呂氏，<span class="inlay1"><span class="number">一</span></span>然劉澤卒南面稱孤者三世。事發相重，<span class="inlay1"><span class="number">二</span></span>豈不為偉乎！<span class="inlay1"><span class="number">三</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：謂田子春欲王劉澤，先使張卿說封呂產，乃恐以大臣觖望，澤卒得王，故為權激諸呂也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：晉灼曰：「澤以金與田生以事張卿，張卿言之呂后，而劉澤得王，故曰『事發相重』。或曰事起於相重也。」　《索隱》：按：謂先發呂氏令重，我亦得其功，是事發相重也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：偉者盛也，蓋盛其能激發也。</span></p><p><span class="inlay2">　【《索隱》述贊】：劉賈初從，首定三秦。旣渡白馬，遂圍壽春。始迎黥布，絕閒周殷。賞功胙士，與楚為鄰。營陵始爵，勳由擊陳。田生遊說，受賜千斤。權激諸呂，事發榮身。徙封傳嗣，亡於郢人。</span></p>
