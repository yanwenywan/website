<h2>史記卷一百二十五</h2><h2>佞幸列傳第六十五</h2><p>　　諺曰「力田不如逢年，善仕不如遇合」，<span class="inlay1"><span class="number">一</span></span>固無虛言。<span class="inlay1">孫守真按：無=不→非。暴亦甚也。文學語言。不學無術，相對位置有相關字義：不⇌無；學⇌術，抽換字面爾。</span>非獨女以色媚，而士宦亦有之。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「遇，一作『偶』。」</span></p><p>　　昔以色幸者多矣。至漢興，高祖至暴抗也，<span class="inlay1"><span class="number">一</span>孫守真按：抗=亢，字形結構換部首；單字想複詞：不卑不亢。</span>然籍孺以佞幸；孝惠時有閎孺。<span class="inlay1"><span class="number">二</span></span>此兩人非有材能，徒以婉佞貴幸，與上臥起，公卿皆因關說。<span class="inlay1"><span class="number">三</span>孫守真按：因=憑=仗。因憑=憑仗，同義複詞。</span>故孝惠時郎侍中皆冠鵕䴊，貝帶，<span class="inlay1"><span class="number">四</span></span>傅脂粉，<span class="inlay1"><span class="number">五</span></span>化閎、籍之屬也。<span class="inlay1">孫守真按：添字還原：化於。單字想複詞：潛移默化。</span>兩人徙家安陵。<span class="inlay1"><span class="number">六</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：暴伉。伉音苦浪反。言暴猛伉直。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：籍，閎，皆名也。孺，幼小也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：按：關訓通也。謂公卿因之而通其詞說。劉氏云「有所言說，皆關由之」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：《漢書音義》曰：「鵔鸃，鳥名。以毛羽飾冠，以貝飾帶。」　《索隱》：鵔鸃，應劭云：「鳥名，毛可以飾冠。」許愼云：「鷩鳥也。」《淮南子》云：「趙武靈王服貝帶鵔鸃。」《漢官儀》云：「秦破趙，以其冠賜侍中。」《三倉》云：「鵔鸃，神鳥也，飛光映天者也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：上音付。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：惠帝陵邑。</span></p><p><br/>　　孝文時中寵臣，士人則鄧通，宦者則趙同、<span class="inlay1"><span class="number">一</span></span>北宮伯子<span class="inlay1"><span class="number">二</span></span>。北宮伯子以愛人長者；<span class="inlay1">孫守真按：人民，同義複詞。單字想複詞。或為唐太宗諱改，本書《索隱》《正義》多有。〇又此不當作分號，蓋「以……幸」言此二人來由也。上下文、前後文+對錯問題,回到二作+斷句。下文「常為文帝參乘」，先抓動詞+找對主詞，亦指此二人，非但鄧通也。</span>而趙同以星氣幸，常為文帝參乘；鄧通無伎能。鄧通，蜀郡南安人也，<span class="inlay1"><span class="number">三</span></span>以濯船<span class="inlay1"><span class="number">四</span>孫守真按：對錯問題,回到二作：武英殿本亦作「濯」，注文引稱亦作「濯」，則此必為字形結構換部首：「櫂」耳。櫂=棹，字形結構換聲符耳。</span>為黃頭郎。<span class="inlay1"><span class="number">五</span></span>孝文帝夢欲上天，不能，有一黃頭郎從後推之上天，顧見其衣裻<span class="inlay1"><span class="number">六</span></span>帶後穿。覺<span class="inlay1"><span class="number">七</span></span>而之漸臺，<span class="inlay1"><span class="number">八</span></span>以夢中陰目求推者郎，<span class="inlay1">孫守真按：斷句+單字想複詞+添字還原：以夢中所見陰目求推者郎。陰暗，同義複詞。單字想複詞：暗中。目求，物色，以眼求尋也。</span>卽見鄧通，其衣後穿，夢中所見也。召問其名姓，姓鄧氏，名通，文帝說焉，<span class="inlay1"><span class="number">九</span></span>尊幸之日異。<span class="inlay1">孫守真按：斷句+單字想複詞：尊幸之，日異。日新月異。謂日益得寵也。日異，謂後一日殊異於前一日也。找對主詞對象：什麼東東異？或「益=異」，字形結構兼音義·同音通假？</span>通亦愿謹，不好外交，雖賜洗沐，不欲出。於是文帝賞賜通巨萬以十數，<span class="inlay1"><span class="number">一〇</span>孫守真按：巨萬，文學語言·七星大法。</span>官至上大夫。文帝時時如鄧通家遊戲。然鄧通無他能，不能有所薦士，獨自謹其身以媚上而已。上使善相者相通，曰「當貧餓死」。文帝曰：「能富通者在我也。何謂貧乎？」於是賜鄧通蜀嚴道銅山，<span class="inlay1"><span class="number">一一</span></span>得自鑄錢，「鄧氏錢」<span class="inlay1"><span class="number">一二</span></span>布天下。其富如此。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：《漢書》作「趙談」，此云「同」者，避太史公父名也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：顏云「姓北宮，名伯子」也。按：伯子，名。北宮之宦者也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「後屬犍為。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：濯音棹，遲敎反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「著黃帽也。」駰案：《漢書音義》曰「善濯船池中也。一說能持擢行船也。土，水之母，故施黃旄於船頭，因以名其郎曰黃頭郎」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「一無此字。」　《索隱》：音篤。裻者，衫襦之橫腰者。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：覺音敎。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《正義》：《括地志》云：「漸臺在長安故城中。《關中記》云未央宮西有蒼池，池中有漸臺，王莽死於此臺。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：《漢書》云：「上曰『鄧猶登也』，悅之。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《正義》：言賜通巨萬以至於十也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《正義》：《括地志》云：「雅州榮經縣北三里有銅山，卽鄧通得賜銅山鑄錢者。」案：榮經卽嚴道。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《正義》：錢譜云：「文字稱兩，同漢四銖文。」</span></p><p>　　文帝嘗病癰，鄧通常為帝唶吮之。<span class="inlay1"><span class="number">一</span>孫守真按：唶=吸，字形結構換聲符耳。</span>文帝不樂，從容問通曰：「天下誰最愛我者乎？」通曰：「宜莫如太子。」太子入問病，文帝使唶癰，唶癰而色難之。已而聞鄧通常為帝唶吮之，心慙，由此怨通矣。及文帝崩，景帝立，鄧通免，家居。居無何，人有告鄧通盜出徼外鑄錢。下吏驗問，頗有之，遂竟案，盡沒入鄧通家，尚負責數巨萬。長公主<span class="inlay1"><span class="number">二</span></span>賜鄧通，吏輒隨沒入之，<span class="inlay1"><span class="number">三</span></span>一簪不得著身。於是長公主乃令假衣食。<span class="inlay1"><span class="number">四</span></span>竟不得名一錢，<span class="inlay1"><span class="number">五</span>孫守真按：倒序重組+添字還原：終不得一錢得歸屬名下。單字想複詞：名，名下的名。</span>寄死人家。<span class="inlay1">孫守真按：客寄∴寄死=客死。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：唶，仕格反。吮，仕兗反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：韋昭曰：「景帝姊也。」　《索隱》：案：卽館陶公主也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：吏輒沒入。謂長公主別有物賜通，吏輒沒入以充贓也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：謂公主令人假與衣食。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：按：始天下名「鄧氏錢」，今皆沒入，卒竟無一錢之名也。</span></p><p><br/>　　孝景帝時，中無寵臣，然獨郎中令周文仁，<span class="inlay1"><span class="number">一</span></span>仁寵最過庸，<span class="inlay1"><span class="number">二</span>孫守真按：庸=常=恆∴過庸=逾恆。</span>乃不甚篤。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：《漢書》稱「周仁」，此上稱「周文」，今兼「文」作，恐後人加耳。案：仁字文。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：寵最過庸。案：庸，常也。言仁最被恩寵，過於常人，乃不甚篤，如韓嫣也。</span></p><p><br/>　　今天子中寵臣，士人則韓王孫嫣，<span class="inlay1"><span class="number">一</span></span>宦者則李延年。嫣者，弓高侯<span class="inlay1"><span class="number">二</span></span>孽孫也。今上為膠東王時，嫣與上學書相愛。及上為太子，愈益親嫣。嫣善騎射，善佞。上卽位，欲事伐匈奴，而嫣先習胡兵，以故益尊貴，官至上大夫，賞賜擬於鄧通。時嫣常與上臥起。江都王入朝，有詔得從入獵上林中。天子車駕蹕道未行，而先使嫣乘副車，從數十百騎，騖馳視獸。江都王望見，以為天子，辟從者，伏謁道傍。嫣驅不見。旣過，江都王怒，為皇太后泣曰：「請得歸國入宿衞，<span class="inlay1"><span class="number">三</span>孫守真按：對錯問題,回到二作：為=向=對。單字想複詞：不足為外人道也。先抓動詞·找對詞性：介詞。〇歸還，單字想複詞。</span>比韓嫣。」太后由此嗛嫣。<span class="inlay1"><span class="number">四</span></span>嫣侍上，出入永巷不禁，以姦聞皇太后。皇太后怒，使使賜嫣死。上為謝，終不能得，嫣遂死。而案道侯韓說，<span class="inlay1"><span class="number">五</span></span>其弟也，亦佞幸。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：音偃，又音於建反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「韓王信之子頹當也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：謂還爵封於天子，而請入宿衞。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「嗛，讀與『銜』同，《漢書》作『銜』字。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：音悅。嫣弟。</span></p><p>　　李延年，中山人也。父母及身兄弟及女，<span class="inlay1">孫守真按：1.單字想複詞：自身。2.親身，同義複詞，事必躬親。</span>皆故倡也。延年坐法腐，給事狗中。<span class="inlay1"><span class="number">一</span></span>而平陽公主言延年女弟善舞，上見，心說之，及入永巷，而召貴延年。延年善歌，為變新聲，而上方興天地祠，欲造樂詩歌弦之。延年善承意，弦次初詩。<span class="inlay1"><span class="number">二</span>孫守真按：弦次=倚聲。倚=次；弦=聲。</span>其女弟亦幸，有子男。延年佩二千石印，號協聲律。與上臥起，甚貴幸，埒如韓嫣也。<span class="inlay1"><span class="number">三</span></span>久之，寖與中人亂，<span class="inlay1"><span class="number">四</span></span>出入驕恣。及其女弟李夫人卒後，愛弛，則禽誅延年昆弟也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「主獵犬也。」　《索隱》：或犬監也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：歌初詩。按：初詩，卽所新造樂章。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「埒，等也。蜀都賦曰『卓鄭埒名』。又云埒者，疇等之名。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「一云坐弟季與中人亂。」</span></p><p>　　自是之後，內寵嬖臣大底外戚之家，然不足數也。<span class="inlay1">孫守真按：數說，同義複詞。單字想複詞：數一數二。</span>衞靑、霍去病亦以外戚貴幸，然頗用材能自進。</p><p><br/>　　太史公曰：甚哉愛憎之時！彌子瑕<span class="inlay1"><span class="number">一</span></span>之行，足以觀後人佞幸矣。雖百世可知也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：衞靈公之臣，事見《說苑》也。</span></p><p><span class="inlay2">　【《索隱》述贊】：傳稱令色，詩刺巧言。冠鸃入侍，傅粉承恩。黃頭賜蜀，宦者同軒。新聲都尉，挾彈王孫。泣魚竊駕，著自前論。</span></p>
