<h2>史記卷六十二</h2><h2>管晏列傳第二</h2><p>　　管仲夷吾者，潁上人也。<span class="inlay1"><span class="number">一</span></span>少時常與鮑叔牙游，鮑叔知其賢。管仲貧困，常欺鮑叔，<span class="inlay1"><span class="number">二</span></span>鮑叔終善遇之，不以為言。已而鮑叔事齊公子小白，管仲事公子糾。及小白立為桓公，公子糾死，管仲囚焉。鮑叔遂進管仲。<span class="inlay1"><span class="number">三</span></span>管仲旣用，任政於齊，<span class="inlay1"><span class="number">四</span></span>齊桓公以霸，九合諸侯，一匡天下，管仲之謀也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：潁，水名。《地理志》潁水出陽城。漢有潁陽、臨潁二縣，今亦有潁上縣。《正義》：韋昭云：「夷吾，姬姓之後，管嚴之子敬仲也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：《呂氏春秋》：「管仲與鮑叔同賈南陽，及分財利，而管仲嘗欺鮑叔，多自取。鮑叔知其有母而貧，不以為貪也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：齊世家云：「鮑叔牙曰：『君將治齊，則高傒與叔牙足矣。君且欲霸王，非管夷吾不可。夷吾所居國國重，不可失也。』於是桓公從之。」韋昭云：「鮑叔，齊大夫，姒姓之後，鮑叔之子叔牙也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：《管子》云：「相齊以九惠之敎，一曰老，二曰慈，三曰孤，四曰疾，五曰獨，六曰病，七曰通，八曰賑，九曰絕也。」</span></p><p>　　管仲曰：「吾始困時，嘗與鮑叔賈，<span class="inlay1"><span class="number">一</span></span>分財利多自與，鮑叔不以我為貪，知我貧也。吾嘗為鮑叔謀事而更窮困，鮑叔不以我為愚，知時有利不利也。吾嘗三仕三見逐於君，鮑叔不以我為不肖，知我不遭時也。吾嘗三戰三走，鮑叔不以我怯，知我有老母也。公子糾敗，召忽死之，吾幽囚受辱，鮑叔不以我為無恥，知我不羞小節而恥功名不顯於天下也。生我者父母，知我者鮑子也。」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：音古。</span></p><p>　　鮑叔旣進管仲，以身下之。子孫世祿於齊，有封邑者十餘世<span class="inlay1"><span class="number">一</span></span>，常為名大夫。天下不多管仲之賢而多鮑叔能知人也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：《系本》云「莊仲山產敬仲夷吾，夷吾產武子鳴，鳴產桓子啟方，啟方產成子孺，孺產莊子盧，盧產悼子其夷，其夷產襄子武，武產景子耐涉，耐涉產微，凡十代」。《系譜》同。</span></p><p><br/>　　管仲旣任政相齊，<span class="inlay1"><span class="number">一</span></span>以區區之齊在海濱，<span class="inlay1"><span class="number">二</span></span>通貨積財，富國彊兵，與俗同好惡。故其稱曰：<span class="inlay1"><span class="number">三</span>孫守真按：稱說、稱謂，同義複詞。</span>「倉廩實而知禮節，衣食足而知榮辱，上服度則六親固。<span class="inlay1"><span class="number">四</span></span>四維不張，國乃滅亡。<span class="inlay1"><span class="number">五</span></span>下令如流水之原，令順民心。」故論卑而易行。<span class="inlay1"><span class="number">六</span></span>俗之所欲，因而予之；俗之所否，因而去之。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：《國語》云：「齊桓公使鮑叔為相，辭曰：『臣之不若夷吾者五：寬和惠民，不若也；治國家不失其柄，不若也；忠惠可結於百姓，不若也；制禮義可法於四方，不若也；執枹鼓立於軍門，使百姓皆加勇，不若也。』」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：齊國東濱海也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：是夷吾著書所稱《管子》者，其書有此言，故略舉其要。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：上之服御物有制度，則六親堅固也。六親謂外祖父母一，父母二，姊妹三，妻兄弟之子四，從母之子五，女之子六也。王弼云「父、母、兄、弟、妻、子也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：《管子》曰：「四維，一曰禮，二曰義，三曰廉，四曰恥。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：言為政令卑下鮮少，而百姓易作行也。</span></p><p>　　其為政也，善因禍而為福，轉敗而為功。貴輕重，<span class="inlay1"><span class="number">一</span></span>慎權衡。<span class="inlay1"><span class="number">二</span></span>桓公實怒少姬，<span class="inlay1"><span class="number">三</span></span>南襲蔡，管仲因而伐楚，責包茅不入貢於周室。<span class="inlay1">孫守真按：入=內=納，字形結構換部首部件。</span>桓公實北征山戎，而管仲因而令燕修召公之政。於柯之會，<span class="inlay1"><span class="number">四</span></span>桓公欲背曹沫之約，<span class="inlay1"><span class="number">五</span></span>管仲因而信之，<span class="inlay1"><span class="number">六</span></span>諸侯由是歸齊。故曰：「知與之為取，政之寶也。」<span class="inlay1"><span class="number">七</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：輕重謂錢也。今《管子》有輕重篇。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：輕重謂恥辱也，權衡謂得失也。有恥辱甚貴重之，有得失甚戒慎之。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：按：謂怒蕩舟之姬，歸而未絕，蔡人嫁之。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：今齊州東阿也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：沫音昧，亦音末。《左傳》作「曹劌」。　《正義》：沫，莫葛反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：以劫許之，歸魯侵地。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：《老子》曰「將欲取之，必固與之」，是知此為政之所寶也。</span></p><p>　　管仲富擬於公室，有三歸、反坫，<span class="inlay1"><span class="number">一</span></span>齊人不以為侈。管仲卒，<span class="inlay1"><span class="number">二</span></span>齊國遵其政，常彊於諸侯。後百餘年而有晏子焉。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：三歸，三姓女也。婦人謂嫁曰歸。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：《括地志》云：「管仲冢在靑州臨淄縣南二十一里牛山之阿。說苑云『齊桓公使管仲治國，管仲對曰：「賤不能臨貴。」桓公以為上卿，而國不治，曰：「何故？」管仲對曰：「貧不能使富。」桓公賜之齊巿租，而國不治。桓公曰：「何故？」對曰：「疏不能制近。」桓公立以為仲父，齊國大安，而遂霸天下。』孔子曰：『管仲之賢而不得此三權者，亦不能使其君南面而稱伯。』」</span></p><p><br/>　　晏平仲嬰者，萊之夷維人也。<span class="inlay1"><span class="number">一</span></span>事齊靈公、莊公、景公<span class="inlay1"><span class="number">二</span></span>，以節儉力行重於齊。旣相齊，食不重肉，妾不衣帛。其在朝，君語及之，卽危言；<span class="inlay1"><span class="number">三</span>孫守真按：道貌岸然，仰之彌高，故云危也。</span>語不及之，卽危行。<span class="inlay1"><span class="number">四</span></span>國有道，卽順命；無道，卽衡命。<span class="inlay1"><span class="number">五</span></span><span class="inlay1">孫守真按：度量衡，同義複詞。單字想複詞。</span>以此三世顯名於諸侯。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：劉向《別錄》曰：「萊者，今東萊地也。」　《索隱》：名嬰，平謚，仲字。父桓子名弱也。　《正義》：晏氏齊記云齊城三百里有夷安，卽晏平仲之邑。漢為夷安縣，屬高密國。應劭云故萊夷維邑。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：系家及《系本》靈公名環，莊公名光，景公名杵臼也。　<span class="inlay1"><span class="number">三</span></span>《正義》：謂己謙讓，非雲功能。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：行，下孟反。謂君不知己，增修業行，畏責及也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《正義》：衡，秤也。謂國無道制秤量之，可行卽行。</span></p><p>　　越石父賢，在縲紲中。<span class="inlay1"><span class="number">一</span></span>晏子出，遭之塗，<span class="inlay1">孫守真按：遭逢，同義複詞。塗=途=涂，字形結構換部首部件耳。</span>解左驂贖之，載歸。弗謝，入閨。久之，越石父請絕。晏子戄然，<span class="inlay1"><span class="number">二</span>孫守真按：戄=矍=懼=瞿，字形結構換部首部件耳。</span>攝衣冠謝曰：「嬰雖不仁，免子於戹，何子求絕之速也？」石父曰：「不然。吾聞君子詘於不知己而信於知己者。<span class="inlay1"><span class="number">三</span></span>方吾在縲紲中，彼不知我也。夫子旣已感寤而贖我，是知己；知己而無禮，固不如在縲紲之中。」晏子於是延入為上客。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：縲音力追反。縲，黑索也。紲，系也。晏子《春秋》云：「晏子之晉，至中牟，睹獘冠反裘負薪，息於途側。晏子問曰：『何者？』對曰：『我石父也。苟免飢凍，為人臣僕。』晏子解左驂贖之，載與俱歸。」按：與此文小異也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：戄，床縛反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：信讀曰申，古《周禮》皆然也。申於知己謂以彼知我而我志獲申。</span></p><p>　　晏子為齊相，出，其御之妻從門閒而窺其夫。其夫為相御，擁大蓋，策駟馬，意氣揚揚甚自得也。旣而歸，其妻請去。夫問其故。妻曰：「晏子長不滿六尺，身相齊國，名顯諸侯。今者妾觀其出，志念深矣，常有以自下者。<span class="inlay1">孫守真按：卑下、謙下、低聲下氣、不恥下問、禮賢下士、求賢下士。單字想複詞。</span>今子長八尺，乃為人僕御，然子之意自以為足，妾是以求去也。」<span class="inlay1">孫守真按：是以=所以；是=所=此；以=因。∴是以=所以=因此。倒序重組。</span>其後夫自抑損。晏子怪而問之，御以實對。晏子薦以為大夫。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《皇覽》曰：「晏子冢在臨災城南淄水南桓公冢西北。」《正義》：注《皇覽》云：「晏子冢在臨淄城南菑水南桓公冢西北。」《括地志》云：「齊桓公墓在靑州臨淄縣東南二十三里鼎足上。」又云：「齊晏嬰冢在齊子城北門外。晏子云『吾生近市，死豈易吾志』。乃葬故宅後，人名曰淸節里。」按：恐《皇覽》誤，乃管仲冢也。</span></p><p><br/>　　太史公曰：吾讀管氏牧民、山高、乘馬、輕重、九府，<span class="inlay1"><span class="number">一</span></span>及《晏子春秋》，<span class="inlay1"><span class="number">二</span></span>詳哉其言之也。旣見其著書，欲觀其行事，故次其傳。<span class="inlay1">孫守真按：編次，單字想複詞。</span>至其書，世多有之，是以不論，論其軼事。<span class="inlay1"><span class="number">三</span>孫守真按：論=綸=經綸→整理、編次。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：劉向《別錄》曰：「九府書民閒無有。山高一名形勢。」　《索隱》：皆管氏所著書篇名也。按：九府，蓋錢之府藏，其書論鑄錢之輕重，故云輕重九府。餘如《別錄》之說。　《正義》：《七略》云《管子》十八篇，在法家。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：嬰所著書名晏子《春秋》。今其書有七篇，故下云「其書世多有」也。《正義》：《七略》云晏子《春秋》七篇，在儒家。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：軼音逸。</span></p><p>　　管仲世所謂賢臣，然孔子小之。豈以為周道衰微，桓公旣賢，而不勉之至王，乃稱霸哉？<span class="inlay1"><span class="number">一</span></span>語曰「將順其美，匡救其惡，<span class="inlay1">孫守真按：相對位置有相關字義：將順⇌匡救。</span>故上下能相親也」。<span class="inlay1"><span class="number">二</span></span>豈管仲之謂乎？</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：言管仲世所謂賢臣，孔子所以小之者，蓋以為周道衰，桓公賢主，管仲何不勸勉輔弼至於帝王，乃自稱霸主哉？故孔子小之云。蓋為前疑夫子小管仲為此。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：言管仲相齊，順百姓之美，匡救國家之惡，令君臣百姓相親者，是管之能也。</span></p><p>　　方晏子伏莊公尸哭之，成禮然後去，<span class="inlay1"><span class="number">一</span></span>豈所謂「見義不為無勇」者邪？至其諫說，犯君之顏，此所謂「進思盡忠，退思補過」者哉！假令晏子而在，余雖為之執鞭，所忻慕焉。<span class="inlay1"><span class="number">二</span>孫守真按：忻=欣，字形結構換部首。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：《左傳》崔杼弒莊公，晏嬰入，枕莊公尸股而哭之，成禮而出，崔杼欲殺之是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：太史公之羨慕仰企平仲之行，假令晏生在世，己雖與之為僕隸，為之執鞭，亦所忻慕。其好賢樂善如此。賢哉良史，可以示人臣之炯戒也。</span></p><p><span class="inlay2">　【《索隱》述贊】夷吾成霸，平仲稱賢。粟乃實廩，豆不掩肩。轉禍為福，危言獲全。孔賴左衽，史忻執鞭。成禮而去，人望存焉。</span></p>
