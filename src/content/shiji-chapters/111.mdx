<h2>史記卷一百一十一</h2><h2>衞將軍驃騎列傳第五十一</h2><p>　　大將軍衞靑者，平陽人也。<span class="inlay1"><span class="number">一</span></span>其父鄭季，為吏，給事平陽侯家，與侯妾衞媼通，<span class="inlay1"><span class="number">二</span></span>生靑。靑同母兄衞長子，而姊衞子夫自平陽公主家得幸天子，<span class="inlay1"><span class="number">三</span></span>故冒姓為衞氏。字仲卿。長子更字長君。長君母號為衞媼。媼長女衞孺，<span class="inlay1"><span class="number">四</span></span>次女少兒，次女卽子夫。後子夫男弟步、廣<span class="inlay1"><span class="number">五</span></span>皆冒衞氏。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：《漢書》云「其父鄭季，河東平陽人，以縣吏給事平陽侯之家」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：衞，姓也。媼，婦人老少通稱。《漢書》曰：與主家僮衞媼通。案：旣云家僮，故非老。或者媼是老稱，後追稱媼耳。又《外戚傳》云「薄姬父與魏王宗女魏媼通」，則亦魏是媼姓。而小顏云「衞者，舉其夫姓也」。然案此云「侯妾衞媼」，似更無別夫也。下云「同母兄衞長子及姊子夫皆冒衞姓」，又似有夫。其所冒之姓為父與母，皆未明也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「曹參曾孫平陽夷侯，時尚武帝姊平陽公主，生子襄。」　《索隱》：案：如淳云「本陽信長公主，為平陽侯所尚，故稱平陽公主」。按徐廣云「夷侯，曹參曾孫，名襄」。又按系家及《功臣表》「時」或作「疇」，《漢書》作「壽」，並文字殘缺，故不同也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：《漢書》云「君孺」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「步，一作『少』。」</span></p><p>　　靑為侯家人，少時歸其父，其父使牧羊。先母之子<span class="inlay1"><span class="number">一</span></span>皆奴畜之，不以為兄弟數。<span class="inlay1"><span class="number">二</span></span>靑嘗從入至甘泉居室，<span class="inlay1"><span class="number">三</span></span>有一鉗徒<span class="inlay1"><span class="number">四</span></span>相靑曰：「貴人也，官至封侯。」靑笑曰：「人奴之生，得毋笞罵卽足矣，安得封侯事乎！」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：服虔曰：「先母，適妻也。靑之適母。」　《索隱》：《漢書》作「民母」。服虔云「母，適妻也。靑之適母」。顧氏云「鄭季本妻編於民戶之閒，故曰民母」。今本亦或作「民母」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：音去聲。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：按：居室，署名，武帝改曰保宮。灌夫繫居室是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：張晏曰：「甘泉中徒所居也。」</span></p><p>　　靑壯，為侯家騎，從平陽主。建元二年春，靑姊子夫得入宮幸上。皇后，堂邑大長公主女也，<span class="inlay1"><span class="number">一</span></span>無子，妒。大長公主聞衞子夫幸，有身，妒之，乃使人捕靑。靑時給事建章，<span class="inlay1"><span class="number">二</span></span>未知名。大長公主執囚靑，欲殺之。其友騎郎公孫敖與壯士往篡取之，<span class="inlay1"><span class="number">三</span></span>以故得不死。上聞，乃召靑為建章監，侍中，及同母昆弟貴，賞賜數日閒累千金。孺為太僕公孫賀妻。少兒故與陳掌通，<span class="inlay1"><span class="number">四</span></span>上召貴掌。公孫敖由此益貴。子夫為夫人。靑為大中大夫。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「堂邑安侯陳嬰之孫夷侯午，尚景帝姊長公主，子季須。元鼎元年，季須坐姦自殺。」　《正義》：文穎云：「陳皇后，武帝姑女也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：案：晉灼云「上林中宮名也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：篡猶劫也，奪也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「陳平曾孫，名掌也。」</span></p><p><br/>　　元光五年，靑為車騎將軍，擊匈奴，出上谷；太僕公孫賀為輕車將軍，出雲中；大中大夫公孫敖為騎將軍，出代郡；衞尉李廣為驍騎將軍，出雁門：軍各萬騎。靑至蘢城，斬首虜數百。騎將軍敖亡七千騎；衞尉李廣為虜所得，得脫歸：皆當斬，贖為庶人。賀亦無功。</p><p>　　元朔元年春，衞夫人有男，<span class="inlay1"><span class="number">一</span></span>立為皇后。其秋，靑為車騎將軍，出雁門，三萬騎擊匈奴，斬首虜數千人。明年，匈奴入殺遼西太守，虜略漁陽二千餘人，敗韓將軍軍。漢令將軍李息擊之，出代；令車騎將軍靑出雲中以西至高闕。<span class="inlay1"><span class="number">二</span></span>遂略河南地，至于隴西，捕首虜數千，畜數十萬，走白羊、樓煩王。遂以河南地為朔方郡。<span class="inlay1"><span class="number">三</span></span>以三千八百戶封靑為長平侯。靑校尉蘇建有功，以千一百戶封建為平陵侯。使建築朔方城。<span class="inlay1"><span class="number">四</span></span>靑校尉張次公有功，封為岸頭侯。<span class="inlay1"><span class="number">五</span></span>天子曰：「匈奴逆天理，亂人倫，暴長虐老，以盜竊為務，行詐諸蠻夷，造謀藉兵，數為邊害，<span class="inlay1"><span class="number">六</span></span>故興師遣將，以征厥罪。詩不云乎，『薄伐玁狁，<span class="inlay1"><span class="number">七</span></span>至于太原』，『出車彭彭，城彼朔方』。<span class="inlay1"><span class="number">八</span></span>今車騎將軍靑度西河<span class="inlay1"><span class="number">九</span></span>至高闕，獲首虜二千三百級，車輜畜產畢收為鹵，已封為列侯，遂西定河南地，按榆谿舊塞，<span class="inlay1"><span class="number">一〇</span></span>絕梓領，梁北河，<span class="inlay1"><span class="number">一一</span></span>討蒲泥，破符離，<span class="inlay1"><span class="number">一二</span></span>斬輕銳之卒，捕伏聽者三千七十一級，<span class="inlay1"><span class="number">一三</span></span>執訊獲醜，<span class="inlay1"><span class="number">一四</span></span>驅馬牛羊百有餘萬，全甲兵而還，益封靑三千戶。」其明年，匈奴入殺代郡太守友，<span class="inlay1"><span class="number">一五</span></span>入略鴈門千餘人。其明年，匈奴大入代、定襄、上郡，殺略漢數千人。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：卽衞太子據也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：山名也。小顏云「一曰塞名，在朔方之北」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：按：謂北地郡之北，黃河之南。　《正義》：今夏州也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：《括地志》云：「夏州朔方縣北什賁故城是。」按：蘇建築，什賁之號蓋出蕃語也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：案：晉灼云「河東皮氏縣之亭名也」。　《正義》：服虔云：「鄕名也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：張晏曰：「從蠻夷借兵鈔邊也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：薄伐獫狁。此《小雅》六月詩，美宣王北伐也。薄伐者，言逐出之也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：《小雅》出車之詩也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《正義》：卽雲中郡之西河，今勝州東河也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：如淳曰：「案，行也。榆谿，舊塞名。」或曰按，尋也。　《索隱》：按榆谷舊塞。如淳云：「按，行也，尋也。榆谷，舊塞名也。」案：水經云「上郡之北有諸次水，東經榆林塞為榆谿」，是榆谷舊塞也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《集解》：如淳曰：「絕，度也。為北河作橋梁。」　《正義》：《括地志》云：「梁北河在靈州界也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《集解》：晉灼曰：「二王號。」　《索隱》：晉灼云：「二王號。」崔浩云：「漠北塞名。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《集解》：張晏曰：「伏於隱處，聽軍虛實。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《正義》：訊，問也。醜，衆。言執其生口問之，知虜處，獲得衆類也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《集解》：徐廣曰：「友者，太守名也。姓共也。」</span></p><p>　　其明年，元朔之五年春，漢令車騎將軍靑將三萬騎，出高闕；衞尉蘇建為游擊將軍，左內史李沮<span class="inlay1"><span class="number">一</span></span>為彊弩將軍，太僕公孫賀為騎將軍，代相李蔡為輕車將軍，皆領屬車騎將軍，俱出朔方；大行李息、岸頭侯張次公為將軍，出右北平：咸擊匈奴。匈奴右賢王當衞靑等兵，以為漢兵不能至此，飲醉。漢兵夜至，圍右賢王，右賢王驚，夜逃，獨與其愛妾一人壯騎數百馳，潰圍北去。漢輕騎校尉郭成等逐數百里，不及，得右賢裨王十餘人，<span class="inlay1"><span class="number">二</span></span>衆男女萬五千餘人，畜數千百萬，於是引兵而還。至塞，天子使使者持大將軍印，卽軍中拜車騎將軍靑為大將軍，諸將皆以兵屬大將軍，大將軍立號而歸。<span class="inlay1"><span class="number">三</span></span>天子曰：「大將軍靑躬率戎士，師大捷，獲匈奴王十有餘人，益封靑六千戶。」而封靑子伉為宜春侯，<span class="inlay1"><span class="number">四</span></span>靑子不疑為陰安侯，靑子登為發干侯。靑固謝曰：「臣幸得待罪行閒，賴陛下神靈，軍大捷，皆諸校尉力戰之功也。陛下幸已益封臣靑。臣靑子在𦇤緥中，<span class="inlay1"><span class="number">五</span></span>未有勤勞，上幸列地封為三侯，非臣待罪行閒所以勸士力戰之意也。伉等三人何敢受封！」天子曰：「我非忘諸校尉功也，今固且圖之。」乃詔御史曰：「護軍都尉公孫敖三從大將軍擊匈奴，常護軍，傅校獲王，<span class="inlay1"><span class="number">六</span></span>以千五百戶封敖為合騎侯。<span class="inlay1"><span class="number">七</span></span>都尉韓說從大將軍出窳渾，<span class="inlay1"><span class="number">八</span></span>至匈奴右賢王庭，為麾下搏戰獲王，<span class="inlay1"><span class="number">九</span></span>以千三百戶封說為龍覦侯。騎將軍公孫賀從大將軍獲王，以千三百戶封賀為南窌侯。<span class="inlay1"><span class="number">一〇</span></span>輕車將軍李蔡再從大將軍獲王，以千六百戶封蔡為樂安侯。校尉李朔，校尉趙不虞，校尉公孫戎奴，各三從大將軍獲王，以千三百戶封朔為涉軹侯，以千三百戶封不虞為隨成侯，以千三百戶封戎奴為從平侯。將軍李沮、李息及校尉豆如意有功，賜爵關內侯，食邑各三百戶。」其秋，匈奴入代，殺都尉朱英。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：文穎曰：「音俎。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：裨王十人。賈逵云：「裨，益也。」小顏云：「裨王，小王也，若裨將然。音頻移反。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：案：謂立大將軍之號令而歸。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：伉音口浪反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《正義》：襁長尺二寸，闊八寸，以約小兒於背。褓，小兒被也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：顧祕監云：「傅，領也。五百人謂之校。」小顏云：「傅音附。言敖總護諸軍，每附部校，以致克捷而獲王也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：案：非邑地，而以戰功為號。謂以軍合驃騎，故云「合騎」，若「冠軍」、「從驃」然也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：徐廣曰：「窳渾在朔方，音庾。」　《索隱》：音庾。服虔云「塞名」。徐廣云「在朔方」。《漢書》作「寘渾」，寘音田也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：搏音博。搏，擊也。小顏同。今史、漢本多作「傳」，傳猶轉也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：徐廣曰：「窌宜作『奅』，音匹孝反。」　《索隱》：徐音匹敎反。韋昭云縣名。或作「窖」，音干校反。《字林》云「大」下「卯」與「穴」下「卯」並音匹孝反。</span></p><p>　　其明年春，大將軍靑出定襄，合騎侯敖為中將軍，太僕賀為左將軍，翕侯趙信為前將軍，衞尉蘇建為右將軍，郎中令李廣為後將軍，右內史李沮為彊弩將軍，咸屬大將軍，斬首數千級而還。月餘，悉復出定襄擊匈奴，斬首虜萬餘人。右將軍建、前將軍信幷軍三千餘騎，獨逢單于兵，與戰一日餘，漢兵且盡。前將軍故胡人，降為翕侯，見急，匈奴誘之，遂將其餘騎可八百，奔降單于。右將軍蘇建盡亡其軍，獨以身得亡去，自歸大將軍。大將軍問其罪正閎、<span class="inlay1"><span class="number">一</span></span>長史安、<span class="inlay1"><span class="number">二</span></span>議郎周霸等：<span class="inlay1"><span class="number">三</span></span>「建當云何？」霸曰：「自大將軍出，未嘗斬裨將。今建弃軍，可斬以明將軍之威。」閎、安曰：「不然。兵法『小敵之堅，大敵之禽也』。今建以數千當單于數萬，力戰一日餘，士盡，不敢有二心，自歸。自歸而斬之，是示後無反意也。不當斬。」大將軍曰：「靑幸得以肺腑待罪行閒，不患無威，而霸說我以明威，甚失臣意。且使臣職雖當斬將，以臣之尊寵而不敢自擅專誅於境外，而具歸天子，天子自裁之，於是以見為人臣不敢專權，不亦可乎？」軍吏皆曰「善」。遂囚建詣行在所。<span
