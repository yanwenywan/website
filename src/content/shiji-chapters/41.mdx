<h2>史記卷四十一</h2><h2>越王句踐世家第十一</h2><p>　　越王句踐，其先禹之苗裔，<span class="inlay1"><span class="number">一</span></span>而夏后帝少康之庶子也。封於會稽，以奉守禹之祀。文身斷髮，<span class="inlay1">孫守真按：華夏蓄髮，身體髮膚……，故以為異也。今俗均斷髮矣。</span>披草萊而邑焉。後二十餘世，至於允常。<span class="inlay1"><span class="number">二</span></span>允常之時，與吳王闔廬戰而相怨伐。允常卒，子句踐立，是為越王。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：《吳越春秋》云：「禹周行天下，還歸大越，登茅山以朝四方羣臣，封有功，爵有德，崩而葬焉。至少康，恐禹跡宗廟祭祀之絕，乃封其庶子於越，號曰無餘。」賀循會稽記云：「少康，其少子號曰於越，越國之稱始此。」《越絕記》云：「無餘都，會稽山南故越城是也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：《輿地志》云：「越侯傳國三十餘葉，歷殷至周敬王時，有越侯夫譚，子曰允常，拓土始大，稱王，《春秋》貶為子，號為於越。」杜注云：「於，語發聲也。」</span></p><p>　　元年，吳王闔廬聞允常死，乃興師伐越。越王句踐使死士挑戰，三行，<span class="inlay1">孫守真按：單字想複詞：行列，行伍。對錯問題,回到二作。</span>至吳陳，呼而自剄。吳師觀之，越因襲擊吳師，吳師敗於檇李，<span class="inlay1"><span class="number">一</span></span>射傷吳王闔廬。闔廬且死，告其子夫差曰：「必毋忘越。」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：杜預曰：「吳郡嘉興縣南有檇李城。」　《索隱》：事在《左傳》魯定公十四年。</span></p><p>　　三年，句踐聞吳王夫差日夜勒兵，<span class="inlay1">孫守真按：勒=練，雙聲通假？字形結構兼音義。</span>且以報越，越欲先吳未發往伐之。范蠡諫曰：「不可。臣聞兵者凶器也，戰者逆德也，爭者事之末也。陰謀逆德，好用凶器，試身於所末，上帝禁之，行者不利。」越王曰：「吾已決之矣。」遂興師。吳王聞之，悉發精兵擊越，敗之夫椒。<span class="inlay1"><span class="number">一</span>孫守真按：伍子胥傳作「夫湫」，同音通假耳·字形結構兼音義·聲調不重要。嘴巴張不夠開，「椒」音就成了「湫」了。</span>越王乃以餘兵五千人保棲於會稽。<span class="inlay1"><span class="number">二</span></span>吳王追而圍之。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：杜預曰：「夫椒在吳郡吳縣，太湖中椒山是也。」　《索隱》：夫音符。椒音焦，本又作「湫」，音酒小反。賈逵云地名。《國語》云敗之五湖，則杜預云在椒山為非。事具哀公元年。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：杜預曰：「上會稽山也。」　《索隱》：鄒誕云：「保山曰棲，猶鳥棲於木以避害也，故《六韜》曰『軍處山之高者則曰棲』。」</span></p><p>　　越王謂范蠡曰：<span class="inlay1"><span class="number">一</span></span>「以不聽子故至於此，為之柰何？」蠡對曰：「持滿者與天，<span class="inlay1"><span class="number">二</span></span>定傾者與人，<span class="inlay1"><span class="number">三</span>孫守真按：盈滿⇌傾覆，單字想複詞+相對位置有相關字義。保持、保定、必定，同義複詞。下「節」，調節、節制，故非必定者也。此二句言必然可保者則與天（倚天），必然不可保者則與人（投靠人）。</span>節事者以地。<span class="inlay1"><span class="number">四</span>孫守真按：節制，同義複詞。</span>卑辭厚禮以遺之，不許，而身與之市。」<span class="inlay1"><span class="number">五</span>孫守真按：市，賣，市身猶云賣身也。</span>句踐曰：「諾。」乃令大夫種行成於吳，<span class="inlay1"><span class="number">六</span></span>膝行頓首曰：「君王亡臣句踐使陪臣種敢告下執事：句踐請為臣，妻為妾。」吳王將許之。子胥言於吳王曰：「天以越賜吳，勿許也。」種還，以報句踐。句踐欲殺妻子，燔寶器，<span class="inlay1">孫守真按：燔即焚，同音通假耳。燔，形聲造字；焚，會意造字。字形結構兼音義·聲音變化的關係。</span>觸戰以死。<span class="inlay1">孫守真按：抵觸、抗戰，同義複詞。∴觸戰=抗戰，抽換字面爾。單字想複詞。抗=亢，卑以擊亢也，猶以卵擊石，玉石俱焚。</span>種止句踐曰：「夫吳太宰嚭貪，可誘以利，請閒行<span class="inlay1"><span class="number">七</span>孫守真按：1.間行，猶云走後門也。2.倒序重組：行間（離間）。</span>言之。」於是句踐以美女寶器令種閒獻吳太宰嚭。<span class="inlay1"><span class="number">八</span></span>嚭受，乃見大夫種於吳王。種頓首言曰：「願大王赦句踐之罪，盡入其寶器。<span class="inlay1">孫守真按：入=內=納，字形結構換部首部件。單字想複詞：當得起=擔待=擔得起。擔當，同義複詞。</span>不幸不赦，句踐將盡殺其妻子，燔其寶器，悉五千人觸戰，必有當也。」<span class="inlay1"><span class="number">九</span>孫守真按：1.當，宜，適當，適宜，同義複詞，單字想複詞。2.當=償→代價（典當之當，單字想複詞）字形結構換部首部件。</span>嚭因說吳王曰：「越以服為臣，<span class="inlay1">孫守真按：以=已，同音通假。</span>若將赦之，此國之利也。」吳王將許之。子胥進諫曰：「今不滅越，後必悔之。句踐賢君，種、蠡良臣，若反國，將為亂。」吳王弗聽，卒赦越，罷兵而歸。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：會稽典錄云：「范蠡字少伯，越之上將軍也。本是楚宛三戶人，佯狂倜儻負俗。文種為宛令，遣吏謁奉。吏還曰：『范蠡本國狂人，生有此病。』種笑曰：『吾聞士有賢俊之姿，必有佯狂之譏，內懷獨見之明，外有不知之毀，此固非二三子之所知也。』駕車而往，蠡避之。後知種之必來謁，謂兄嫂曰：『今日有客，願假衣冠。』有頃種至，抵掌而談，旁人觀者聳聽之矣。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：韋昭曰：「與天，法天也。天道盈而不溢。」　《索隱》：與天，天與也。言持滿不溢，與天同道，故天與之。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：虞翻曰：「人道尚謙卑以自牧。」　《索隱》：人主有定傾之功，故人與之也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：韋昭曰：「時不至，不可彊生；事不究，不可彊成。」　《索隱》：《國語》「以」作「與」，此作「以」，亦與義也。言地能財成萬物，<span class="inlay1">孫守真按：財=裁，字形結構換部首部件耳（「裁」从「𢦔」；「𢦔」从「才」，訛為「𢦏」耳。）。字形結構兼音義·同音通假。</span>人主宜節用以法地，故地與之。韋昭等解恐非。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：韋昭曰：「市，利也。謂委管籥屬國家，以身隨之。」　《正義》：卑作言辭，厚遺珍寶。不許平，越王身往事之，如市賈貨易以利，此是定傾危之計。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：大夫，官；種，名也。一曰大夫姓，猶司馬、司徒之比，蓋非也。成者，平也，求和於吳也。　《正義》：《吳越春秋》云：「大夫種姓文名種，字子禽。荆平王時為宛令，之三戶之里，范蠡從犬竇蹲而吠之，從吏恐文種慙，令人引衣而鄣之。<span class="inlay1">孫守真按：鄣=鄣，字形結構換部首部件位置耳。</span>文種曰：『無鄣也。吾聞犬之所吠者人，今吾到此，有聖人之氣，行而求之，來至於此。且人身而犬吠者，謂我是人也。』乃下車拜，蠡不為禮。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：閒音紀閑反。閒行猶微行。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：《國語》云：「越飾美女二人，使大夫種遺太宰嚭。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：言悉五千人觸戰，或有能當吳兵者，故《國語》作「耦」，耦亦相當對之名。又下云「無乃傷君王之所愛乎」，是有當則相傷也。</span></p><p>　　句踐之困會稽也，喟然嘆曰：「吾終於此乎？」種曰：「湯繫夏臺，<span class="inlay1">孫守真按：「御史臺、烏臺」之臺。單字想複詞。</span>文王囚羑里，晉重耳犇翟，齊小白犇莒，其卒王霸。由是觀之，何遽不為福乎？」<span class="inlay1">孫守真按：倒序重組：遽何。遽=詎，同音通假，字形結構兼音義。</span></p><p>　　吳旣赦越，越王句踐反國，乃苦身焦思，置膽於坐，坐臥卽仰膽，<span class="inlay1">孫守真按：仰服，同義複詞。相對位置有相關字義：仰⇌嘗=嚐，字形結構換部首。</span>飲食亦嘗膽也。曰：「女忘會稽之恥邪？」身自耕作，夫人自織，食不加肉，衣不重采，<span class="inlay1">孫守真按：二「重」字，平聲、仄聲聲可，對錯問題,回到二作：平聲則重複、多重→多樣→華飾，仄聲乃注重→講究。</span>折節下賢人，厚遇賓客，振貧弔死，<span class="inlay1"><span class="number">一</span></span>與百姓同其勞。欲使范蠡治國政，蠡對曰：「兵甲之事，種不如蠡；塡<span class="inlay1"><span class="number">二</span></span>撫國家，親附百姓，蠡不如種。」於是舉國政屬大夫種，而使范蠡與大夫柘稽<span class="inlay1"><span class="number">三</span></span>行成，為質於吳。二歲而吳歸蠡。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「弔，一作『葬』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：鎭音。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：越大夫也。《國語》作「諸稽郢」。</span></p><p>　　句踐自會稽歸七年，拊循其士民，欲用以報吳。大夫逢同<span class="inlay1"><span class="number">一</span></span>諫曰：「國新流亡，今乃復殷給，繕飾備利，吳必懼，懼則難必至。<span class="inlay1">孫守真按：國難。單字想複詞。</span>且鷙鳥之擊也，必匿其形。今夫吳兵加齊、晉，怨深於楚、越，名高天下，實害周室，德少而功多，必淫自矜。為越計，莫若結齊，親楚，附晉，以厚吳。吳之志廣，必輕戰。是我連其權，<span class="inlay1">孫守真按：權力，同義複詞。</span>三國伐之，越承其弊，可克也。」句踐曰：「善。」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：逢，姓；同，名。故楚有逢伯。</span></p><p>　　居二年，吳王將伐齊。子胥諫曰：「未可。臣聞句踐食不重味，與百姓同苦樂。此人不死，必為國患。吳有越，腹心之疾，齊與吳，疥㿅<span class="inlay1"><span class="number">一</span>孫守真按：瘲=癬，字形結構換聲符耳·聲音變化的關係。</span>也。願王釋齊先越。」吳王弗聽，遂伐齊，敗之艾陵<span class="inlay1"><span class="number">二</span></span>，虜齊高、國<span class="inlay1"><span class="number">三</span></span>以歸。讓子胥。子胥曰：「王毋喜！」王怒，子胥欲自殺，王聞而止之。越大夫種曰：「臣觀吳王政驕矣，請試嘗之貸粟，以卜其事。」<span class="inlay1">孫守真按：借貸，同義複詞。卜，測，試也。</span>請貸，吳王欲與，子胥諫勿與，王遂與之，越乃私喜。子胥言曰：「王不聽諫，後三年吳其墟乎！」太宰嚭聞之，乃數與子胥爭越議，因讒子胥曰：「伍員貌忠而實忍人，其父兄不顧，安能顧王？王前欲伐齊，員彊諫，已而有功，用是反怨王。王不備伍員，員必為亂。」與逢同共謀，讒之王。王始不從，乃使子胥於齊，聞其託子於鮑氏，王乃大怒，曰：「伍員果欺寡人！」役反，<span class="inlay1">孫守真按：使役，同義複詞。單字想複詞。反=返，字形結構換部首。∴役反即使返，抽換字面爾。</span>使人賜子胥屬鏤劍以自殺。子胥大笑曰：「我令而父霸，<span class="inlay1"><span class="number">四</span></span>我又立若，<span class="inlay1"><span class="number">五</span></span>若初欲分吳國半予我，<span class="inlay1">孫守真按：而=爾=若=汝=你，字形結構兼音義，雙聲通假，均係抽換字面爾。</span>我不受，已，今若反以讒誅我。<span class="inlay1">孫守真按：斷句：「已」字實不必斷。已=矣，同音通假。</span>嗟乎，嗟乎，一人固不能獨立！」報使者曰：「必取吾眼置吳東門，以觀越兵入也！」<span class="inlay1"><span class="number">六</span></span>於是吳任嚭政。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：疥脚音介尟。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：在魯哀十一年。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：國惠子、高昭子。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：而，汝也。父，闔廬也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：若亦汝也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：《國語》云吳王慍曰「孤不使大夫得見」，乃盛以鴟夷，投之于江也。</span></p><p>　　居三年，句踐召范蠡曰：「吳已殺子胥，導諛者衆，可乎？」對曰：「未可。」</p><p>　　至明年春，吳王北會諸侯於黃池，<span class="inlay1"><span class="number">一</span></span>吳國精兵從王，惟獨老弱與太子留守。<span class="inlay1"><span class="number">二</span></span>句踐復問范蠡，蠡曰「可矣」。乃發習流二千人，<span class="inlay1"><span class="number">三</span>。孫守真按：習=諳，單字想複詞：習慣=諳於。流，字形結構兼音義。單字想複詞：水流。「習流」即「諳水性」之換句話說耳。</span>敎士四萬人，<span class="inlay1"><span class="number">四</span>孫守真按：教練，同義複詞。∴教士=練士。</span>君子六千人，<span class="inlay1"><span class="number">五</span></span>諸御千人，<span class="inlay1"><span class="number">六</span></span>伐吳。吳師敗，遂殺吳太子。吳告急於王，王方會諸侯於黃池，懼天下聞之，乃祕之。吳王已盟黃池，乃使人厚禮以請成越。越自度亦未能滅吳，乃與吳平。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：在哀十三年。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「狡，一作『郊』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：越絕云：「九術：一曰尊天事鬼；二曰重財幣以遺其君；三曰貴糴粟稿以空其邦；四曰遺之好美以熒其志；五曰遺之巧匠，使起宮室高臺，以盡其財，以疲其力；六曰貴其諛臣，使之易伐；七曰彊其諫臣，使之自殺；八曰邦家富而備器利；九曰堅甲利兵以承其弊。」</span></p><p><br/>　　句踐卒，<span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：鼫音石。與音餘。按：《紀年》云「於粵子句踐卒，是菼執。次鹿郢立，六年卒」。樂資云「越語謂鹿郢為鼫與也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：《紀年》云：「不壽立十年見殺，是為盲姑。次朱句立。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：《紀年》於粵子朱句三十四年滅滕，三十五年滅郯，三十七年朱句卒。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：《紀年》云：「翳三十三年遷于吳，三十六年七月太子諸咎弒其君翳，十月粵殺諸咎。粵滑，吳人立子錯枝為君。明年，大夫寺區定粵亂，立無余之。十二年，寺區弟忠弒其君莽安，次無顓立。無顓八年薨，是為菼蠋卯。」故莊子云「越人三弒其君，子搜患之，逃乎丹穴不肯出，越人薰之以艾，乘以王輿」。樂資云「號曰無顓」。蓋無顓後乃次無彊也，則王之侯卽無余之也。</span></p><p><span
