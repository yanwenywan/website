<h2>史記卷一百二十四</h2><h2>游俠列傳第六十四</h2><p><span class="inlay2">　　《集解》：荀悅曰：「立氣齊，作威福，結私交，以立彊於世者，謂之游俠。」</span></p><p>　　《韓子》曰：「儒以文亂法，<span class="inlay1"><span class="number">一</span></span>而俠以武犯禁。」二者皆譏<span class="inlay1"><span class="number">二</span></span>，而學士多稱於世云。<span class="inlay1">孫守真按：學士、術，即韓子所謂「儒」也。找對主詞。然于下文「此皆學士所謂有道仁人也」則此「學士」同於「世論」之「世」也。上下文、前後文+對錯問題,回到二作。</span>至如以術取宰相卿大夫，輔翼其世主，功名俱著於春秋，<span class="inlay1"><span class="number">三</span></span>固無可言者。及若季次、原憲，閭巷人也，<span class="inlay1"><span class="number">四</span></span>讀書懷獨行君子<span class="inlay1"><span class="number">五</span></span>之德，義不苟合當世，當世亦笑之。故季次、原憲終身空室蓬戶，<span class="inlay1"><span class="number">六</span></span>褐衣疏食不厭。<span class="inlay1"><span class="number">七</span></span>死而已四百餘年，而弟子志之不倦。<span class="inlay1">孫守真按：志=誌，與前春秋「著、稱」相應，抽換字面爾，上下文、前後文+對錯問題,回到二作。</span>今游俠，其行雖不軌於正義，然其言必信，其行必果，已諾必誠，不愛其軀，赴士之阨困，<span class="inlay1"><span class="number">八</span></span>旣已存亡死生矣，<span class="inlay1">孫守真按：先抓動詞+找對主詞+添字還原：存、死，動詞；存人亡，死我身。</span>而不矜其能，羞伐其德，蓋亦有足多者焉。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：言文之蔽，小人以僿。謂細碎苛法亂政。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：譏，非言也。儒敝亂法，俠盛犯禁，二道皆非，而學士多稱於世者，故太史公引《韓子》，欲陳游俠之美。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：功名俱著春秋。案：春秋謂國史也。以言人臣有功名則見記于其國之史，是俱著春秋者也。</span><span class="inlay1">孫守真按：此蓋用魯春秋（楚檮杌、晉乘）之義。單字想複詞。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「《仲尼弟子傳》曰公皙哀字季次，未嘗仕，孔子稱之。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：行音下孟反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：《莊子》云「原憲處居環堵之室，蓬戶不完。以桑為樞而甕牖，上漏下溼，獨坐而弦歌」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：不饜。饜，飽也，於豔反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：上音厄。</span></p><p>　　且緩急，人之所時有也。太史公曰：昔者虞舜窘於井廩，<span class="inlay1">孫守真按：添字還原：井、廩，井（與）廩。</span>伊尹負於鼎俎，<span class="inlay1">孫守真按：單字想複詞：身負、負擔，同義複詞。上下文、前後文+相對位置有相關字義：窘⇌負⇌匿⇌困……均係負面辭之抽換字面爾。</span>傅說匿於傅險，呂尚困於棘津，<span class="inlay1"><span class="number">一</span></span>夷吾桎梏，百里飯牛，仲尼畏匡，菜色陳、蔡。此皆學士所謂有道仁人也，猶然遭此菑，況以中材而涉亂世之末流乎？其遇害何可勝道哉！</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「在廣川。」　《正義》：尉繚子云太公望行年七十，賣食棘津云。古亦謂之石濟津，故南津。</span></p><p>　　鄙人有言曰：「何知仁義，已饗其利<span class="inlay1"><span class="number">一</span></span>者為有德。」故伯夷醜周，<span class="inlay1">孫守真按：單字想複詞家醜之醜，羞也。羞恥，同義複詞。</span>餓死首陽山，而文武不以其故貶王；<span class="inlay1">孫守真按：單字想複詞：貶損，同義複詞。</span>跖、蹻暴戾，其徒誦義無窮。<span class="inlay1">孫守真按：義氣。單字想複詞。</span>由此觀之，「竊鉤者誅，<span class="inlay1"><span class="number">二</span></span>竊國者侯，侯之門仁義存」<span class="inlay1"><span class="number">三</span>孫守真按：存有、在在，同義複詞。</span>，非虛言也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：已音以。饗音享，受也。言已受其利則為有德，何知必仁義也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：以言小竊則為盜而受誅也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：言人臣委質於侯王門，則須存于仁義。若游俠輕健，亦何必肯存仁義也。</span></p><p>　　今拘學或抱咫尺之義，久孤於世，<span class="inlay1"><span class="number">一</span>孫守真按：拘泥，單字想複詞，食古不化。拘，迂，字形結構兼音義·疊韻通假？拘=句→曲。字形結構兼音義·字形結構換部首。</span>豈若卑論儕俗，<span class="inlay1">孫守真按：此當作頓號，意未足盡也。添字還原：卑己論而儕其俗。</span>與世沈浮而取榮名哉！而布衣之徒，設取予然諾，<span class="inlay1">孫守真按：單字想複詞：設使=假使→即使。添字還原：設能。下已用「使」故此改用「設」爾。設使，同義複詞，抽換字面爾。</span>千里誦義，為死不顧世，<span class="inlay1">孫守真按：單字想複詞：義無反顧。</span>此亦有所長，非苟而已也。故士窮窘而得委命，此豈非人之所謂賢豪閒者邪？<span class="inlay1">孫守真按：間，ㄐ丨ㄢ，武英殿本作「間」。謂側身於賢豪，猶云賢豪之類、流也。豪傑，單字想複詞。</span>誠使鄕曲之俠，予季次、原憲比權量力，效功於當世，不同日而論矣。<span class="inlay1">孫守真按：班彪乃至《漢書》本傳所贊「道游俠，則賤守節而貴俗功」「序遊俠則退處士而進姦雄」者，即謂此也。○予=與，字形結構兼音義·同音通假。此猶「給與」之「與」假借為「予」，故「相與」之「與」亦借「予」為之矣。蓋當初用字不若今求正字，但音通義達，則假取活用，求其靈通，適意而已，如春申君傳，太史公曰，故用「旄」字，皆不以錯別為意也。此吾後來以正字為鵠的圭臬者當知避嫌，但勿以小人之心度君子之腹則可矣。</span>要以功見言信，<span class="inlay1">孫守真按：要以，總以、總之。1.相對位置有相關字義+添字還原：功見而言信。言必信，功足見。2.功見方言信（可信），言其行誼足信服人。</span>俠客之義又曷可少哉！<span class="inlay1">孫守真按：少，薄，輕視也。妄自菲薄之薄。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：言拘學守義之士或抱咫尺纖微之事，遂久以當代，孤負我志，而不若卑論儕俗以取榮寵也。</span></p><p>　　古布衣之俠，靡得而聞已。近世延陵、<span class="inlay1"><span class="number">一</span>孫守真按：靡=無=不。</span>孟嘗、春申、平原、信陵之徒，皆因王者親屬，藉於有土卿相之富厚，招天下賢者，顯名諸侯，不可謂不賢者矣。比如順風而呼，聲非加疾，其埶激也。至如閭巷之俠，脩行砥名，聲施<span class="inlay1"><span class="number">二</span></span>於天下，莫不稱賢，是為難耳。然儒、墨皆排擯不載。自秦以前，匹夫之俠，湮滅不見，余甚恨之。以余所聞，漢興有朱家、田仲、王公、劇孟、郭解之徒，雖時扞當世之文罔，<span class="inlay1"><span class="number">三</span>孫守真按：文，單字想複詞：法律條文；罔，字形結構換部首→網。→文網=法網。</span>然其私義廉絜退讓，有足稱者。名不虛立，士不虛附。至如朋黨宗彊比周，設財役貧，豪暴侵淩孤弱，恣欲自快，游俠亦醜之。余悲世俗不察其意，而猥以朱家、郭解等令與暴豪之徒同類而共笑之也。<span class="inlay1">孫守真按：猥，輕薄，故輕易、率爾，故有「遽、遂、猝、驟」之義。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「代郡亦有延陵縣。」駰案：《韓子》云「趙襄子召延陵生，令車騎先至晉陽」。襄子時趙已幷代，可有延陵之號，但未詳是此人非耳。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：施音以豉反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：扞卽捍也。違扞當代之法網，謂犯於法禁也。</span></p><p><br/>　　魯朱家者，與高祖同時。魯人皆以儒敎，而朱家用俠聞。所藏活豪士以百數，其餘庸人不可勝言。然終不伐其能，歆其德，<span class="inlay1">孫守真按：歆=矜，字形結構換聲符？字形結構兼音義·同音通假？</span>諸所嘗施，唯恐見之。<span class="inlay1">孫守真按：下「終身不見也」即是例說。上下文、前後文。</span>振人不贍，先從貧賤始。家無餘財，衣不完采，食不重味，乘不過軥牛。<span class="inlay1"><span class="number">一</span></span>專趨人之急，甚己之私。旣陰脫季布將軍之阸，<span class="inlay1"><span class="number">二</span></span>及布尊貴，終身不見也。自關以東，莫不延頸願交焉。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「音雊。」駰案：《漢書音義》曰「小牛」。　《索隱》：上音古豆反。案：大牛當軶，小為軥牛。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：陰脫季將軍之厄。案：季布為漢所購求，朱家以布髡鉗為奴，載以廣柳車而出之，及尊貴而不見之，亦高介至義之士。然布竟不見報朱家之恩。</span></p><p>　　楚田仲以俠聞，喜劍，父事朱家，自以為行弗及。田仲已死，而雒陽有劇孟。周人以商賈為資，而劇孟以任俠顯諸侯。<span class="inlay1">孫守真按：東周都洛陽故也。</span>吳楚反時，條侯為太尉，乘傳車將至河南，得劇孟，喜曰：「吳楚舉大事而不求孟，吾知其無能為已矣。」天下騷動，宰相得之若得一敵國云。劇孟行大類朱家，而好博，<span class="inlay1"><span class="number">一</span></span>多少年之戲。然劇孟母死，自遠方送喪蓋千乘。及劇孟死，家無餘十金之財。而符離人王孟亦以俠稱江淮之閒。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：六博戲也。</span></p><p>　　是時濟南瞯氏、<span class="inlay1"><span class="number">一</span></span>陳周庸<span class="inlay1"><span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「陝，疑當作『郟』字，潁川有郟縣。南越傳曰『郟壯士韓千秋』也。」　《索隱》：陝當為「郟」。陝音如冄反，郟音紀洽反。《漢書》作「寒孺」。</span></p><p><br/>　　郭解，軹人也，<span class="inlay1"><span class="inlay1">孫守真按：上下文、前後文+對錯問題,回到二作：「矜」即前文「歆」，抽換字面爾。</span>其陰賊著於心，卒發於睚眦如故云。而少年慕其行，亦輒為報仇，不使知也。解姊子負解之勢，<span class="inlay1"><span class="number">五</span></span>與人飲，使之嚼。<span class="inlay1"><span class="number">六</span>孫守真按：嚼=爵，字形結構換部首，爵盡即乾杯，一飲而盡也。一爵即一杯，此轉品作動詞耳。先抓動詞。</span>非其任，<span class="number">一</span>孫守真按：既=已→止、厭足。豈有既乎=伊于胡底。先抓動詞：貌。榮，虛榮，單字想複詞。上下文、前後文+對錯問題,回到二作：此句承前「皆慕其聲，言俠者皆引以為名」，找對主詞：乃謂人皆好攀附以取榮，豈如徐廣所云，故不敢同也。貌，本為名詞，轉品，此類多讀作以其後所接之名詞受詞為其本身也。如「恥之」即「以之為恥」也。則此「貌榮名」即「以榮名為貌」，猶云「德潤身」，此則「榮名潤其（言俠者）貌」也，貌似而非真也，向聲背實，以刺如文中「盜跖居民間、不足數者」。故下乃云：「於戲，惜哉！」也，惜如郭解、朱家之真材實料非以貌取貌襲者之佳人難再得已。徐氏此解，一如吾先人常態，往往不講究文法而遽以靈感推敲想當然者爾，全然不周顧上下文也。於戲，惜哉！○於戲，惜哉！亦惜「郭解、朱家」之為聲名所累、被後輩牽誤也。故往往不為大雅君子所容，非其殊無可取之處。——文中「余悲世俗不察其意，而猥以朱家、郭解等令與暴豪之徒同類而共笑之也。」不平之鳴，即與相呼應。「於戲，惜哉！」即「悲」字注腳。</span>於戲，惜哉！</p><p><span
