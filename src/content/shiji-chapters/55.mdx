<h2>史記卷五十五</h2><h2>留侯世家第二十五</h2><p>　　留侯<span class="inlay1"><span class="number">一</span></span>張良者，<span class="inlay1"><span class="number">二</span></span>其先韓人也。<span class="inlay1"><span class="number">三</span></span>大父開地，<span class="inlay1"><span class="number">四</span></span>相韓昭侯、宣惠王、襄哀王。父平，相釐王、悼惠王。<span class="inlay1"><span class="number">五</span></span>悼惠王二十三年，平卒。卒二十歲，秦滅韓。良年少，未宦事韓。韓破，良家僮三百人，弟死不葬，悉以家財求客刺秦王，為韓報仇，以大父、父五世相韓故。<span class="inlay1"><span class="number">六</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：韋昭云「留，今屬彭城」。按：良求封留，以始見高祖於留故也。《正義》：《括地志》云：「故留城在徐州沛縣東南五十五里。今城內有張良廟也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：《漢書》云字子房。按：王符、皇甫謐並以良為韓之公族，姬姓也。秦索賊急，乃改姓名。而韓先有張去疾乃張譴，恐非良之先代。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：良旣歷代相韓，故知其先韓人。顧氏按：《後漢書》云「張良出於城父」，城父縣屬潁川也。《正義》：《括地志》云：「城父在汝州郟城縣東三十里，韓（里）\〔地〕也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：應劭曰：「大父，祖父。開地，名。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：《韓系家》及《系本》作桓惠王。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：謂大父及父相韓五王，故云五代。</span></p><p>　　良嘗學禮淮陽。<span class="inlay1"><span class="number">一</span></span>東見倉海君。<span class="inlay1"><span class="number">二</span></span>得力士，為鐵椎重百二十斤。秦皇帝東游，良與客狙<span class="inlay1"><span class="number">三</span></span>擊秦皇帝博浪沙中，<span class="inlay1"><span class="number">四</span></span>誤中副車。<span class="inlay1"><span class="number">五</span></span>秦皇帝大怒，大索天下，求賊甚急，為張良故也。良乃更名姓，亡匿下邳。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：今陳州也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：如淳曰：「秦郡縣無倉海。或曰東夷君長。」《索隱》：姚察以武帝時東夷穢君降，為倉海郡，或因以名，蓋得其近也。　《正義》：《漢書》武帝紀云「\〔元朔〕元年，東夷穢君南閭等降，為倉海郡，今貊穢國」，得之。太史公修史時已降為郡，自書之。《括地志》云：「穢貊在高麗南，新羅北，東至大海西。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：服虔曰：「狙，伺候也。」應劭曰：「狙，七預反，伺也。」徐廣曰：「伺候也，音千恕反。」《索隱》：按：應劭云「狙，伺也。」一曰狙，伏伺也，音七豫反。謂狙之伺物，必伏而候之，故今雲「狙候」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：服虔云「地在陽武南」。按：今浚儀西北四十里有博浪城。《正義》：晉地理記云「鄭陽武縣有博浪沙」。按：今當官道也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：按：《漢官儀》天子屬車三十六乘。屬車卽副車，而奉車郎御而從後。</span></p><p><br/>　　良嘗閒從容<span class="inlay1"><span class="number">一</span></span>步游下邳<span class="inlay1"><span class="number">二</span></span>圯上，<span class="inlay1"><span class="number">三</span></span>有一老父，衣褐，至良所，直墮其履圯下，<span class="inlay1"><span class="number">四</span></span>顧謂良曰：「孺子，下取履！」良鄂然，欲毆之。<span class="inlay1"><span class="number">五</span></span>為其老，彊忍，下取履。父曰：「履我！」良業為取履，因長跪履之。<span class="inlay1"><span class="number">六</span></span>父以足受，笑而去。良殊大驚，隨目之。父去里所，復還，<span class="inlay1"><span class="number">七</span></span>曰：「孺子可敎矣。後五日平明，與我會此。」良因怪之，跪曰：「諾。」五日平明，良往。父已先在，怒曰：「與老人期，後，何也？」去，曰：「後五日早會。」五日雞鳴，良往。父又先在，復怒曰：「後，何也？」去，曰：「後五日復早來。」五日，良夜未半往。有頃，父亦來，喜曰：「當如是。」出一編書，<span class="inlay1"><span class="number">八</span></span>曰：「讀此則為王者師矣。後十年興。十三年孺子見我濟北，穀城山下黃石卽我矣。」<span class="inlay1"><span class="number">九</span></span>遂去，無他言，不復見。旦日視其書，乃《太公兵法》也。<span class="inlay1"><span class="number">一〇</span></span>良因異之，常習誦讀之。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：嘗訓經也。閒，閒字也。從容，閒暇也。從容謂從任其容止，不矜莊也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：邳，被眉反。按：《地理志》下邳縣屬東海。又云邳在薛，後徙此。有上邳，故此曰下邳也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「圯，橋也，東楚謂之圯。音怡。」《索隱》：李奇云「下邳人謂橋為圯，音怡」。文穎曰「沂水上橋也」。應劭云「沂水之上也」。姚察見《史記》本有作土旁者，乃引今會稽東湖大橋名為靈圯。圯亦音夷，理或然也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：崔浩云「直猶故也」，亦恐不然。直言正也，謂至良所正墮其履也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「一云『良怒，欲罵之』。」《索隱》：毆音烏後反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：業猶本先也。謂良心先已為取履，故遂跪而履之。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《集解》：徐廣曰：「一曰『為其老，強忍，下取履，因進之。父以足受，笑而去。良殊大驚。父去里所，複還』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：徐廣曰：「編，一作『篇』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《正義》：《括地志》云：「穀城山一名黃山，在濟州東阿縣東。濟州，故濟北郡。孔文祥云『黃石公\〔狀〕，鬚眉皆白，（狀）杖丹黎，履赤舄』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《正義》：《七錄》云：「《太公兵法》一袠三卷。太公，姜子牙，周文王師，封齊侯也。」</span></p><p>　　居下邳，為任俠。項伯常殺人，從良匿。</p><p><br/>　　後十年，陳涉等起兵，良亦聚少年百餘人。景駒自立為楚假王，在留。良欲往從之，道遇沛公。沛公將數千人，略地下邳西，遂屬焉。沛公拜良為廄將。<span class="inlay1"><span class="number">一</span></span>良數以《太公兵法》說沛公，沛公善之，常用其策。良為他人言，皆不省。良曰：「沛公殆天授。」<span class="inlay1"><span class="number">二</span></span>故遂從之，不去見景駒。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：《漢書音義》曰：「官名。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：殆訓近也。</span></p><p>　　及沛公之薛，見項梁。項梁立楚懷王。良乃說項梁曰：「君已立楚後，而韓諸公子橫陽君成賢，<span class="inlay1">孫守真按：橫陽君成四字高清版有私名號，此蓋有損壓縮，失真已。先抓動詞·找對詞性：賢。</span>可立為王，益樹黨。」項梁使良求韓成，立以為韓王。以良為韓申徒，<span class="inlay1"><span class="number">一</span></span>與韓王將千餘人西略韓地，得數城，秦輒復取之，往來為游兵潁川。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「卽司徒耳，但語音訛轉，故字亦隨改。」</span></p><p>　　沛公之從雒陽南出轘轅，<span class="inlay1">孫守真按：轘轅ㄏㄨㄢˋ ㄩㄢˊ，即軒轅。此軒=轘，猶馮諼為馮驩也。字形結構兼音義·聲音變化的關係·聲調不重要。</span>良引兵從沛公，下韓十餘城，擊破楊熊軍。沛公乃令韓王成留守陽翟，與良俱南，攻下宛，西入武關。沛公欲以兵二萬人擊秦嶢下軍，<span class="inlay1"><span class="number">一</span></span>良說曰：「秦兵尚彊，未可輕。臣聞其將屠者子，賈豎易動以利。願沛公且留壁，使人先行，為五萬人具食，<span class="inlay1"><span class="number">二</span></span>益為張旗幟<span class="inlay1"><span class="number">三</span></span>諸山上，為疑兵，令酈食其持重寶啗秦將。」秦將果畔，欲連和俱西襲咸陽，<span class="inlay1">孫守真按：和，單字想複詞：和約、和諧、我和你的和。</span>沛公欲聽之。良曰：「此獨其將欲叛耳，恐士卒不從。不從必危，不如因其解<span class="inlay1"><span class="number">四</span></span>擊之。」沛公乃引兵擊秦軍，大破之。（遂）\〔逐〕北至藍田，再戰，秦兵竟敗。遂至咸陽，秦王子嬰降沛公。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「嶢音堯。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「五，一作『百』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：音其試二音。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：謂卒將離心而懈怠。</span></p><p>　　沛公入秦宮，宮室帷帳狗馬重寶婦女以千數，意欲留居之。樊噲諫沛公出舍，沛公不聽。<span class="inlay1"><span class="number">一</span></span>良曰：「夫秦為無道，故沛公得至此。夫為天下除殘賊，宜縞素為資。<span class="inlay1"><span class="number">二</span></span>今始入秦，卽安其樂，此所謂『助桀為虐』。且『忠言逆耳利於行，毒藥苦口利於病』，<span class="inlay1"><span class="number">三</span></span>願沛公聽樊噲言。」沛公乃還軍霸上。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「一本『噲諫曰：「沛公欲有天下邪？將欲為富家翁邪？」沛公曰：「吾欲有天下。」噲曰：「今臣從入秦宮，所觀宮室帷帳珠玉重寶鍾鼓之飾，奇物不可勝極，入其後宮，美人婦女以千數，此皆秦所以亡天下也。願沛公急還霸上，無留宮中。」沛公不聽』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：晉灼曰：「資，藉也。欲沛公反秦奢泰，服儉素以為藉也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：按：此語見《孔子家語》。</span></p><p>　　項羽至鴻門下，欲擊沛公，項伯乃夜馳入沛公軍，私見張良，欲與俱去。良曰：「臣為韓王送沛公，今事有急，亡去不義。」乃具以語沛公。沛公大驚，曰：「為將柰何？」良曰：「沛公誠欲倍項羽邪？」沛公曰：「鯫生<span class="inlay1"><span class="number">一</span></span>敎我距關無內諸侯，秦地可盡王，故聽之。」良曰：「沛公自度能卻項羽乎？」沛公默然良久，曰：「固不能也。今為柰何？」良乃固要項伯。項伯見沛公。沛公與飲為壽，結賓婚。令項伯具言沛公不敢倍項羽，所以距關者，備他盜也。及見項羽後解，語在《項羽》事中。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「呂靜曰鯫，魚也，音此垢反。」《索隱》：呂靜云「鯫，魚也，謂小魚也，音此垢反」。臣瓚按：《楚漢春秋》鯫生本姓（解）\〔鯫〕。</span></p><p><br/>　　漢元年正月，沛公為漢王，王巴蜀。漢王賜良金百溢，珠二斗，良具以獻項伯。漢王亦因令良厚遺項伯，使請漢中地。<span class="inlay1"><span class="number">一</span></span>項王乃許之，遂得漢中地。漢王之國，良送至襃中，<span class="inlay1"><span class="number">二</span></span>遣良歸韓。良因說漢王曰：「王何不燒絕所過棧道，示天下無還心，以固項王意。」乃使良還。行，燒絕棧道。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：如淳曰：「本但與巴蜀，故請漢中地。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：《括地志》云：「褒谷在梁州褒城縣北五十里南中山。昔秦欲伐蜀，路無由入，乃刻石為牛五頭，置金於後，偽言此牛能屎金，以遺蜀。蜀侯貪，信之，乃令五丁共引牛，塹山堙谷，致之成都。秦遂尋道伐之，因號曰石牛道。《蜀賦》以石門在漢中之西，襃中之北是。」又云：「斜水源出褒城縣西北衙嶺山，與襃水同源而流派。《漢書·溝洫志》云襃水通沔，斜水通渭，皆以行船。」</span></p><p>　　良至韓，韓王成以良從漢王故，項王不遣成之國，從與俱東。良說項王曰：「漢王燒絕棧道，無還心矣。」乃以齊王田榮反，書告項王。項王以此無西憂漢心，而發兵北擊齊。</p><p>　　項王竟不肯遣韓王，乃以為侯，又殺之彭城。良亡，閒行歸漢王，漢王亦已還定三秦矣。復以良為成信侯，從東擊楚。至彭城，漢敗而還。至下邑，漢王下馬踞鞍而問曰：「吾欲捐關以東等弃之，誰可與共功者？」良進曰：「九江王黥布，楚梟將，與項王有郄；彭越與齊王田榮反梁地：此兩人可急使。而漢王之將獨韓信可屬大事，當一面。卽欲捐之，捐之此三人，則楚可破也。」漢王乃遣隨何說九江王布，而使人連彭越。及魏王豹反，使韓信將兵擊之，因舉燕、代、齊、趙。然卒破楚者，此三人力也。</p><p>　　張良多病，未嘗特將也，<span class="inlay1">孫守真按：獨特，同義複詞。</span>常為畫策臣，時時從漢王。<span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：晉灼曰：「鼓行而西，言無所畏也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：司馬彪曰：「長安縣東有曲郵聚。」《索隱》：郵音尤。按：司馬彪《漢書》郡國志長安有曲郵聚。今在新豐西，俗謂之郵頭。《漢書》舊儀云「五里一郵，郵人居閒，相去二里半」。按：郵乃今之候也。</span></p><p>　　漢十二年，上從擊破布軍歸，疾益甚，愈欲易太子。留侯諫，不聽，因疾不視事。叔孫太傅稱說引古今，以死爭太子。上詳許之，猶欲易之。及燕，置酒，太子侍。四人從太子，年皆八十有餘，鬚眉皓白，衣冠甚偉。上怪之，問曰：「彼何為者？」四人前對，各言名姓，曰東園公，角里先生，綺里季，夏黃公。上乃大驚，曰：「吾求公數歲，公闢逃我，今公何自從吾兒游乎？」四人皆曰：「陛下輕士善罵，臣等義不受辱，故恐而亡匿。竊聞太子為人仁孝，恭敬愛士，天下莫不延頸欲為太子死者，故臣等來耳。」上曰：「煩公幸卒調護太子。」<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：如淳曰：「調護猶營護也。」</span></p><p>　　四人為壽已畢，趨去。上目送之，召戚夫人指示四人者曰：「我欲易之，彼四人輔之，羽翼已成，難動矣。呂后眞而主矣。」戚夫人泣，上曰：「為我楚舞，吾為若楚歌。」歌曰：「鴻鵠高飛，一舉千里。羽翮已就，橫絕四海。橫絕四海，當可柰何！雖有矰繳，<span class="inlay1"><span class="number">一</span></span>尚安所施！」歌數闋，<span class="inlay1"><span class="number">二</span></span>戚夫人噓唏流涕，上起去，罷酒。竟不易太子者，留侯本招此四人之力也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：韋昭曰：「繳，弋射也。其矢曰矰。」《索隱》：馬融注《周禮》云：「矰者，繳繫短矢謂之矰。」一說云矰，一弦，可以仰高射，故云矰也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：音曲穴反，謂曲終也。《說文》曰：「闋，事\〔已閉門〕也。」</span></p><p><br/>　　留侯從上擊代，出奇計馬邑下，<span class="inlay1"><span class="number">一</span></span>及立蕭何相國，<span class="inlay1"><span class="number">二</span></span>所與上從容言天下事甚衆，非天下所以存亡，故不著。留侯乃稱曰：「家世相韓，及韓滅，不愛萬金之資，為韓報讎彊秦，天下振動。今以三寸舌<span class="inlay1"><span class="number">三</span></span>為帝者師，封萬戶，位列侯，此布衣之極，於良足矣。願弃人閒事，欲從赤松子<span class="inlay1"><span class="number">四</span></span>游耳。」乃學辟<span class="inlay1"><span class="number">五</span></span>穀，道引輕身。<span class="inlay1"><span class="number">六</span></span>會高帝崩，呂后德留侯，乃彊食之，曰：「人生一世閒，如白駒過隙，何至自苦如此乎！」留侯不得已，彊聽而食。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「一云『出奇計下馬邑』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：《漢書音義》曰：「何時未為相國，良勸高祖立之。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：《春秋》緯云：「舌在口，長三寸，象鬥玉衡。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：《列仙傳》：「神農時雨師也，能入火自燒，昆崙山上隨風雨上下也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：賓亦反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「一云『乃學道引，欲輕舉』也。」</span></p><p>　　後八年卒，謚為文成侯。子不疑代侯。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「文成侯立十六年卒，子不疑代立。十年，坐與門大夫吉謀殺故楚內史，當死，贖為城旦，國除。」</span></p><p>　　子房始所見下邳圯上老父與太公書者，後十三年從高帝過濟北，果見穀城山下黃石，取而葆祠之。<span class="inlay1"><span class="number">一</span></span>留侯死，幷葬黃石（冢）。<span class="inlay1"><span class="number">二</span></span>每上冢伏臘，祠黃石。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「《史記》珍寶字皆作『葆』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：《括地志》云：「漢張良墓在徐州沛縣東六十五里，與留城相近也。」</span></p><p>　　留侯不疑，孝文帝五年坐不敬，國除。</p><p><br/>　　太史公曰：學者多言無鬼神，然言有物。<span class="inlay1"><span class="number">一</span>孫守真按：怪物，同義複詞，故下文抽換字面用「怪」。</span>至如留侯所見老父予書，亦可怪矣。<span class="inlay1"><span class="number">二</span></span>高祖離困者數矣，而留侯常有功力焉，豈可謂非天乎？上曰：「夫運籌筴帷帳之中，決勝千里外，吾不如子房。」余以為其人計魁梧奇偉，<span class="inlay1"><span class="number">三</span></span>至見其圖，狀貌如婦人好女。蓋孔子曰：「以貌取人，失之子羽。」<span class="inlay1"><span class="number">四</span></span>留侯亦云。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：物謂精怪及藥物也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：詩緯云「風後，黃帝師，又化為《老子》，以書授張良」。亦異說。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：應劭曰：「魁梧，丘虛壯大之意。」《索隱》：蘇林云「梧音忤」。蕭該云「今讀為吾，非也」。小顏云「言其可驚悟」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：子羽，澹台滅明字也。《仲尼弟子傳》云「狀貌甚惡」。又《韓子》云「子羽有君子之容，而行不稱其貌」，與《史記》文相反。</span></p><p><span class="inlay2">　【《索隱》述贊】留侯倜儻，志懷憤惋。五代相韓，一朝歸漢。進履宜假，運籌神算。橫陽旣立，申徒作捍。灞上扶危，固陵靜亂。人稱三傑，辯推八難。赤松願游，白駒難絆。嗟彼雄略，曾非魁岸。</span></p>
