<h2>史記集解序</h2><p><span class="inlay2">　裴駰<span class="inlay1"><span class="number">一</span></span></span></p><p>　　班固有言曰：<span class="inlay1"><span class="number">二</span></span>「司馬遷<span class="inlay1"><span class="number">三</span></span>據左氏、《國語》，<span class="inlay1"><span class="number">四</span></span>採《世本》、戰《國策》，<span class="inlay1"><span class="number">五</span></span>述楚漢《春秋》，<span class="inlay1"><span class="number">六</span></span>接其後事，訖於天漢。<span class="inlay1"><span class="number">七</span></span>其言秦漢詳矣。至於採經摭傳，<span class="inlay1"><span class="number">八</span></span>分散數家之事，甚多疏略，或有抵捂。<span class="inlay1"><span class="number">九</span></span>亦其所涉獵者廣博，貫穿經傳，馳騁古今上下數千載閒，斯已勤矣。<span class="inlay1"><span class="number">一〇</span></span>又其是非頗謬於聖人，<span class="inlay1"><span class="number">一一</span></span>論大道則先黃老而後六經，<span class="inlay1"><span class="number">一二</span></span>序游俠則退處士而進奸雄，<span class="inlay1"><span class="number">一三</span></span>述貨殖<span class="inlay1"><span class="number">一四</span></span>則崇勢利<span class="inlay1"><span class="number">一五</span></span>而羞貧賤：此其所蔽也。<span class="inlay1"><span class="number">一六</span></span>然自劉向、楊雄博極羣書，皆稱遷有良史之才，服其善序事理，辯而不華，質而不俚，<span class="inlay1"><span class="number">一七</span></span>其文直，其事核，不虛美，不隱惡，故謂之實錄。」駰以為固之所言，世稱其當。<span class="inlay1"><span class="number">一八</span></span>雖時有紕繆，<span class="inlay1"><span class="number">一九</span></span>實勒成一家，<span class="inlay1"><span class="number">二〇</span></span>總其大較，<span class="inlay1"><span class="number">二一</span></span>信命世之宏才也。<span class="inlay1"><span class="number">二二</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：駰字龍駒，河東人，宋中郎外兵參軍。父松之，太中大夫。《正義》：裴駰採九經諸史並《漢書》音義及衆書之目而解《史記》，故題《史記》《集解》：序。序，緒也。孫炎云，謂端緒也。孔子作易《序卦》，子夏作詩序，序之義其來尚矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：固撰《漢書》，作司馬遷傳，評其作《史記》所採之書，兼論其得失，故裴駰此序先引之為說也。案：固字孟堅，扶風人，後漢明帝時仕至中護軍。祖穉，廣川太守。父彪，徐令，續太史公書也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：字子長，左馮翊人也，漢武帝時為太史令，撰《史記》百三十篇。父談，亦為太史令。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：仲尼作《春秋》經，魯史左丘明作傳，合三十篇，故曰《左氏傳》。《國語》亦丘明所撰。上起周穆王，下訖敬王。其諸侯之事，起魯莊公迄《春秋》末，凡二十一篇。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：劉向云：「《世本》，古史官明於古事者之所記也。錄黃帝已來帝王諸侯及卿大夫系謚名號，凡十五篇也。」《戰國策》，高誘云六國時縱橫之說也，一曰短長書，亦曰國事，劉向撰為三十三篇，名曰《戰國策》。案：此是班固取其後名而書之，非遷時已名《戰國策》也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：漢太中大夫楚人陸賈所撰，記項氏與漢高祖初起及說惠文閒事。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：武帝年號。言太史公所記迄至武帝天漢之年也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：案字書，摭，拾也，音之赤反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：抵音丁禮反。梧音吾故反。抵者，觸也。捂亦斜相抵觸之名。案：今屋梁上斜柱曰「柱捂」是也。直觸橫觸皆曰抵，斜觸謂之捂，下觸謂之捂。抵捂，言其參差也。以言彼此二文同出一家，而自相乖舛也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《正義》：言作《史記》採經傳百家之事上下二千餘年，此其甚勤於撰錄也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：聖人謂周公、孔子也。言周孔之教皆宗儒尚德，今太史公乃先黃老，崇勢利，是謬於聖人也。《正義》：太史公《史記》各顯六家之宗，黃老道家之宗，六經儒家之首，序游俠則退處士，述貨殖則崇勢利，處士賤貧，原憲非病。夫作史之體，務涉多時，有國之規，備陳臧否，天人地理咸使該通，而遷天縱之才，述作無滯，故異周孔之道。班固詆之，裴駰引序，亦通人之蔽也。而固作《漢書》，與《史記》同者五十餘卷，謹寫《史記》，少加異者，不弱卽劣，何更非剝《史記》，乃是後士妄非前賢。又《史記》五十二萬六千五百言，敘二千四百一十三年事，《漢書》八十一萬言，敘二百二十五年事；司馬遷引父致意，班固父修而蔽之，優劣可知矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《正義》：大道者，皆稟乎自然，不可稱道也。道在天地之前，先天地生，不知其名，字之曰「道」。黃帝、《老子》遵崇斯道，故太史公論大道，須先黃老而後六經。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《索隱》：游俠，謂輕死重氣，如荊軻、豫讓之輩也。游，從也，行也。俠，挾也，持也。言能相從游行挾持之事，又曰，同是非曰俠也。《正義》：奸雄，奸猾雄豪之人。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《正義》：殖，生也。言貨物滋生也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《正義》：趨利之人。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一六</span></span>《正義》：此三者是司馬遷不達理也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一七</span></span>《索隱》：俚音里。劉德曰「俚卽鄙也」，崔浩云「世有鄙俚之語」，則俚亦野也，俗也。不俚，謂詞不鄙樸也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一八</span></span>《正義》：駰音因。當音丁浪反。裴駰以班固所論司馬遷《史記》是非，世人稱班固之言。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一九</span></span>《索隱》：紕音匹之反。紕猶錯也。亦作「悂」。字書云織者兩絲同齒曰悂。繆亦與「謬」同。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二〇</span></span>《正義》：雖有小紕繆，實編勒成一家之書矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二一</span></span>《索隱》：較音角。較猶略也，則大較猶言大略也。《正義》：較猶明也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二二</span></span>《索隱》：案：《孟子》云「五百年之閒必有名世者」。趙岐曰「名世，次聖之才，物來能名，正一世者，生於聖人之閒也」。此言命者名也，言賢人有名於世也。宏才，大才，謂史遷也。</span></p><p>　　考較此書，文句不同，有多有少，莫辯其實，而世之惑者，定彼從此，是非相貿，眞偽舛雜。<span class="inlay1"><span class="number">一</span></span>故中散大夫東莞徐廣研核衆本，為作音義，<span class="inlay1"><span class="number">二</span></span>具列異同，兼述訓解，<span class="inlay1"><span class="number">三</span></span>粗有所發明，而殊恨省略。<span class="inlay1"><span class="number">四</span></span>聊以愚管，<span class="inlay1"><span class="number">五</span></span>增演徐氏。<span class="inlay1"><span class="number">六</span></span>採經傳百家並先儒之說，<span class="inlay1"><span class="number">七</span></span>豫是有益，悉皆抄內。<span class="inlay1"><span class="number">八</span></span>刪其游辭，取其要實<span class="inlay1"><span class="number">九</span></span>，或義在可疑，則數家兼列。<span class="inlay1"><span class="number">一〇</span></span>《漢書》音義稱「臣瓚」者，莫知氏姓，<span class="inlay1"><span class="number">一一</span></span>今直云「瓚曰」。又都無姓名者，但云「《漢書》音義」。<span class="inlay1"><span class="number">一二</span></span>時見微意，有所裨補。<span class="inlay1"><span class="number">一三</span></span>譬嘒星之繼朝陽，<span class="inlay1"><span class="number">一四</span></span>飛塵之集華嶽。<span class="inlay1"><span class="number">一五</span></span>以徐為本，<span class="inlay1"><span class="number">一六</span></span>號曰《集解》：。未詳則闕，弗敢臆說。<span class="inlay1"><span class="number">一</span></span>人心不同，<span class="inlay1"><span class="number">一八</span></span>聞見異辭，<span class="inlay1"><span class="number">一九</span></span>班氏所謂「疏略抵捂」者，依違不悉辯也。<span class="inlay1"><span class="number">二〇</span></span>愧非胥臣之多聞<span class="inlay1"><span class="number">二一</span></span>，子產之博物，<span class="inlay1"><span class="number">二二</span></span>妄言末學，蕪穢舊史，豈足以關諸畜德，庶賢無所用心而已。<span class="inlay1"><span class="number">二三</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：貿音茂。舛音昌轉反。言世之迷惑淺識之人，或定彼從此，本更相貿易，眞偽雜亂，不能辯其是非。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：作音義十三卷，裴駰為注，散入百三十篇。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：徐作音義，具列異同之本，兼述訓解釋也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：殊，絕也。《左傳》曰「斬其木不殊」，言絕恨其所撰大省略也。《正義》：省音山景反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：案：東方朔雲「以管窺天，以蠡測海」，皆喻小也。然此語本出《莊子》文，今雲「愚管」者，是駰謙言己愚陋管見，所識不能遠大也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：演音羊善反。增，益也。言裴駰更增益演徐氏之說。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《正義》：採，取也。或取傳說，採諸子百家，兼取先儒之義。先儒謂孔安國、鄭玄、服虔、賈逵等是也。言百家，廣其非一。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《正義》：並採經傳之說，有裨益《史記》，盡抄內其中。抄音楚交反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《正義》：刪音師顏反。刪，除也。去經傳諸家浮游之辭，取其精要之實。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《正義》：數家之說不同，各有道理，致生疑惑，不敢偏棄，故皆兼列。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：案：卽傅瓚，而劉孝標以為於瓚，非也。據何法盛《晉書》，於瓚以穆帝時為大將軍，誅死，不言有注《漢書》之事。又其注《漢書》有引祿秩令及茂陵書，然彼二書亡於西晉，非於所見也。必知是傅瓚者，案：穆天子傳目錄云傅瓚為校書郎，與荀勖同校定穆天子傳，卽當西晉之朝，在於之前，尚見茂陵等書。又稱「臣」者，以其職典秘書故也。瓚音殘岸反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《正義》：《漢書》音義中有全無姓名者，裴氏注《史記》直云「《漢書》音義」。案：大顏以為無名義，今有六卷，題云孟康，或云服虔，蓋後所加，皆非其實，未詳指歸也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《正義》：見音賢見反。裨音卑，又音頻移反。裨，益也。裴氏云時見己之微意，亦有所補益也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《索隱》：嘒，微小貌也。詩云：「嘒彼小星，三五在東。」言衆無名微小之星，各隨三心五噣出在東方，亦能繼朝陽之光。嘒音火慧反。朝陽，日也。嘒星繼朝陽，喻己淺薄而注《史記》也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《正義》：西嶽華山極高大。裴氏自喻才藻輕小，如飛塵之集華嶽，亦能成其高大。《管子》云：「海不辭水，故能成其大；山不辭土，故能成其高。」華音胡化反，又如字。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一六</span></span>《正義》：徐廣音義辨諸家異同，故以徐為本也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一七</span></span>《正義》：有未詳審之處則闕而不論，不敢以胸臆之中而妄解說也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一八</span></span>《正義》：言人心旣不同，所見亦殊別也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一九</span></span>《正義》：耳聞目見，心意旣乖，其辭所以各異也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二〇</span></span>《索隱》：裴氏言今或依違，不敢複更辯明之也。案：周公世家敘傳曰「依之違之，周公綏之」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二一</span></span>《索隱》：晉大夫臼季名曰胥臣。案：《國語》稱晉文公使趙衰為卿，辭曰：「欒枝貞慎，先軫有謀，胥臣多聞，皆可以為輔。」又胥臣對文公黃帝二十五子及屯豫皆八等事，是多聞也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二二</span></span>《索隱》：鄭卿公孫僑字子產。案：《左氏傳》子產聘晉，言晉侯之疾非實沈、台駘之祟，乃說飲食哀樂及內官不及同姓，則能生疾。晉侯聞子產之言，曰「博物君子也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二三</span></span>《索隱》：關，預也。畜德，謂積德多學之人也。裴氏謙言己今此《集解》：豈足關預於積學多識之士乎！正是冀望聖賢，勝於飽食終日，無所用心，愈於《論語》「不有博弈者乎」之人耳。</span></p><h2>史記索隱序</h2><p><span class="inlay2">　朝散大夫國子博士弘文館學士河內司馬貞</span></p><p>　　《史記》者，漢太史司馬遷父子之所述也。遷自以承五百之運，繼《春秋》而纂是史，其褒貶覈實頗亞於丘明之書，於是上始軒轅，下訖天漢，作十二本紀，十表，八書，三十系家，七十列傳，凡一百三十篇，始變《左氏》之體，而年載悠邈，簡冊闕遺，勒成一家，其勤至矣。又其屬稾先據《左氏》、《國語》、《系本》、《戰國策》、《楚漢春秋》及諸子百家之書，而後貫穿經傳，馳騁古今，錯綜櫽括，各使成一國一家之事，故其意難究詳矣。比於班書，微為古質，故漢晉名賢未知見重，所以魏文侯聽古樂則唯恐臥，良有以也。</p><p>　　逮至晉末，有中散大夫東莞徐廣始考異同，作音義十三卷。宋外兵參軍裴駰又取經傳訓釋作《集解》：，合為八十卷。雖粗見微意，而未窮討論。南齊輕車錄事鄒誕生亦作音義三卷，音則微殊，義乃更略。爾後其學中廢。貞觀中，諫議大夫崇賢館學士劉伯莊達學宏才，鉤深探賾，又作《音義》二十卷，比於徐鄒，音則具矣。殘文錯節，異音微義，雖知獨善，不見旁通，欲使後人從何準的。</p><p>　　貞謏聞陋識，頗事鑽研，而家傳是書，不敢失墜。初欲改更舛錯，裨補疏遺，義有未通，兼重注述。然以此書殘缺雖多，實為古史，忽加穿鑿，難允物情。今止探求異聞，採摭典故，解其所未解，申其所未申者，釋文演注，又重為述贊，凡三十卷，號曰《史記》《索隱》：。雖未敢藏之書府，亦欲以貽厥孫謀云。</p><h2>史記索隱後序</h2><p>　　夫太史公紀事，上始軒轅，下訖天漢，雖博採古文及傳記諸子，其閒殘闕蓋多，或旁搜異聞以成其說，然其人好奇而詞省，故事核而文微，是以後之學者多所未究。其班氏之書，成於後漢。彪旣後遷而述，所以條流更明，是兼採衆賢，羣理畢備，故其旨富，其詞文，是以近代諸儒共行鑽仰。其訓詁蓋亦多門，蔡謨《集解》：之時已有二十四家之說，所以於文無所滯，於理無所遺。而太史公之書，旣上序軒黃，中述戰國，或得之於名山壞壁，或取之以舊俗風謠，故其殘文斷句難究詳矣。</p><p>　　然古今為注解者絕省，音義亦希。始後漢延篤乃有《音義》一卷，又別有章隱五卷，不記作者何人，近代鮮有二家之本。宋中散大夫徐廣作《音義》十三卷，唯記諸家本異同，於義少有解釋。又中兵郎裴駰，亦名家之子也，作《集解》：注本，合為八十卷，見行於代。仍云亦有《音義》，前代久已散亡。南齊輕車錄事鄒誕生亦撰《音義》三卷，音則尚奇，義則罕說。隋秘書監柳顧言尤善此史。劉伯莊云，其先人曾從彼公受業，或音解隨而記錄，凡三十卷。隋季喪亂，遂失此書。伯莊以貞觀之初，奉敕於弘文館講授，遂採鄒徐二說，兼記憶柳公音旨，遂作《音義》二十卷。音乃周備，義則更略，惜哉！古史微文遂由數賢秘寶，故其學殆絕。</p><p>　　前朝吏部侍郎許子儒亦作注義，不睹其書。崇文館學士張嘉會獨善此書，而無注義。貞少從張學，晚更研尋，初以殘闕處多，兼鄙褚少孫誣謬，因憤發而補《史記》，遂兼注之，然其功殆半。乃自唯曰：「千載古史，良難閒然。」因退撰《音義》，重作贊述，蓋欲以剖盤根之錯節，遵北轅於司南也。凡為三十卷，號曰《史記索隱》云。</p><h2>史記正義序</h2><p><span class="inlay2">　諸王侍讀宣議郎守右淸道率府長史張守節上</span></p><p>　　《史記》者，漢太史公司馬遷作。遷生龍門，耕牧河山之陽，南游江淮，講學齊魯之郡，紹太史，繼《春秋》，括文魯史而包《左氏》、《國語》，採《世本》、《戰國策》而摭《楚漢春秋》，貫紬經傳，旁搜史子，上起軒轅，下旣天漢。作十二本紀，帝王興廢悉詳；三十世家，君國存亡畢著；八書，贊陰陽禮樂；十表，定代系年封；七十列傳，忠臣孝子之誠備矣。筆削冠於史籍，題目足以經邦。裴駰服其善序事理，辯而不華，質而不俚，其文直，其事核，不虛美，不隱惡，故謂之實錄。自劉向、楊雄皆稱良史之才。況墳典湮滅，簡冊闕遺，比之《春秋》，言辭古質，方之兩漢，文省理幽。</p><p>　　守節涉學三十餘年，六籍九流地里蒼雅銳心觀採，評史漢詮衆訓釋而作《正義》，郡國城邑委曲申明，古典幽微竊探其美，索理允愜，次舊書之旨，兼音解注，引致旁通，凡成三十卷，名曰《史記正義》：。發揮膏肓之辭，思濟滄溟之海，未敢侔諸秘府，冀訓詁而齊流，庶貽厥子孫，世疇茲史。</p><p>　　于時歲次丙子，開元二十四年八月，殺靑斯竟。</p><h2>史記正義</h2><p><span class="inlay2">　諸王侍讀宣議郎守右淸道率府長史張守節上</span></p><h2>史記正義·論史例</h2><p>　　古者帝王右《史記》言，左《史記》事，言為《尚書》，事為《春秋》。太史公兼之，故名曰《史記》。並採六家雜說以成一史，備論君臣父子夫妻長幼之序，天地山川國邑名號殊俗物類之品也。</p><p>　　太史公作《史記》，起黃帝、高陽、高辛、康堯、虞舜、夏、殷、周、秦，訖於漢武帝天漢四年，合二千四百一十三年。作本紀十二，象歲十二月也。作表十，象天之剛柔十日，以記封建世代終始也。作書八，象一歲八節，以記天地日月山川禮樂也。作世家三十，象一月三十日，三十輻共一轂，以記世祿之家輔弼股肱之臣忠孝得失也。作列傳七十，象一行七十二日，言七十者舉全數也。餘二日象閏餘也，以記王侯將相英賢略立功名於天下，可序列也。合百三十篇，象一歲十二月及閏餘也。而太史公作此五品，廢一不可，以統理天地，勸獎箴誡，為後之楷模也。</p><h2>史記正義·論注例</h2><p>　　《史記》文與《古文尚書》同者，則取孔安國注。若與伏生《尚書》同者，則用鄭玄、王肅、馬融所釋。與三傳同者，取杜元凱、服虔、何休、賈逵、范寧等注。與三禮、《論語》、孝經同者，則取鄭玄、馬融、王肅之注。與《韓詩》同者，則取《毛傳》、鄭箋等釋。與《周易》同者，則依王氏之注。與諸子諸史雜書及先儒解釋善者，而裴駰並引為注。又徐中散作音訓，校集諸本異同，或義理可通者，稱「一本云」「又一本云」，自是別記異文，裴氏亦引之為注。</p><h2>史記正義·論字例</h2><p>　　《史》《漢》文字相承已久，若「悅」字作「說」，「閑」字作「閒」，「智」字作「知」，「汝」字作「女」，「早」字作「蚤」，「後」字作「后」，「旣」字作「漑」，「勑」字作「飭」，「制」字作「剬」，此之般流，緣古少字通共用之。《史》《漢》本有此古字者，乃為好本。程邈變篆為隸，楷則有常，後代作文，隨時改易。衞宏官書數體，呂忱或字多奇，鍾、王等家以能為法，致令楷文改變，非復一端，咸著祕書，傳之歷代。又字體乖日久，其「黼黻」之字法從「黹」，<span class="inlay2">七低反，切帝反。</span>費<span class="inlay2">非味反，用也。又音「祕」，邑也。扶味反，姓也。</span></p><h2>史記正義·謚法解</h2><p>　　惟周公旦、太公望開嗣王業，建功於牧野，終將葬，乃制謚，遂敘謚法。謚者，行之跡；號者，功之表；<span class="inlay2">古者有大功，則賜之善號以為稱也。</span>車服者，位之章也。是以大行受大名，細行受細名。行出於己，名生於人。<span class="inlay2">名謂號謚。</span></p><p>　　　民無能名曰神。<span class="inlay2">不名一善。</span><br/>　　　靖民則法曰皇。<span class="inlay2">靖安。</span><br/>　　　德象天地曰帝。<span class="inlay2">同於天地。</span><br/>　　　仁義所往曰王。<span class="inlay2">民往歸之。</span><br/>　　　立志及衆曰公。<span class="inlay2">志無私也。</span><br/>　　　執應八方曰侯。<span class="inlay2">所執行八方應之。</span><br/>　　　賞慶刑威曰君。<span class="inlay2">能行四者。</span><br/>　　　從之成羣曰君。<span class="inlay2">民從之。</span><br/>　　　揚善賦簡曰聖。所稱得人，所善得實，所賦得簡。<br/>　　　敬賓厚禮曰聖。厚於禮。<br/>　　　照臨四方曰明。以明照之。<br/>　　　譖訴不行曰明。逆知之，故不行。<br/>　　　經緯天地曰文。成其道。<br/>　　　道德博聞曰文。無不知。<br/>　　　學勤好問曰文。不恥下問。<br/>　　　慈惠愛民曰文。惠以成政。<br/>　　　愍民惠禮曰文。惠而有禮。<br/>　　　賜民爵位曰文。與同升。<br/>　　　綏柔士民曰德。安民以居，安士以事。<br/>　　　諫爭不威曰德。不以威拒諫。<br/>　　　剛強直理曰武。剛無欲，強不屈。懷忠恕，正曲直。<br/>　　　威強敵德曰武。與有德者敵。<br/>　　　克定禍亂曰武。以兵征，故能定。</p><p>　　　刑民克服曰武。法以正民，能使服。<br/>　　　誇志多窮曰武。大志行兵，多所窮極。<br/>　　　安民立政曰成。政以安定。<br/>　　　淵源流通曰康。性無忌。<br/>　　　溫柔好樂曰康。好豐年，勤民事。<br/>　　　安樂撫民曰康。無四方之虞。<br/>　　　合民安樂曰康。富而教之。<br/>　　　布德執義曰穆。故穆穆。<br/>　　　中情見貌曰穆。性公露。<br/>　　　容儀恭美曰昭。有儀可象，行恭可美。<br/>　　　昭德有勞曰昭。能勞謙。<br/>　　　聖聞周達曰昭。聖聖通合。<br/>　　　治而無眚曰平。無災罪也。<br/>　　　執事有制曰平。不任意。<br/>　　　布綱治紀曰平。施之政事。</p><p>　　　由義而濟曰景。用義而成。<br/>　　　耆意大慮曰景。耆，強也。<br/>　　　布義行剛曰景。以剛行義。<br/>　　　淸白守節曰貞。行淸白執志固。<br/>　　　大慮克就曰貞。能大慮非正而何。<br/>　　　不隱無屈曰貞。坦然無私。<br/>　　　闢土服遠曰桓。以武正定。<br/>　　　克敬動民曰桓。敬以使之。<br/>　　　闢土兼國曰桓。兼人故啟土。<br/>　　　能思辯衆曰元。別之，使各有次。<br/>　　　行義說民曰元。民說其義。<br/>　　　始建國都曰元。非善之長，何以始之。<br/>　　　主義行德曰元。以義為主，行德政。<br/>　　　聖善周聞曰宣。聞，謂所聞善事也。<br/>　　　兵甲亟作曰莊。以數徵為嚴。</p><p>　　　叡圉克服曰莊。通邊圉，使能服。<br/>　　　勝敵志強曰莊。不撓，故勝。<br/>　　　死於原野曰莊。非嚴何以死難。<br/>　　　屢征殺伐曰莊。以嚴厘之。<br/>　　　武而不遂曰莊。武功不成。<br/>　　　柔質慈民曰惠。知其性。<br/>　　　愛民好與曰惠。與謂施。<br/>　　　夙夜警戒曰敬。敬身思戒。<br/>　　　合善典法曰敬。非敬何以善之。<br/>　　　剛德克就曰肅。成其敬使為終。<br/>　　　執心決斷曰肅。言嚴果。<br/>　　　不生其國曰聲。生於外家。<br/>　　　愛民好治曰戴。好民治。<br/>　　　典禮不愆曰戴。無過。<br/>　　　未家短折曰傷。未家，未娶。</p><p>　　　短折不成曰殤。有知而夭殤。<br/>　　　隱拂不成曰隱。不以隱括改其性。<br/>　　　不顯尸國曰隱。以閒主國。<br/>　　　見美堅長曰隱。美過其令。<br/>　　　官人應實曰知。能官人。<br/>　　　肆行勞祀曰悼。放心勞於淫祀，言不修德。<br/>　　　年中早夭曰悼。年不稱志。<br/>　　　恐懼從處曰悼。從處，言險圮。<br/>　　　凶年無穀曰荒。不務耕稼。<br/>　　　外內從亂曰荒。家不治，官不治。<br/>　　　好樂怠政曰荒。淫於聲樂，怠於政事。<br/>　　　在國遭憂曰愍。仍多大喪。<br/>　　　在國逢竊豁。兵寇之事。<br/>　　　禍亂方作曰愍。國無政，動長亂。<br/>　　　使民悲傷曰愍。苛政賊害。</p><p>　　　貞心大度曰匡。<span class="inlay2">心正而用察少。</span><br/>　　　德正應和曰莫。<span class="inlay2">正其德，應其和。</span><br/>　　　施勤無私曰類。<span class="inlay2">無私，唯義所在。</span><br/>　　　果慮果遠曰明。<span class="inlay2">自任多，近於專。</span><br/>　　　嗇於賜與曰愛。<span class="inlay2">言貪恡。</span><br/>　　　危身奉上曰忠。險不辭難。<br/>　　　克威捷行曰魏。有威而敏行。<br/>　　　克威惠禮曰魏。雖威不逆禮。<br/>　　　教誨不倦曰長。以道教之。<br/>　　　肇敏行成曰直。始疾行成，言不深。<br/>　　　疏遠繼位曰紹。非其弟過得之。<br/>　　　好廉自克曰節。自勝其情欲。<br/>　　　好更改舊曰易。變故改常。<br/>　　　愛民在刑曰克。道之以政，齊之以法。<br/>　　　除殘去虐曰湯。</p><p>　　　一德不懈曰簡。一不委曲。<br/>　　　平易不訾曰簡。不信訾毀。<br/>　　　尊賢貴義曰恭。尊事賢人，寵貴義士。<br/>　　　敬事供上曰恭。供奉也。<br/>　　　尊賢敬讓曰恭。敬有德，讓有功。<br/>　　　旣過能改曰恭。言自知。<br/>　　　執事堅固曰恭。守正不移。<br/>　　　愛民長弟曰恭。順長接弟。<br/>　　　執禮御賓曰恭。迎待賓也。<br/>　　　芘親之闕曰恭。修德以蓋之。<br/>　　　尊賢讓善曰恭。不專己善，推於人。<br/>　　　威儀悉備曰欽。威則可畏，儀則可象。<br/>　　　大慮靜民曰定。思樹惠。<br/>　　　純行不爽曰定。行一不傷。<br/>　　　安民大慮曰定。以慮安民。</p><p>　　　安民法古曰定。不失舊意。<br/>　　　闢地有德曰襄。取之以義。<br/>　　　甲胄有勞曰襄。亟征伐。<br/>　　　小心畏忌曰僖。思所當忌。<br/>　　　質淵受諫曰厘。深故能受。<br/>　　　有罰而還曰厘。知難而退。<br/>　　　溫柔賢善曰懿。性純淑。<br/>　　　心能制義曰度。制事得宜。<br/>　　　聰明睿哲曰獻。有通知之聰。<br/>　　　知質有聖曰獻。有所通而無蔽。<br/>　　　五宗安之曰孝。五世之宗。<br/>　　　慈惠愛親曰孝。周愛族親。<br/>　　　秉德不回曰孝。順於德而不違。<br/>　　　協時肇享曰孝。協合肇始。<br/>　　　執心克莊曰齊。能自嚴。</p><p>　　　資輔共就曰齊。資輔佐而共成。<br/>　　　甄心動懼曰頃。甄精。<br/>　　　敏以敬慎曰頃。疾於所慎敬。<br/>　　　柔德安衆曰靖。成衆使安。<br/>　　　恭己鮮言曰靖。恭己正身，少言而中。<br/>　　　寬樂令終曰靖。性寬樂義，以善自終。<br/>　　　威德剛武曰圉。御亂患。<br/>　　　彌年壽考曰胡。久也。<br/>　　　保民耆艾曰胡。六十曰耆，七十曰艾。<br/>　　　追補前過曰剛。勤善以補過。<br/>　　　猛以剛果曰威。猛則少寬。果，敢行。<br/>　　　猛以強果曰威。強甚於剛。<br/>　　　強義執正曰威。問正言無邪。<br/>　　　治典不殺曰祁。秉常不衰。<br/>　　　大慮行節曰考。言成其節。</p><p>　　　治民克盡曰使。克盡無恩惠。<br/>　　　好和不爭曰安。生而少斷。<br/>　　　道德純一曰思。道大而德一。<br/>　　　大省兆民曰思。大親民而不殺。<br/>　　　外內思索曰思。言求善。<br/>　　　追悔前過曰思。思而能改。<br/>　　　行見中外曰愨。表裏如一。<br/>　　　狀古述今曰譽。立言之稱。<br/>　　　昭功寧民曰商。明有功者。<br/>　　　克殺秉政曰夷。秉政不任賢。<br/>　　　安心好靜曰夷。不爽政。<br/>　　　執義揚善曰懷。稱人之善。<br/>　　　慈仁短折曰懷。短未六十，折未三十。<br/>　　　述義不克曰丁。不能成義。<br/>　　　有功安民曰烈。以武立功。</p><p>　　　秉德尊業曰烈。<br/>　　　剛克為伐曰翼。<span class="inlay2">伐功也。</span><br/>　　　思慮深遠曰翼。<span class="inlay2">小心翼翼。</span><br/>　　　外內貞複曰白。<span class="inlay2">正而復，終始一。</span><br/>　　　不勤成名曰靈。<span class="inlay2">任本性，不見賢思齊。</span><br/>　　　死而志成曰靈。<span class="inlay2">志事不𠫤命。</span><br/>　　　死見神能曰靈。<span class="inlay2">有鬼不為厲。</span><br/>　　　亂而不損曰靈。<span class="inlay2">不能以治損亂。</span><br/>　　　好祭鬼怪曰靈。瀆鬼神不致遠。<br/>　　　極知鬼神曰靈。其智能聰徹。<br/>　　　殺戮無辜曰厲。<br/>　　　愎很遂過曰刺。去諫曰愎，反是曰很。<br/>　　　不思忘愛曰刺。忘其愛己者。<br/>　　　蚤孤短折曰哀。早未知人事。<br/>　　　恭仁短折曰哀。體恭質仁，功未施。</p><p>　　　好變動民曰躁。數移徙。<br/>　　　不悔前過曰戾。知而不改。<br/>　　　怙威肆行曰醜。肆意行威。<br/>　　　壅遏不通曰幽。弱損不凌。<br/>　　　蚤孤鋪位曰幽。鋪位卽位而卒。<br/>　　　動祭亂常曰幽。易神之班。<br/>　　　柔質受諫曰慧。以虛受人。<br/>　　　名實不爽曰質。不爽言相應。<br/>　　　溫良好樂曰良。言其人可好可樂。<br/>　　　慈和遍服曰順。能使人皆服其慈和。<br/>　　　博聞多能曰憲。雖多能，不至於大道。<br/>　　　滿志多窮曰惑。自足者必不惑。<br/>　　　思慮不爽曰厚。不差所思而得。<br/>　　　好內遠禮曰煬。朋淫於家，不奉禮。<br/>　　　去禮遠衆曰煬。不率禮，不親長。</p><p>　　　內外賓服曰正。<span class="inlay2">言以正服之。</span><br/>　　　彰義揜過曰堅。<span class="inlay2">明義以蓋前過。</span><br/>　　　華言無實曰誇。<span class="inlay2">恢誕。</span><br/>　　　逆天虐民曰抗。<span class="inlay2">背尊大而逆之。</span><br/>　　　名與實爽曰繆。<span class="inlay2">言名美而實傷。</span><br/>　　　擇善而從曰比。<span class="inlay2">比方善而從之。</span></p><p>　　　隱，哀也。景，武也。施德為文。除惡為武。闢地為襄。服遠為桓。剛克為僖。施而不成為宣。惠無內德為平。亂而不損為靈。由義而濟為景。餘皆象也。以其所為謚象其事行。和，會也。勤，勞也。遵，循也。爽，傷也。肇，始也。怙，恃也。享，祀也。胡，大也。秉，順也。就，會也。錫，與也。典，常也。肆，放也。康，虛也。睿，聖也。惠，愛也。綏，安也。堅，長也。耆，彊也。考，成也。周，至也。懷，思也。式，法也。布，施也。敏，疾也，速也。載，事也。彌，久也。</p><p><span class="inlay2">　　以前《周書·謚法》。周代君王並取作謚，故全寫一篇，以傳後學。</span></p><h2>史記正義·列國分野</h2><p>　　《漢書·地理志》云：「本秦京師為內史。」<span class="inlay2">顏師古云：「京師，天子所居畿內也。秦並天下，改立郡縣，而京畿所統，時號內史，言其在內，以別於諸郡守也。」百官表云：「內史，《周官》，秦因之，掌治京師。景帝二年，分置左內史、右內史。武帝太初元年，更名京兆尹，左內史名馮翊。主爵中尉，秦官，掌列侯。景帝六年，更名都尉，武帝太初元年，更名右扶風，治內史，與左馮翊、京兆尹，是為三輔也。」</span></p><p>　　秦地於天官東井、輿鬼之分野，其界自弘農故關以西，京兆、扶風、馮翊、北地、上郡、西河、安定、天水、隴西；南有巴、蜀、廣漢、犍為、武都；西有金城、武威、張掖、酒泉、敦煌；又西南有牂柯、越嶲、益州。</p><p>　　魏地觜觿、參之分野。其界自高陵以東，盡河東、河內；南有陳留及汝南之召陵、濦強、新汲、西華、長平，潁川之舞陽、郾陵，河南之開封、中牟、陽武、酸棗、卷。卷，去權反。</p><p>　　周地柳、七星、張之分野。今之河南洛陽、穀城、平陰、偃師、鞏、緱氏。</p><p>　　韓地角、亢、氐之分野。韓分晉，得南陽郡及潁川之父城、定陵、襄城、潁陽、潁陰、長社、陽翟、郟；東接汝南，西接弘農，得新安、宜陽、鄭，今河南之新鄭及成皋、滎陽，潁川之崇高、陽城。</p><p>　　趙地昴、畢之分野。趙分晉得趙國，北有信都、眞定、常山，又得涿郡之高陽莫州鄉；東有廣平、鉅鹿、淸河、河閒，又得渤海郡之東平舒、中邑、文安、束州、成平、章武，河以北也；南至浮水、繁陽、內黃、斥丘；西有太原、定襄、雲中、五原、上黨。</p><p>　　燕地尾、箕之分野。召公封於燕，後三十六世與六國俱稱王。東有漁陽、右北平、遼西、遼東；西有上谷、代郡、鴈門；南有涿郡之易、容城、範陽；北有新成、故安、涿縣、良鄉、新昌及渤海之安次，樂浪、玄菟亦宜屬焉。</p><p>　　齊地虛、危之分野。東有災川、東萊、琅邪、高密、膠東；南有泰山、城陽；北有千乘、淸河以南，渤海之高樂、高城、重合、陽信；西有濟南、平原。</p><p>　　魯地奎、婁之分野。東至東海；南有泗水，至淮得臨淮之下相、睢陵、僮、取慮。</p><p>　　宋地房、心之分野。今之沛、梁、楚、山陽、濟陰、東平及東郡之須昌、壽張，今之睢陽。</p><p>　　衞地營室、東壁之分野。今之東郡及魏郡之黎陽，河內之野王、朝歌。</p><p>　　楚地翼、軫之分野。今之南郡、江夏、零陵、桂陽、武陵、長沙及漢中、汝南郡，後陳、魯屬焉。</p><p>　　吳地斗、牛之分野。今之會稽、九江、丹陽、豫章、廬江、廣陵、六安、臨淮郡。</p><p>　　粵地牽牛、婺女之分野。今蒼梧、鬱林、合浦、交址、九眞、南海、日南。</p><p><span class="inlay2">　　以前是戰國時諸國界域，及相侵伐，犬牙深入，然亦不能委細，故略記之，用知大略。</span></p>
