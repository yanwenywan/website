<h2>史記卷十四</h2><h2>十二諸侯年表第二</h2><p><span class="inlay2">　　《索隱》：案：篇言十二，實敘十三者，賤夷狄不數吳，又霸在後故也。不數而敘之者，闔閭霸盟上國故也。</span></p><p>　　太史公讀《春秋》曆譜諜，<span class="inlay1"><span class="number">一</span></span>至周厲王，未嘗不廢書而歎也。曰：嗚呼，師摯見之矣！<span class="inlay1"><span class="number">二</span></span>紂為象箸<span class="inlay1"><span class="number">三</span></span>而箕子唏。<span class="inlay1"><span class="number">四</span></span>周道缺，詩人本之衽席，《關雎》作。仁義陵遲，《鹿鳴》刺焉。及至厲王，以惡聞其過，<span class="inlay1"><span class="number">五</span></span>公卿懼誅而禍作，厲王遂奔于彘，<span class="inlay1"><span class="number">六</span></span>亂自京師始，而共和行政焉。是後或力政，彊乘弱，興師不請天子。然挾王室之義，<span class="inlay1"><span class="number">七</span></span>以討伐為會盟主，政由五伯，<span class="inlay1"><span class="number">八</span></span>諸侯恣行，<span class="inlay1"><span class="number">九</span></span>淫侈不軌，賊臣譍子滋起矣。齊、晉、秦、楚其在成周微甚，封或百里或五十里。晉阻三河，齊負東海，楚介江淮，<span class="inlay1"><span class="number">一〇</span></span>秦因雍州之固，四海迭興，更為伯主，文武所褒大封，皆威而服焉。是以孔子明王道，干七十餘君，莫能用，故西觀周室，論史記舊聞，興於魯而次《春秋》，上記隱，下至哀之獲麟，約其辭文，去其煩重，<span class="inlay1"><span class="number">一一</span></span>以制義法，王道備，人事浹。七十子之徒口受其傳指，<span class="inlay1"><span class="number">一二</span></span>為有所刺譏褒諱挹損之文辭不可以書見也。魯君子左丘明懼弟子人人異端，各安其意，失其眞，故因孔子《史記》具論其語，成左氏《春秋》。鐸椒為楚威王傳，為王不能盡觀《春秋》，采取成敗，卒四十章，為鐸氏微。<span class="inlay1"><span class="number">一三</span></span>趙孝成王時，其相虞卿上采《春秋》，下觀近勢，亦著八篇，為《虞氏春秋》。<span class="inlay1"><span class="number">一四</span></span>呂不韋者，秦莊襄王相，亦上觀尚古，刪拾《春秋》，集六國時事，以為八覽、六論、十二紀，為《呂氏春秋》。及如荀卿、《孟子》、公孫固、韓非<span class="inlay1"><span class="number">一五</span></span>之徒，各往往捃摭《春秋》之文以著書，不同勝紀。漢相張蒼曆譜五德，<span class="inlay1"><span class="number">一六</span></span>上大夫董仲舒推《春秋》義，頗著文焉<span class="inlay1"><span class="number">一七</span></span>。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：案：劉杳云「三代系表旁行邪上，其放周譜。譜起周代。《藝文志》有古帝王譜。又自古為《春秋》學者，有年曆、譜諜之說，故杜元凱作《春秋》長曆及公子譜。蓋因於舊說，故太史公得讀焉」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：鄭玄曰：「師摯，太師之名。周道衰微，鄭衞之音作，正樂廢而失節，魯太師摯識《關雎》之聲，首理其亂也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：鄒氏及劉氏皆音直慮反，卽筯也。今案：箕子云「為象箸者必為玉桮」，則箸者是樽也，音治略反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：唏，嗚歎聲，音許旣反。又音希，希亦聲餘，故記曰：「夫子曰嘻其甚也」，亦餼音也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：惡，烏故反。過，古臥反。故《國語》云「厲王止謗，道路以目」是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：彘，地名，在河東，後為永安縣也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：挾音協也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：伯音霸。五霸者，齊桓公、晉文公、秦穆公、宋襄公、楚莊王也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：下孟反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《索隱》：介音界，言楚以江淮為界。一云介者夾也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：文去重。去，羌呂反。重，逐龍反。言約《史記》脩《春秋》，去其重文也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《索隱》：傳音逐宣反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《索隱》：鐸椒所撰。名鐸氏微者，《春秋》有微婉之詞故也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《正義》：案：其文八篇，《藝文志》云十五篇，虞卿撰。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一五</span></span>《索隱》：荀況、孟軻、韓非皆著書，自稱「子」。宋有公孫固，無所述。此固，齊人韓固，傳詩者。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一六</span></span>《索隱》：案：張蒼著終始五德傳也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一七</span></span>《索隱》：作《春秋》繁露是。</span></p><p>　　太史公曰：儒者斷其義，馳說者騁其辭，不務綜其終始；曆人取其年月，數家<span class="inlay1"><span class="number">一</span></span>隆於神運，<span class="inlay1"><span class="number">二</span></span>譜諜獨記世謚，其辭略，欲一觀<span class="inlay1"><span class="number">三</span></span>諸要難。<span class="inlay1"><span class="number">四</span></span>於是譜十二諸侯，自共和訖孔子，表見《春秋》、《國語》學者所譏盛衰大指著于篇，為成學治古文者<span class="inlay1"><span class="number">五</span></span>要刪焉<span class="inlay1"><span class="number">六</span></span>。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：上音疏具反，謂陰陽術數之家也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「一作『通』也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：壹觀。音官。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：下奴丹反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「一云『治國聞者』也。」</span></p><p><span
