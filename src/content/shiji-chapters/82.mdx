<h2>史記卷八十二</h2><h2>田單列傳第二十二</h2><p>　　田單者，<span class="inlay1"><span class="number">一</span></span>齊諸田疏屬也。<span class="inlay1">孫守真按：不見題目只見關鍵字+單字想複詞：有親屬，就有疏屬。</span>湣王時，單為臨菑市掾，不見知。及燕使樂毅伐破齊，齊湣王出奔，已而保莒城。燕師長驅平齊，而田單走安平，<span class="inlay1"><span class="number">二</span></span>令其宗人盡斷其車軸末<span class="inlay1"><span class="number">三</span></span>而傅鐵籠。<span class="inlay1"><span class="number">四</span>孫守真按：傅=縛，字形結構換部首；傅=附，字形結構換聲符+字形結構換部首部件，字形結構兼音義·同音通假</span>已而燕軍攻安平，城壞，齊人走，爭塗，以轊折車敗，<span class="inlay1"><span class="number">五</span></span>為燕所虜，唯田單宗人以鐵籠故得脫，東保卽墨。燕旣盡降齊城，唯獨莒、卽墨不下。燕軍聞齊王在莒，並兵攻之。淖齒<span class="inlay1"><span class="number">六</span></span>旣殺湣王於莒，因堅守，距燕軍，數年不下。燕引兵東圍卽墨，卽墨大夫出與戰，敗死。城中相與推田單，曰：「安平之戰，田單宗人以鐵籠得全，習兵。」立以為將軍，以卽墨距燕。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：單音丹。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「今之東安平也，在靑州臨菑縣東十九里。古紀之酅邑，齊改為安平，秦滅齊，改為東安平縣，屬齊郡，以定州有安平，故加『東』字。」《索隱》：按：《地理志》東安平屬淄川國也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：斷音都緩反。斷其軸，恐長相撥也。以鐵裹軸頭，堅而易進也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「傅音附。」《索隱》：傅音附。按：截其軸與轂齊，以鐵鍱附軸末，施轄於鐵中以制轂也。又《方言》曰「車轊，齊謂之籠」。郭璞云「車軸也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「轊，車軸頭也。音衞。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「多作『悼齒』也。」</span></p><p>　　頃之，燕昭王卒，惠王立，與樂毅有隙。田單聞之，乃縱反閒於燕，宣言曰：「齊王已死，城之不拔者二耳。樂毅畏誅而不敢歸，以伐齊為名，實欲連兵南面而王齊。齊人未附，故且緩攻卽墨以待其事。齊人所懼，唯恐他將之來，卽墨殘矣。」燕王以為然，使騎劫代樂毅。</p><p>　　樂毅因歸趙，燕人士卒忿。而田單乃令城中人食必祭其先祖於庭，飛鳥悉翔舞城中下食。燕人怪之。田單因宣言曰：「神來下敎我。」乃令城中人曰：「當有神人為我師。」有一卒曰：「臣可以為師乎？」因反走。田單乃起，引還，東鄕坐，師事之。卒曰：「臣欺君，誠無能也。」田單曰：「子勿言也！」因師之。每出約束，必稱神師。乃宣言曰：「吾唯懼燕軍之劓所得齊卒，置之前行，<span class="inlay1"><span class="number">一</span></span>與我戰，卽墨敗矣。」燕人聞之，如其言。城中人見齊諸降者盡劓，皆怒，堅守，唯恐見得。單又縱反閒曰：「吾懼燕人掘吾城外冢墓，僇先人，可為寒心。」燕軍盡掘壟墓，燒死人。卽墨人從城上望見，皆涕泣，俱欲出戰，怒自十倍。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：胡郎反。</span></p><p>　　田單知士卒之可用，乃身操版插，<span class="inlay1"><span class="number">一</span>孫守真按：插=鍤，字形結構換部首。「版插」疑即「畚鍤」，同音通假·字形結構兼音義·聲音變化的關係·聲調不重要。謂築防禦工事也。</span>與士卒分功，妻妾編於行伍之閒，盡散飲食饗士。令甲卒皆伏，<span class="inlay1">孫守真按：埋伏，單字想複詞。</span>使老弱女子乘城，遣使約降於燕，燕軍皆呼萬歲。田單又收民金，得千溢，令卽墨富豪遺燕將，曰：「卽墨卽降，願無虜掠吾族家妻妾，令安堵。」<span class="inlay1">孫守真按：堵，又作「都」。</span>燕將大喜，許之。燕軍由此益懈。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：操音七高反。插音初洽反。《正義》：古之軍行，常負版插也。</span></p><p>　　田單乃收城中得千餘牛，為絳繒衣，畫以五彩龍文，束兵刃於其角，而灌脂束葦於尾，<span class="inlay1">孫守真按：先抓動詞：灌、束。看到「於、于、在」，要想到：倒序重組：於尾束葦灌脂。油脂、澆灌，同義複詞。</span>燒其端。鑿城數十穴，<span class="inlay1">孫守真按：洞穴，同義複詞。</span>夜縱牛，壯士五千人隨其後。牛尾熱，怒而奔燕軍，燕軍夜大驚。牛尾炬火光明炫耀，燕軍視之皆龍文，所觸盡死傷。五千人因銜枚擊之，而城中鼓噪從之，老弱皆擊銅器為聲，聲動天地。燕軍大駭，敗走。齊人遂夷殺其將騎劫。燕軍擾亂奔走，齊人追亡逐北，所過城邑皆畔燕而歸田單，兵日益多，乘勝，燕日敗亡，卒至河上，<span class="inlay1"><span class="number">一</span></span>而齊七十餘城皆復為齊。乃迎襄王於莒，入臨菑而聽政。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：河上卽齊之北界，近河東，齊之舊地。</span></p><p>　　襄王封田單，號曰安平君。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：以單初起安平，故以為號。</span></p><p>　　太史公曰：兵以正合，以奇勝。<span class="inlay1"><span class="number">一</span></span>善之者，<span class="inlay1"><span class="number">二</span></span>出奇無窮。<span class="inlay1"><span class="number">三</span></span>奇正還相生，<span class="inlay1"><span class="number">四</span></span>如環之無端。<span class="inlay1"><span class="number">五</span></span>夫始如處女，<span class="inlay1"><span class="number">六</span></span>適人開戶；<span class="inlay1"><span class="number">七</span>孫守真按：不見題目只見關鍵字+單字想複詞：門戶洞開。</span>後如脫兔，適不及距：<span class="inlay1"><span class="number">八</span></span>其田單之謂邪！</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：魏武帝曰：「先出合戰為正，後出為奇也。正者當敵，奇兵擊不備。」《索隱》：按：奇謂權詐也。注引魏武，蓋亦軍令也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：兵不厭詐，故云「善之」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：謂權變多也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：猶當合也。言正兵當陣，張左右翼掩其不備，則奇正合敗敵也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：言用兵之術，或用正法，或用奇計，使前敵不可測量，如尋環中不知端際也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：言兵之始，如處女之軟弱也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《集解》：徐廣曰：「適音敵。」《索隱》：適音敵。若我如處女之弱，則敵人輕侮，開戶不為備也。《正義》：敵人謂燕軍也。言燕軍被田單反閒，易將及劓卒燒壟墓，而令齊卒甚怒，是敵人為單開門戶也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：魏武帝曰：「如女示弱，脫兔往疾也。」《索隱》：言克敵之後，卷甲而趨，如兔之得脫而走疾也。敵不及距者，若脫兔忽過，而敵忘其所距也。</span></p><p><br/>　　初，淖齒之殺湣王也，莒人求湣王子法章，得之太史嬓之家<span class="inlay1"><span class="number">一</span></span>，為人灌園。<span class="inlay1">孫守真按：單字想複詞：灌溉，澆灌，田園，同義複詞。灌園=灌溉田園。</span>嬓女憐而善遇之。後法章私以情告女，女遂與通。及莒人共立法章為齊王，以莒距燕，而太史氏女遂為后，所謂「君王后」也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：嬓音皎。</span></p><p>　　燕之初入齊，聞畫邑人王蠋賢，<span class="inlay1"><span class="number">一</span></span>令軍中曰「環畫邑三十里無入」，以王蠋之故。已而使人謂蠋曰：「齊人多高子之義，吾以子為將，封子萬家。」蠋固謝。燕人曰：「子不聽，吾引三軍而屠畫邑。」王蠋曰：「忠臣不事二君，貞女不更二夫。齊王不聽吾諫，故退而耕於野。國旣破亡，吾不能存；今又劫之以兵為君將，是助桀為暴也。與其生而無義，固不如烹！」遂經其頸<span class="inlay1"><span class="number">二</span></span>於樹枝，自奮絕脰而死。<span class="inlay1"><span class="number">三</span></span>齊亡大夫聞之，曰：「王蠋，布衣也，義不北面於燕，況在位食祿者乎！」乃相聚如莒，求諸子，立為襄王。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：劉熙曰：「齊西南近邑。畫音獲。」《索隱》：畫，一音獲，又音胡卦反。劉熙云：「齊西南近邑。」蠋音觸，又音歜。　《正義》：《括地志》云：「戟里城在臨淄西北三十里，《春秋》時棘邑，又云澅邑。」蠋所居卽此邑，因澅水為名也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：經猶系也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：何休云：「脰，頸，齊語也。音豆。」</span></p><p><span class="inlay2">　【《索隱》述贊】：軍法以正，實尚奇兵。斷軸自免，反閒先行。羣鳥或衆，五牛揚旌。卒破騎劫，皆復齊城。襄王嗣位，乃封安平。</span></p>
