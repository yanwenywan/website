<h2>史記卷六十一</h2><h2>伯夷列傳第一</h2><p><span class="inlay2">　　《索隱》：列傳者，謂敘列人臣事跡，令可傳於後世，故曰列傳。　《正義》：其人行跡可序列，故云列傳。</span></p><p>　　夫學者載籍極博，猶考信於六蓺。《詩》《書》雖缺，<span class="inlay1"><span class="number">一</span></span>然虞夏之文可知也。<span class="inlay1"><span class="number">二</span></span>堯將遜位，讓於虞舜，舜禹之閒，岳牧咸薦，乃試之於位，典職數十年，<span class="inlay1"><span class="number">三</span></span>功用旣興，然後授政。示天下重器，<span class="inlay1"><span class="number">四</span></span>王者大統，傳天下若斯之難也。而說者曰堯讓天下於許由，<span class="inlay1"><span class="number">五</span></span>許由不受，恥之逃隱。及夏之時，有卞隨、務光者。此何以稱焉？<span class="inlay1"><span class="number">六</span></span>太史公曰：余登箕山，<span class="inlay1"><span class="number">七</span></span>其上蓋有許由冢云。孔子序列古之仁聖賢人，如吳太伯、伯夷之倫詳矣。余以所聞由、光<span class="inlay1"><span class="number">八</span></span>義至高，<span class="inlay1"><span class="number">九</span></span>其文辭不少槪見，何哉？<span class="inlay1"><span class="number">一〇</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：》：按：《孔子系家》稱古詩三千餘篇，孔子刪三百五篇為《詩》，今亡五篇。又《書緯》稱孔子求得黃帝玄孫帝魁之書，迄秦穆公，凡三千三百三十篇，乃刪以一百篇為《尚書》，十八篇為《中候》。今百篇之內見亡四十二篇，是《詩》《書》又有缺亡者也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：》：按：《尚書》有《堯典》、《舜典》、《大禹謨》，備言虞夏禪讓之事，故云「虞夏之文可知也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：》：舜禹皆典職事二十餘年，然後踐帝位。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：》：言天下者是王者之重器，故《莊子》云「天下大器」是也。則大器亦重器也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《正義》：》：皇甫謐《高士傳》云：「許由字武仲。堯聞，致天下而讓焉，乃退而遁於中岳潁水之陽，箕山之下隱。堯又召為九州長，由不欲聞之，洗耳於潁水濱。時有巢父牽犢欲飲之，見由洗耳，問其故。對曰：『堯欲召我為九州長，惡聞其聲，是故洗耳。』巢父曰：『子若處高岸深谷，人道不通，誰能見子？子故浮游，欲聞求其名譽。污吾犢口。』牽犢上流飲之。許由歿，葬此山，亦名許由山。」在洛州陽城縣南十三里。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：》：按：「說者」謂諸子雜記也。然堯讓於許由，及夏時有卞隨、務光等，殷湯讓之天下，並不受而逃，事具莊周《讓王篇》。　《正義》：經史唯稱伯夷、叔齊，不及許由、卞隨、務光者，不少槪見，何以哉？故言「何以稱焉」，為不稱說之也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：》：蓋楊惲、東方朔見其文稱「餘」，而加「太史公曰」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：》：謂太史公聞莊周所說許由、務光等。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：》：謂堯讓天下於許由，由遂逃箕山，洗耳於潁水；卞隨自投於桐水；務光負石自沈於盧水：是義至高。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《索隱》：》：按：槪是梗槪，謂略也。蓋以由、光義至高，而詩書之文辭遂不少梗槪載見，何以如此哉？是太史公疑說者之言或非實也。　《正義》：槪，古代反。</span></p><p>　　孔子曰：「伯夷、叔齊，不念舊惡，怨是用希。」「求仁得仁，又何怨乎？」余悲伯夷之意，睹軼詩可異焉。<span class="inlay1"><span class="number">一</span></span>其傳曰：</p><p>　　伯夷、叔齊，孤竹君之二子也。<span class="inlay1"><span class="number">二</span></span>父欲立叔齊，及父卒，叔齊讓伯夷。伯夷曰：「父命也。」遂逃去。叔齊亦不肯立而逃之。國人立其中子。於是伯夷、叔齊聞西伯昌善養老，盍往歸焉。<span class="inlay1"><span class="number">三</span></span>及至，西伯卒，武王載木主，號為文王，東伐紂。伯夷、叔齊叩馬而諫曰：「父死不葬，爰及干戈，可謂孝乎？以臣弒君，可謂仁乎？」左右欲兵之。太公曰：「此義人也。」扶而去之。武王已平殷亂，天下宗周，而伯夷、叔齊恥之，義不食周粟，隱於首陽山，<span class="inlay1"><span class="number">四</span></span>采薇而食之。<span class="inlay1"><span class="number">五</span></span>及餓且死，作歌。其辭曰：「登彼西山兮，<span class="inlay1"><span class="number">六</span></span>采其薇矣。以暴易暴兮，不知其非矣。<span class="inlay1"><span class="number">七</span></span>神農、虞、夏忽焉沒兮，我安適歸矣？<span class="inlay1"><span class="number">八</span></span>于嗟徂兮，命之衰矣！<span class="inlay1">孫守真按：矣=唉，字形結構換部首。于音汙=嗚，字形結構換部首+字形結構換聲符。</span>」<span class="inlay1"><span class="number">九</span></span>遂餓死於首陽山。</p><p>　　由此觀之，怨邪非邪？<span class="inlay1"><span class="number">一〇</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：謂悲其兄弟相讓，又義不食周粟而餓死。睹音覩。軼音逸。謂見逸詩之文，卽下《采薇》之詩是也。不編入三百篇，故云逸詩也。可異焉者，按《論語》云「求仁得仁，又何怨乎」。今其詩云「我安適歸矣，于嗟徂兮，命之衰矣」。是怨詞也，故云可異焉。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：「其傳」蓋《韓詩外傳》及《呂氏春秋》也。其傳云孤竹君，是殷湯三月丙寅日所封。相傳至夷、齊之父，名初，字子朝。伯夷名允，字公信。叔齊名致，字公達。解者云夷，齊，謚也；伯，仲，又其長少之字。按：《地理志》孤竹城在遼西令支縣。應劭云伯夷之國也。其君姓墨胎氏。　《正義》：本前注「丙寅」作「殷湯正月三日丙寅」。《括地志》云：「孤竹古城在盧龍縣南十二里，殷時諸侯孤竹國也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：劉氏云：「盍者，疑辭。蓋謂其年老歸就西伯也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：馬融曰：「首陽山在河東蒲阪華山之北，河曲之中。」　《正義》：曹大家注《幽通賦》云：「夷、齊餓於首陽山，在隴西首。」又戴延之《西征記》云：「洛陽東北首陽山有夷齊祠。」今在偃師縣西北。又《孟子》云：「夷、齊避紂，居北海之濱。」首陽山，《說文》云首陽山在遼西。史傳及諸書，夷、齊餓於首陽凡五所，各有案據，先後不詳。《莊子》云：「伯夷、叔齊西至岐陽，見周武王伐殷，曰：『吾聞古之士，遭治世不避其任，遇亂世不為苟存。今天下闇，周德衰，其並乎周以塗吾身也，不若避之以絜吾行。』二子北至於首陽之山，遂飢餓而死。」又下詩「登彼西山」，是今淸源縣首陽山，在岐陽西北，明卽夷、齊餓死處也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：薇，蕨也。《爾雅》云：「蕨，鼈也。」　《正義》：陸璣《毛詩草木疏》云：「薇，山菜也。莖葉皆似小豆，蔓生，其味亦如小豆藿，可作羹，亦可生食也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：按：西山卽首陽山也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：謂以武王之暴臣易殷紂之暴主，而不自知其非矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：言羲、農、虞、夏敦樸禪讓之道，超忽久矣，終沒矣。今逢此君臣爭奪，故我安適歸矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：於嗟，嗟嘆之辭也。徂者，往也，死也。言己今日餓死，亦是運命衰薄，不遇大道之時，至幽憂而餓死。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《索隱》：太史公言己觀此詩之情，夷、齊之行似是有所怨邪？又疑其云非是怨邪？</span></p><p><br/>　　或曰：「天道無親，常與善人。」若伯夷、叔齊，可謂善人者非邪？<span class="inlay1"><span class="number">一</span></span>積仁絜行如此而餓死！且七十子之徒，仲尼獨薦顏淵為好學。然回也屢空，糟穅不厭，<span class="inlay1"><span class="number">二</span></span>而卒蚤夭。天之報施善人，其何如哉？盜蹠日殺不辜，<span class="inlay1"><span class="number">三</span></span>肝人之肉，<span class="inlay1"><span class="number">四</span></span>暴戾恣睢，<span class="inlay1"><span class="number">五</span></span>聚黨數千人橫行天下，竟以壽終。<span class="inlay1"><span class="number">六</span></span>是遵何德哉？<span class="inlay1"><span class="number">七</span></span>此其尤大彰明較著者也。<span class="inlay1"><span class="number">八</span></span>若至近世，操行不軌，專犯忌諱，而終身逸樂，<span class="inlay1"><span class="number">九</span></span>富厚累世不絕。或擇地而蹈之，<span class="inlay1"><span class="number">一〇</span></span>時然後出言，<span class="inlay1"><span class="number">一一</span></span>行不由徑，<span class="inlay1"><span class="number">一二</span></span>非公正不發憤，而遇禍災者，不可勝數也。<span class="inlay1"><span class="number">一三</span></span>余甚惑焉，儻所謂天道，是邪非邪？<span class="inlay1"><span class="number">一四</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：又敘論云若夷、齊之行如此，可謂善人者邪，又非善人者邪，亦疑也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：厭者，飫也，不厭謂不飽也。糟穅，貧者之所餐也，故曰「糟穅之妻」是也。然顏生簞食瓢飲，亦未見「糟穅」之文也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：「蹠」及注作「跖」，並音之石反。按：盜跖，柳下惠之弟，亦見《莊子》，為篇名。　《正義》：按：蹠者，黃帝時大盜之名。以柳下惠弟為天下大盜，故世放古，號之盜蹠。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：劉氏云「謂取人肉為生肝」，非也。按：《莊子》云「跖方休卒太山之陽，膾人肝而餔之」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：暴戾謂凶暴而惡戾也。鄒誕生恣音資，睢音千餘反。劉氏恣音如字，睢音休季反。恣睢謂恣行為睢惡之貌也。　《正義》：睢，仰白目，怒貌也。言盜蹠凶暴，惡戾，恣性，怒白目也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：《皇覽》曰：「盜跖冢在河東大陽，臨河曲，直弘農華陰縣潼鄕。」按：盜跖卽柳下惠弟也。　《索隱》：直音如字。直者，當也。或音值，非也。潼音同。按：潼，水名，因為鄕，今之潼津關是，亦為縣也。　《正義》：《括地志》云：「盜跖冢在陝州河北縣西二十里。河北縣本漢大陽縣也。」又今齊州平陵縣有盜跖冢，未詳也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：言盜蹠無道，橫行天下，竟以壽終，是其人遵行何德而致此哉？</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：按：較，明也。言伯夷有德而餓死，盜跖暴戾而壽終，是賢不遇而惡道長，尤大著明之証也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：謂若魯桓、楚靈、晉獻、齊襄之比皆是。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《索隱》：謂不仕暗君，不飲盜泉，裹足高山之頂，竄跡滄海之濱是也。　《正義》：謂北郭駱、鮑焦等是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：按：《論語》「夫子時然後言」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《索隱》：按：《論語》澹台滅明之行也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《索隱》：謂人臣之節，非公正之事不感激發憤。或出忠言，或致身命，而卒遇禍災者，不可勝數。謂龍逢、比干、屈平、伍胥之屬是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《索隱》：太史公惑於不軌而逸樂，公正而遇災害，為天道之非而又是邪？深惑之也。蓋天道玄遠，聰聽暫遺，或窮通數會，不由行事，所以行善未必福，行惡未必禍，故先達皆猶昧之也。　《正義》：儻音他蕩反。儻，未定之詞也。為天道不敢旳言是非，故云儻也。</span></p><p>　　子曰「道不同不相為謀」，亦各從其志也。<span class="inlay1"><span class="number">一</span></span>故曰「富貴如可求，雖執鞭之士，吾亦為之。<span class="inlay1"><span class="number">二</span></span>如不可求，從吾所好」。<span class="inlay1"><span class="number">三</span></span>「歲寒，然後知松柏之後凋」。<span class="inlay1"><span class="number">四</span></span>舉世混濁，淸士乃見。<span class="inlay1"><span class="number">五</span></span>豈以其重若彼，其輕若此哉？<span class="inlay1"><span class="number">六</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：太史公引孔子之言証前事也。言天道人道不同，一任其運遇，亦各從其志意也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：鄭玄曰：「富貴不可求而得之，當修德以得之。若於道可求而得之者，雖執鞭賤職，我亦為之。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：孔安國曰：「所好者古人之道。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：何晏曰：「大寒之歲，衆木皆死，然後松柏少凋傷；平歲衆木亦有不死者，故須歲寒然後別之。喻凡人處治世，亦能自修整，與君子同，在濁世然後知君子之正不苟容也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：《老子》曰：「國家昏亂，有忠臣」，是舉代混濁，則士之淸絜者乃彰見，故上文「歲寒然後知松柏之後彫」，先為此言張本也。　《正義》：言天下泯亂，淸絜之士不撓，不苟合於盜跖也。</span></p><p>{{　<span class="inlay1"><span class="number">六</span></span>《索隱》：按：謂伯夷讓德之重若彼，而采薇餓死之輕若此。又一解云，操行不軌，富厚累代，是其重若彼；公正發憤而遇禍災，是其輕若此也。　《正義》：重謂盜跖等也。輕謂夷、齊、由、光等也。</p><p>　　「君子疾沒世而名不稱焉。」<span class="inlay1"><span class="number">一</span></span>賈子曰：<span class="inlay1"><span class="number">二</span></span>「貪夫徇財，<span class="inlay1"><span class="number">三</span></span>烈士徇名，夸者死權，<span class="inlay1"><span class="number">四</span></span>衆庶馮生。」<span class="inlay1"><span class="number">五</span></span>「同明相照，<span class="inlay1"><span class="number">六</span></span>同類相求。」<span class="inlay1"><span class="number">七</span></span>「雲從龍，風從虎，<span class="inlay1"><span class="number">八</span></span>聖人作而萬物覩。」<span class="inlay1"><span class="number">九</span></span>伯夷、叔齊雖賢，得夫子而名益彰。<span class="inlay1"><span class="number">一〇</span></span>顏淵雖篤學，附驥尾而行益顯。<span class="inlay1"><span class="number">一一</span></span>巖穴之士，趣舍有時若此，類名堙滅而不稱，悲夫！<span class="inlay1"><span class="number">一二</span></span><span class="inlay1">孫守真按：：當讀作「巖穴之士，趣舍有時。若此類名堙滅而不稱，悲夫！」謂巖穴之士，蓋其取捨、進退、出處亦有時也；如伯夷叔齊。</span>閭巷之人，欲砥行立名者，<span class="inlay1"><span class="number">一三</span></span>非附靑雲之士，惡能施于後世哉？</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：自此已下，雖論伯夷得夫子而名彰，顏回附驥尾而行著，蓋亦欲微見己之著撰不已，亦是疾沒世而名不稱焉，故引賈子「貪夫徇財，烈士徇名」是也。又引「同明相照，同類相求」，「雲從龍，風從虎」者，言物各以類相求。故太史公言己亦是操行廉直而不用於代，卒陷非罪，與伯夷相類，故寄此而發論也。　《正義》：君子疾沒世后懼名堙滅而不稱，若夷、齊、顏回絜行立名，後代稱述，亦太史公欲漸見己立名著述之美也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：賈子，賈誼也。誼作《鵩鳥賦》云然，故太史公引之而稱「賈子」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：徇，才迅反。徇，求也。瓚云：「以身從物曰徇。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：言貪權勢以矜夸者，至死不休，故云「死權」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：馮者，恃也，音凭。言衆庶之情，蓋恃矜其生也。鄒誕本作「每生」。每者，冒也，卽貪冒之義。　《正義》：太史公引《賈子》譬作《史記》，若貪夫徇財，烈士徇名，夸者死權，衆庶馮生，乃成其《史記》。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：已下並《易．繫辭》文也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《正義》：天欲雨而柱礎潤，謂同德者相應。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《集解》：王肅曰：「龍舉而景雲屬，虎嘯而谷風興。」張璠曰：「猶言龍從雲，虎從風也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《集解》：馬融曰：「作，起也。」　《索隱》：按：又引此句者，謂聖人起而居位，則萬物之情皆得覩見，故己今日又得著書言世情之輕重也。　《正義》：此有識也。聖人有養生之德，萬物有長育之情，故相感應也。此以上至「同明相照」是《周易．乾．象辭》也。太史公引此等相感者，欲見述作之意，令萬物有睹也。孔子歿後五百歲而己當之，故作《史記》，使萬物見覩之也。《太史公序傳》云：「先人有言：『自周公卒五百歲而有孔子，孔子卒後至於今五百歲，有能紹名世，正《易傳》，繼《春秋》，本《詩》、《書》、《禮》、《樂》之際？』意在斯乎！小子何敢讓焉。」作述《六經》云：「《易》著天地陰陽四時五行，故長於變。《禮》經紀人倫，故長於行。《書》記先王之事，故長於政。《詩》記山川谿谷禽獸草木牝牡雌雄，故長於風。《樂》樂所以立，故長於和。《春秋》辨是非，故長於治人。是故《禮》以節人，《樂》以發和，《書》以道事，《詩》以達意，《易》以道化，《春秋》以道義。撥亂世反之正，莫近於《春秋》。」按：述作而萬物睹見。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《正義》：伯夷、叔齊雖有賢行，得夫子稱揚而名益彰著。萬物雖有生養之性，得太史公作述而世事益睹見。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《索隱》：按：蒼蠅附驥尾而致千里，以譬顏回因孔子而名彰也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《正義》：趣音趨。舍音捨。趣，向也。捨，廢也。言隱處之士，時有附驥尾而名曉達；若堙滅不稱數者，亦可悲痛。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《正義》：砥音旨。礪行修德在鄕閭者，若不托貴大之士，何得封侯爵賞而名留後代也？</span></p><p><span class="inlay2">　【《索隱》述贊】天道平分，與善徒云。賢而餓死，盜且聚羣。吉凶倚伏，報施糾紛。子罕言命，得自前聞。嗟彼素士，不附靑雲！</span></p>
