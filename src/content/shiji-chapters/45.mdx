<h2>史記卷四十五</h2><h2>韓世家第十五</h2><p>　　韓之先與周同姓，<span class="inlay1"><span class="number">一</span></span>姓姬氏。其後苗裔事晉，得封於韓原，<span class="inlay1"><span class="number">二</span></span>曰韓武子。武子後三世<span class="inlay1"><span class="number">三</span></span>有韓厥，從封姓為韓氏。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：《左氏傳》云「邗、晉、應、韓，武之穆」，是武王之子，故詩稱「韓侯出祖」，是有韓而先滅。今據此文，云「其後裔事晉，封于韓原，曰韓武子」，則武子本是韓侯之後，晉又封之於韓原，卽今之馮翊韓城是也。然按《系本》及《左傳》舊說，皆謂韓萬是曲沃桓叔之子，卽是晉之支庶。又《國語》叔向謂韓宣子能修武子之德，起再拜謝曰「自桓叔已下，嘉吾子之賜」，亦言桓叔是韓之祖也。今以韓侯之後別有桓叔，非關曲沃之桓叔，如此則與太史公之意亦有違。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：《括地志》云：「韓原在同州韓城縣西南八里。又韓城在縣南十八里，故古韓國也。古今地名云韓武子食菜於韓原故城也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：《系本》云：「萬生賕伯，賕伯生定伯簡，簡生輿，輿生獻子厥。</span></p><p>　　韓厥，晉景公之三年，晉司寇屠岸賈將作亂，誅靈公之賊趙盾。趙盾已死矣，欲誅其子趙朔。韓厥止賈，賈不聽。厥告趙朔令亡。朔曰：「子必能不絕趙祀，死不恨矣。」韓厥許之。及賈誅趙氏，厥稱疾不出。程嬰、公孫杵臼之藏趙孤趙武也，厥知之。</p><p>　　景公十一年，厥與郤克將兵八百乘伐齊，敗齊頃公于鞍，<span class="inlay1"><span class="number">一</span></span>獲逢丑父。於是晉作六卿，而韓厥在一卿之位，號為獻子。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：音安。《括地志》云：「故鞍城今俗名馬鞍城，在濟州平陰縣十里。」</span></p><p>　　晉景公十七年，病，卜大業之不遂者為祟。韓厥稱趙成季之功，今後無祀，以感景公。景公問曰：「尚有世乎？」厥於是言趙武，而復與故趙氏田邑，續趙氏祀。</p><p>　　晉悼公之（十）\〔七〕年，韓獻子老。獻子卒，子宣子代。宣字徙居州。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：宣子名起。州，今在河內是也。　《正義》：《括地志》云：「懷州武德縣本周司寇蘇忿生之州邑也。」</span></p><p>　　晉平公十四年，吳季札使晉，曰：「晉國之政卒歸於韓、魏、趙矣。」晉頃公十二年，韓宣子與趙、魏共分祁氏、羊舌氏十縣。晉定公十五年，宣子與趙簡子侵伐范、中行氏。宣子卒，子貞子代立。貞子徙居平陽。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：《系本》作「平子」，名須，宣子子也。又云「景子居平陽」。平陽在山西。宋忠曰「今河東平陽縣」。　《正義》：平陽，晉州城是。</span></p><p>　　貞子卒，子簡子代。<span class="inlay1"><span class="number">一</span></span>簡子卒，子《莊子》代。《莊子》卒，子康子<span class="inlay1"><span class="number">二</span></span>代。康子與趙襄子、魏桓子共敗知伯，分其地，地益大，大於諸侯。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「《史記》多無簡子、莊子，而云貞子生康子。班氏亦同。」　《索隱》：徐廣云：「《史記》多無簡子、莊子，而云貞子生康子。班氏亦同。」按：《系本》有簡子，名不信；莊子，名庚。《趙系家》亦有簡子，名不佞。</span><span class="inlay1">孫守真按：康莊，同義複詞。單字想複詞。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：名虎。</span></p><p>　　康子卒，子武子<span class="inlay1"><span class="number">一</span></span>代。武子二年，伐鄭，殺其君幽公。十六年，武子卒，子景侯立。<span class="inlay1"><span class="number">二</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：名啓章。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：《紀年》及《系本》皆作「景子」，名處。</span></p><p>　　景侯虔元年，伐鄭，取雍丘。二年，鄭敗我負黍。</p><p><br/>　　六年，與趙、魏俱得列為諸侯。</p><p>　　九年，鄭圍我陽翟。景侯卒，子列侯取立。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：《系本》作「武侯」。</span></p><p>　　列侯三年，聶政殺韓相俠累。<span class="inlay1"><span class="number">一</span></span>九年，秦伐我宜陽，取六邑。十三年，列侯卒，子文侯立。<span class="inlay1"><span class="number">二</span></span>是歲魏文侯卒。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「六年救魯也。」　《索隱》：《戰國策》作「殺韓傀」，高誘曰「韓傀，俠侯累也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按：《紀年》無文侯，《系本》無列侯。</span></p><p>　　文侯二年，伐鄭，取陽城。伐宋，到彭城，執宋君。七年，伐齊，至桑丘。鄭反晉。九年，伐齊，至靈丘。<span class="inlay1"><span class="number">一</span></span>十年，文侯卒，子哀侯立。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：靈丘，蔚州縣也，此時屬燕也。</span></p><p>　　哀侯元年，與趙、魏分晉國。二年，滅鄭，因徙都鄭。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：《紀年》魏武侯二十一年，韓滅鄭，哀侯入于鄭。二十二年，晉桓公邑哀侯于鄭。是韓旣徙都，因改號曰鄭，故《戰國策》謂韓惠王曰鄭惠王，猶魏徙大梁稱梁王然也。</span></p><p>　　六年，韓嚴弒其君哀侯。而子懿侯立。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：按：年表懿侯作「莊侯」。又《紀年》云「晉桓公邑哀侯于鄭，韓山堅賊其君哀侯而立韓若山」。若山卽懿侯也，則韓嚴為韓山堅也。而《戰國策》又有韓仲子，名遂，又恐是韓嚴也。</span></p><p>　　懿侯二年，魏敗我馬陵。<span class="inlay1"><span class="number">一</span></span>五年，與魏惠王會宅陽。<span class="inlay1"><span class="number">二</span></span>九年，魏敗我澮。<span class="inlay1"><span class="number">三</span></span>十二年，懿侯卒，子昭侯立。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：在魏州元城縣東南一里。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：在鄭州也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：徐廣曰：「大雨三月也。」　《正義》：澮，古外反，在陵州澮水之上也。</span></p><p>　　昭侯元年，秦敗我西山。二年，宋取我黃池。<span class="inlay1"><span class="number">一</span></span>魏取朱。六年，伐東周，<span class="inlay1"><span class="number">二</span></span>取陵觀、郉丘。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「在平丘。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：河南鞏縣。</span></p><p>　　八年，申不害相韓，脩術行道，國內以治，諸侯不來侵伐。</p><p>　　十年，韓姬弒其君悼公。<span class="inlay1"><span class="number">一</span></span>十一年，昭侯如秦。二十二年，申不害死。二十四年，秦來拔我宜陽。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：《紀年》「姬」亦作「灬」，並音羊之反。姬是韓大夫，而王邵亦云不知悼公何君也。</span></p><p>　　二十五年，旱，作高門。屈宜臼<span class="inlay1"><span class="number">一</span></span>曰：「昭侯不出此門。何也？不時。吾所謂時者，非時日也，人固有利不利時。昭侯嘗利矣，不作高門。往年秦拔宜陽，今年旱，昭侯不以此時卹民之急，而顧益奢，此謂『時絀舉贏』。」<span class="inlay1"><span class="number">二</span></span>二十六年，高門成，昭侯卒，<span class="inlay1"><span class="number">三</span></span>果不出此門。子宣惠王立。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：許愼曰：「屈宜臼，楚大夫，在魏也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「時衰秏而作奢侈。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：按：《紀年》「鄭昭侯武薨，次威侯立。威侯七年，與邯鄲圍襄陵。五月，梁惠王會威侯于巫沙。十月，鄭宣王朝梁」，不見威侯之卒。下敗韓舉在威侯八年，而此系家卽以為宣惠王之年。又上有殺悼公，悼公又不知是誰之謚。則韓微小，國史失代系，故此文及《系本》不同，蓋亦不可復考。</span></p><p>　　宣惠王五年，張儀相秦。八年，魏敗我將韓舉。<span class="inlay1"><span class="number">一</span></span>十一年，君號為王。與趙會區鼠。十四，秦伐敗我鄢。<span class="inlay1"><span class="number">二</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：韓舉則是韓將不疑，而《紀年》云韓舉，趙將，蓋舉先為趙將，後入韓。又《紀年》云其敗當韓威王八年，是不同也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「潁川鄢陵縣。音於乾反。」　《正義》：今許州鄢陵縣西北十五里有鄢陵故城是也。</span></p><p>　　十六年，秦敗我脩魚，<span class="inlay1"><span class="number">一</span></span>虜得韓將貘、申差於濁澤。<span class="inlay1"><span class="number">二</span></span>韓氏急，公仲<span class="inlay1"><span class="number">三</span></span>謂韓王曰：「與國非可恃也。今秦之欲伐楚久矣，王不如因張儀為和於秦，賂以一名都，具甲，與之南伐楚，此以一易二之計也。」<span class="inlay1"><span class="number">四</span></span>韓王曰：「善。」乃警公仲之行，<span class="inlay1"><span class="number">五</span></span>將西購於秦。<span class="inlay1"><span class="number">六</span></span>楚王聞之大恐，召陳軫告之。陳軫曰：「秦之欲伐楚久矣，今又得韓之名都一而具甲，秦韓幷兵而伐楚，此秦所禱祀而求也。今已得之矣，楚國必伐矣。王聽臣為之警四境之內，起師言救韓，命戰車滿道路，發信臣，多其車，重其幣，使信王之救己也。縱韓不能聽我，韓必德王也，<span class="inlay1"><span class="number">七</span></span>必不為鴈行以來，<span class="inlay1"><span class="number">八</span></span>是秦韓不和也，兵雖至，楚不大病也。為能聽我絕和於秦，秦必大怒，以厚怨韓。韓之南交楚，必輕秦；輕秦，其應秦必不敬：是因秦、韓之兵而免楚國之患也。」楚王曰：「善。」乃警四境之內，興師言救韓。命戰車滿道路，發信臣，多其車，重其幣。謂韓王曰：「不穀國雖小，已悉發之矣。願大國遂肆志於秦，不穀將以楚殉韓。」<span class="inlay1"><span class="number">九</span></span>韓王聞之大說，乃止公仲之行。<span class="inlay1"><span class="number">一〇</span></span>公仲曰：「不可。夫以實伐我者秦也，以虛名救我者楚也。王恃楚之虛名，而輕絕彊秦之敵，王必為天下大笑。且楚韓非兄弟之國也，又非素約而謀伐秦也。已有伐形，因發兵言救韓，此必陳軫之謀也。且王已使人報於秦矣，今不行，是欺秦也。夫輕欺彊秦而信楚之謀臣，恐王必悔之。」韓王不聽，遂絕於秦。秦因大怒，益甲伐韓，大戰，楚救不至韓。十九年，大破我岸門。<span class="inlay1"><span class="number">一一</span></span>太子倉質於秦以和。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：地名。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「一云鯁、申差。長社有濁澤。」　《索隱》：貘、申差，二將。貘音瘦，亦作「鯁」。　《正義》：按：濁澤者蓋誤，當作「觀澤」。年表云「秦惠文王更元八年，與韓戰，斬首八萬。韓宣惠王十六年，秦敗我脩魚，得將軍申差。魏哀王二年，齊敗我觀澤。趙武靈王九年，與韓、魏擊秦。齊湣王七年，敗魏、趙觀澤」，濁澤定誤矣。徐廣又云「濁澤在長社」，不曉錯誤之甚。《括地志》云「觀澤在魏州頓丘縣東十八里」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：韓相國，名侈。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：一，謂名都也。二，謂使不伐韓而又與之伐楚也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：警，戒也。《戰國策》作「衞」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：《戰國策》作「講」。講亦謀議，與購求意通。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《索隱》：言韓王信楚之救，雖不能聽待楚救至，折入於秦，猶德於楚也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《索隱》：言韓以楚必救己，己雖隨秦來戰，猶德於王，故不為鴈行而來，言不同心旅進也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《索隱》：殉，從死也。言以死助韓。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《索隱》：止不令西之秦。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>《集解》：徐廣曰：「潁陰有岸亭。」　《正義》：《括地志》云：「岸門在許州長社縣西北十八里，今名西武亭矣。」</span></p><p>　　二十一年，<span class="inlay1"><span class="number">一</span></span>與秦共攻楚，<span class="inlay1"><span class="number">二</span></span>敗楚將屈丐，斬首八萬於丹陽。<span class="inlay1"><span class="number">三</span></span>是歲，宣惠王卒，太子倉立，是為襄王。<span class="inlay1"><span class="number">四</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「周王赧之三年也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「圍景痤也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：故楚都，在今均州。　《正義》：《左傳》\〔釋〕例云：「楚居丹陽，今枝江縣故城是也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「一云周赧王六年，韓襄哀王三年，張儀死。赧王九年，襄哀王六年，秦昭王立。」</span></p><p><br/>　　襄王四年，與秦武王會臨晉。其秋，秦使甘茂攻我宜陽。五年，秦拔我宜陽，<span class="inlay1"><span class="number">一</span></span>斬首六萬。秦武王卒。六年，秦復與我武遂。九年，秦復取我武遂。十年，太子嬰朝秦而歸。<span class="inlay1"><span class="number">二</span></span>十一年，秦伐我，取穰。<span class="inlay1"><span class="number">三</span></span>與秦伐楚，敗楚將唐眛。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：《括地志》云：「故韓城一名宜陽城，在洛州福昌縣東十四里，韓宜陽城也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「與秦會臨晉，因至咸陽而還。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：穰，人羊反，鄧州縣也。郭仲產南雍州記云：「楚之別邑。秦初侵楚，封公子悝為穰侯。後屬韓，秦昭王取之也。」</span></p><p>　　十二年，太子嬰死。公子咎、公子蟣蝨爭為太子。時蟣蝨質於楚。蘇代謂韓咎曰：「蟣蝨亡在楚，楚王欲內之甚。今楚兵十餘萬在方城之外，<span class="inlay1"><span class="number">一</span></span>公何不令楚王築萬室之都雍氏之旁，<span class="inlay1"><span class="number">二</span></span>韓必起兵以救之，公必將矣。公因以韓楚之兵奉蟣蝨而內之，其聽公必矣，必以楚韓封公也。」韓咎從其計。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：方城，楚之北境。之外，北境之北也。　《正義》：《括地志》云：「方城山在許州葉縣西南十八里。《左傳》云楚大夫屈完對齊侯曰『楚國方城以為城』，杜注云『方城山在南陽葉縣南』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「在陽翟。」《正義》：《括地志》云：「故雍氏城在洛州陽翟縣二十五里。故老云黃帝臣雍父作杵臼也。」</span></p><p>　　楚圍雍氏，<span class="inlay1"><span class="number">一</span></span>韓求救於秦。秦未為發，使公孫昧入韓。公仲曰：「子以秦為且救韓乎？」對曰：「秦王之言曰『請道南鄭、藍田，<span class="inlay1"><span class="number">二</span></span>出兵於楚以待公』，殆不合矣。」<span class="inlay1"><span class="number">三</span></span>公仲曰：「子以為果乎？」對曰：「秦王必祖張儀之故智。<span class="inlay1"><span class="number">四</span></span>楚威王攻梁也，張儀謂秦王曰：『與楚攻魏，魏折而入於楚，韓固其與國也，是秦孤也。不如出兵以到之，<span class="inlay1"><span class="number">五</span></span>魏楚大戰，秦取西河之外以歸。』今其狀陽言與韓，其實陰善楚。公待秦而到，必輕與楚戰。楚陰得秦之不用也，必易與公相支也。<span class="inlay1"><span class="number">六</span></span>公戰而勝楚，遂與公乘楚，施三川而歸。<span class="inlay1"><span class="number">七</span></span>公戰不勝楚，楚塞三川守之，<span class="inlay1"><span class="number">八</span></span>公不能救也。竊為公患之。司馬庚<span class="inlay1"><span class="number">九</span></span>三反於郢，甘茂與昭魚<span class="inlay1"><span class="number">一〇</span></span>遇於商於，其言收璽，<span class="inlay1"><span class="number">一一</span></span>實類有約也。」公仲恐，曰：「然則柰何？」曰：「公必先韓而後秦，先身而後張儀。<span class="inlay1"><span class="number">一二</span></span>公不如亟以國合於齊楚，齊楚必委國於公。公之所惡者張儀也，<span class="inlay1"><span class="number">一三</span></span>其實猶不無秦也。」於是楚解雍氏圍。<span class="inlay1"><span class="number">一四</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「《秦本紀》惠王後元十三年，周赧王三年，楚懷王十七年，齊湣王十二年，皆云『楚圍雍氏』。《紀年》於此亦說『楚景翠圍雍氏。韓宣王卒，秦助韓共敗楚屈丐』。又云『齊、宋圍煮棗』。皆與《史記》年表及《田完世家》符同。然則此卷所云『襄王十二年，韓咎從其計』以上，是楚後圍雍氏，赧王之十五年事也。又說『楚圍雍氏』以下，是楚前圍雍氏，赧王之三年事。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：南鄭，梁州縣。藍田，雍州縣。秦王言或出雍州西南至鄭，或出雍州東南歷藍田出嶢關，俱繞楚北境以待韓使而東救雍氏。如此遲緩，近不合於楚矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：殆不合於南鄭。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「祖者，宗之習之謂也。故智，猶前時謀計也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：到，欺也，猶俗云「張到」。然《戰國策》作「勁」，勁，強也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：言楚陰知秦，不為公用，亦必易為公相支拒也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《正義》：施猶設也。三川，周天子都也。言韓戰勝楚，則秦與韓駕御於楚，卽於天子之都，張設救韓之功，行霸王之迹，加威諸侯，乃歸咸陽是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《正義》：楚乃塞南河四關守之，韓不能救三川。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">九</span></span>《集解》：徐廣曰：「一作『唐』。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一〇</span></span>《集解》：徐廣曰：「楚相國。」　《索隱》：《戰國策》謂之昭䱷。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一一</span></span>劉氏云「詐言昭魚來秦，欲得秦官之印璽」。收卽取之義也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一二</span></span>《正義》：先以身存韓之計，而後知張儀為秦到魏之計，不如急以國合於齊楚。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一三</span></span>《正義》：惡，烏故反。公孫昧言公仲所惡者張儀到魏之計，雖以國合於齊楚，其實猶不輕欺無秦也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一四</span></span>《集解》：徐廣曰：「甘茂傳云『楚懷王以兵圍韓雍氏，韓使公仲告急於秦，秦昭王新立，不肯救。甘茂為韓言之，乃下師於殽以救韓也』。又云『周赧王十五年，韓襄王十二年，秦擊楚，斬首二萬，敗楚襄城，殺景缺』。《周本紀》赧王八年之後云『楚圍雍氏』，此當韓襄王十二年，魏哀王十九年。《紀年》於此亦說『楚入雍氏，楚人敗』。然爾時張儀已死十年矣。」　《正義》：自此已上十二年，並是楚後圍雍氏，赧王之十五年一段事也。前注徐廣云「『楚圍雍氏』之下，是楚前圍雍氏，赧王三年事」，徐說非也。徐見下文云「先身而後張儀」及「公之所惡者張儀也」，言張儀尚存，楚又兩度圍雍氏，故生此前後之見，甚誤也。然是公孫昧卻述張儀時事，說韓相公仲耳。</span></p><p>　　蘇代又謂秦太后弟羋戎<span class="inlay1"><span class="number">一</span></span>曰：「公叔伯嬰恐秦楚之內蟣蝨也，<span class="inlay1"><span class="number">二</span></span>公何不為韓求質子於楚？<span class="inlay1"><span class="number">三</span></span>楚王聽入質子於韓，<span class="inlay1"><span class="number">四</span></span>則公叔伯嬰知秦楚之不以蟣蝨為事，必以韓合於秦楚。秦楚挾韓以窘魏，魏氏不敢合於齊，是齊孤也。公又為秦求質子於楚，<span class="inlay1"><span class="number">五</span></span>楚不聽，怨結於韓。韓挾齊魏以圍楚，楚必重公。<span class="inlay1"><span class="number">六</span></span>公挾秦楚之重以積德於韓，公叔伯嬰必以國待公。」於是蟣蝨竟不得歸韓。<span class="inlay1"><span class="number">七</span></span>韓立咎為太子。齊、魏王來。<span class="inlay1"><span class="number">八</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：徐廣曰：「號新城君。」　《索隱》：羋，姓；戎，名。秦宣太后弟，號新城君。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：按《戰國策》，公叔伯嬰與蟣蝨及公子咎並是襄王子。然伯嬰卽太子嬰，嬰前死，故咎與蟣蝨又爭立。此取《戰國策》說，伯嬰未立之先亦與蟣蝨爭立，故事重而文倒也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《索隱》：令韓求楚，更以別人為質，以替蟣蝨也。　《正義》：為，于偽反。後同。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：質子，蟣蝨也。　《正義》：質子，蟣蝨。蘇代令羋戎為韓求蟣蝨入於韓，楚不聽。公叔伯嬰知秦楚不以蟣蝨為事，必以韓合於秦楚。「楚王聽入質子於韓」當云「楚王不聽入質子於韓」，承前脫「不」字耳。次下云「知秦楚不以蟣蝨為事」，重明脫「不」字。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：令羋戎敎秦，於楚索韓所送質子，令入之於秦也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《正義》：言韓合齊魏以圍楚，楚必尊重羋戎以求秦救矣。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《正義》：自此已前蘇代數計皆不成，故韓竟立咎為太子也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">八</span></span>《正義》：蘇代為韓立計，故得齊、魏王來。</span></p><p>　　十四年，與齊、魏王共擊秦，至函谷而軍焉。十六年，秦與我河外及武遂。襄王卒，太子咎立，是為釐王。</p><p><br/>　　釐王三年，使公孫喜率周、魏攻秦。秦敗我二十四萬，虜喜伊闕。五年，秦拔我宛。<span class="inlay1"><span class="number">一</span></span>六年，與秦武遂地二百里。<span class="inlay1"><span class="number">二</span></span>十年，秦敗我師于夏山。十二年，與秦昭王會西周而佐秦攻齊。齊敗，湣王出亡。十四年，與秦會兩周閒。二十一年，使暴䳒<span class="inlay1"><span class="number">三</span></span>救魏，為秦所敗，䳒走開封。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：宛，於元反。宛，鄧州縣也，時屬韓也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：此武遂及上武遂皆宜陽近地。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：音捐。韓將姓名。</span></p><p>　　二十三年，趙、魏攻我華陽。<span class="inlay1"><span class="number">一</span></span>韓告急於秦，秦不救。韓相國謂陳筮<span class="inlay1"><span class="number">二</span></span>曰：「事急，願公雖病，為一宿之行。」陳筮見穰侯。穰侯曰：「事急乎？故使公來。」陳筮曰：「未急也。」穰侯怒曰：「是可以為公之主使乎？夫冠蓋相望，告敝邑甚急，公來言未急，何也？」陳筮曰：「彼韓急則將變而佗從，以未急，故復來耳。」穰侯曰：「公無見王，請今發兵救韓。」八日而至，敗趙、魏於華陽之下。是歲，釐王卒，子桓惠王立。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：司馬彪云：「華陽，山名，在密縣。」鄭州管城縣南四十里。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「一作『筌』。」　《索隱》：徐廣云一作「荃」。《戰國策》作「田荼」。</span></p><p><br/>　　桓惠王元年，伐燕。九年，秦拔我陘，城汾旁。<span class="inlay1"><span class="number">一</span></span>十年，秦擊我於太行，<span class="inlay1"><span class="number">二</span></span>我上黨郡守以上黨郡降趙。十四年，秦拔趙上黨，<span class="inlay1"><span class="number">三</span></span>殺馬服子卒四十餘萬於長平。十七年，秦拔我陽城、負黍。<span class="inlay1"><span class="number">四</span></span>二十二年，秦昭王卒。二十四年，秦拔我城皋、滎陽。二十六年，秦悉拔我上黨。二十九年，秦拔我十三城。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：陘音刑。秦拔陘城於汾水之旁。陘故城在絳州曲沃縣西北二十里汾水之旁也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《正義》：太行山在懷州河內縣北二十五里也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《正義》：韓上黨也。從太行山西北澤、潞等州是也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：徐廣曰：「負黍在陽城。」　《正義》：古今地名云：「負黍在洛州陽城西三十七里也。」</span></p><p>　　三十四年，桓惠王卒，子王安立。</p><p><br/>　　王安五年，秦攻韓，韓急，使韓非使秦，秦留非，因殺之。</p><p>　　九年，秦虜王安，盡入其地，為潁州郡。韓遂亡。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：亡在秦始皇帝十七年。</span></p><p><br/>　　太史公曰：韓厥之感晉景公，紹趙孤之子武，以成程嬰、公孫杵臼之義，此天下之陰德也。韓氏之功，於晉未睹其大者也。然與趙、魏終為諸侯十餘世，宜乎哉！</p><p><span class="inlay2">　【《索隱》述贊】：韓氏之先，實宗周武。事微國小，《春秋》無語。後裔事晉，韓原是處。趙孤克立，智伯可取。旣徙平陽，又侵負黍。景趙俱侯，惠（文）\〔又〕僭主。秦敗脩魚，魏會區鼠。韓非雖使，不禁狼虎。</span></p>
