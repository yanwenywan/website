丨ˋ。找對主詞+添字還原：猶下文「自免」什麼？免職也。此則「遂」什麼？遂意也。上下文、前後文+對錯問題,回到二作。</span>欲自免歸。中郎將袁盎知其賢，惜其去，乃請徙釋之補謁者。<span class="inlay1"><span class="number">三</span>孫守真按：上下文、前後文：前用「調」後用「徙」，抽換字面爾。</span>釋之旣朝畢，因前言便宜事。<span class="inlay1"><span class="number">一</span></span>《索隱》：韋昭堵音赭，又音如字，地名，屬南陽。《正義》：應劭曰：「哀帝改為順陽，水東南入蔡。」《括地志》云：「順陽故城在鄧州穰縣西三十里，楚之郇邑也。及《蘇秦傳》云『楚北有郇陽』，並謂此也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：蘇林曰：「顧錢若出穀也。」如淳曰：「漢儀注訾五百萬得為常侍郎。」《索隱》：訾音子移反。字苑云「貲，積財也」。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：案：卑，下也。欲令且卑下其志，無甚高談論，但令依時事，無說古遠也。</span></p><p>　　釋之從行，登虎圈。<span class="inlay1"><span class="number">一</span>孫守真按：行在之行，單字想複詞。先抓動詞·找對詞性：從。隨從，同義複詞。</span>上問上林尉<span class="inlay1"><span class="number">二</span></span>諸禽獸簿，十餘問，尉左右視，盡不能對。虎圈嗇夫<span class="inlay1"><span class="number">三</span></span>從旁代尉對上所問禽獸簿甚悉，欲以觀其能口對響應無窮者。<span class="inlay1">孫守真按：觀ㄍㄨㄢ，顯示、示範。找對詞性+添字還原：如響之應聲。先抓動詞+找對主詞+添字還原：文帝欲以觀。</span>文帝曰：「吏不當若是邪？尉無賴！」<span class="inlay1"><span class="number">四</span>孫守真按：無賴，對錯問題,回到二作，此猶云沒本事、無能耐。謂賴以為官之能力也。</span>乃詔釋之拜嗇夫為上林令。釋之久之前曰：「陛下以絳侯周勃何如人也？」上曰：「長者也。」又復問：「東陽侯張相如何如人也？」上復曰：「長者。」釋之曰：「夫絳侯、東陽侯稱為長者，此兩人言事曾不能出口，豈斅此嗇夫諜諜<span class="inlay1"><span class="inlay1">孫守真按：陵遲=陵替，一聲之轉·字形結構兼音義。</span>天下土崩。今陛下以嗇夫口辯而超遷之，臣恐天下隨風靡靡，爭為口辯而無其實。且下之化上疾於景響，<span class="inlay1">孫守真按：對錯問題,回到二作+添字還原：且下之化（于）上。</span>舉錯不可不審也。」文帝曰：「善。」乃止不拜嗇夫。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《正義》：求遠反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：張晏曰：「才無可恃。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：晉灼曰：「音牒。」《索隱》：音牒。《漢書》作「喋喋」，口多言。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《索隱》：案：謂空具其文而無其實也。</span></p><p>　　上就車，召釋之參乘，徐行，問釋之秦之敝。具以質言。<span class="inlay1"><span class="number">一</span></span>至宮，上拜釋之為公交車令。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：如淳曰：「質，誠也。」</span></p><p><br/>　　頃之，太子與梁王共車入朝，不下司馬門，<span class="inlay1"><span class="number">一</span></span>於是釋之追止太子、梁王無得入殿門。遂劾不下公門不敬，奏之。薄太后聞之，文帝免冠謝曰：「敎兒子不謹。」薄太后乃使使承詔赦太子、梁王，然后得入。文帝由是奇釋之，拜為中大夫。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：如淳曰：「宮衞令『諸出入殿門公交車司馬門，乘軺傳者皆下，不如令，罰金四兩』。」</span></p><p><br/>　　頃之，至中郎將。從行至霸陵，居北臨廁。<span class="inlay1"><span class="number">一</span></span>是時愼夫人從，上指示愼夫人新豐道，曰：「此走邯鄲道也。」<span class="inlay1"><span class="number">二</span>孫守真按：走向，同義複詞。</span>使愼夫人鼓瑟，上自倚瑟而歌，<span class="inlay1"><span class="number">三</span></span>意慘淒悲懷，顧謂羣臣曰：「嗟乎！以北山石為椁，<span class="inlay1"><span class="number">四</span></span>用紵絮<span class="inlay1"><span class="number">五</span></span>斮陳，<span class="inlay1">孫守真按：斮=錯，字形結構換部首，猶斧ㄈㄨˇ=鈇ㄈㄨ，一从「斤」一从「金」構字耳。</span>蕠漆其閒，<span class="inlay1"><span class="number">六</span></span>豈可動哉！」左右皆曰：「善。」釋之前進曰：<span class="inlay1">孫守真按：前進：斷句+單字想複詞：向前進諫。</span>「使其中有可欲者，雖錮南山猶有郄；<span class="inlay1"><span class="number">七</span></span>使其中無可欲者，雖無石椁，又何戚焉！」文帝稱善。其後拜釋之為廷尉。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：李奇曰：「霸陵北頭廁近霸水，帝登其上，以遠望也。」如淳曰：「居高臨垂邊曰廁也。」蘇林曰：「廁，邊側也。」韋昭曰：「高岸夾水為廁也。」《索隱》：劉氏廁音初吏反。按：李奇曰「霸陵北頭廁近霸水」。蘇林曰「廁，邊側也」。包愷音側，義亦兩通也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：張晏曰：「愼夫人，邯鄲人也。」如淳曰：「走音奏，趨也。」《索隱》：音奏。案：走猶向也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：《漢書音義》曰：「聲氣依倚瑟也。《書》曰：『聲依永』。」《索隱》：倚，於綺反。案：謂歌聲合於瑟聲，相依倚也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《正義》：顏師古云：「美石出京師北山，今宜州石是。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《索隱》：上張呂反，下息慮反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">六</span></span>《集解》：徐廣曰：「斮，一作『錯』。」駰案：《漢書音義》曰「斮絮，以漆著其閒也」。《索隱》：斮陳絮漆其閒。斮音側略反。絮音女居反。案：斮陳絮以漆著其閒也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">七</span></span>《集解》：張晏曰：「錮，鑄也。帝北向，故云『北山』；回顧南向，故云『南山』。」《索隱》：案：張晏云「錮，鑄也。帝北向，故云『北山』；回顧向南，故云『南山』」。今案：大顏云「北山靑石肌理密，堪為碑槨，至今猶然。故《秦本紀》作阿房或作酈山石槨是也」。故帝欲北山之石為槨，取其精牢。釋之答言，但使薄葬，冢中無可貪，雖無石槨，有何憂焉。若使厚殉，冢中有物，雖並錮南山，猶為人所發掘也。言「南山」者，取其高厚之意，張晏殊失其旨也。</span></p><p><br/>　　頃之，上行出中渭橋，<span class="inlay1"><span class="number">一</span></span>有一人從穚下走出，乘輿馬驚。於是使騎捕，屬之廷尉。釋之治問。曰：「縣人來，<span class="inlay1"><span class="number">二</span></span>聞蹕，匿橋下。久之，以為行已過，<span class="inlay1">孫守真按：行ㄒ丨ㄥˊ，行在之行。單字想複詞。</span>卽出，見乘輿車騎，卽走耳。」廷尉奏當，一人犯蹕，當罰金。<span class="inlay1"><span class="number">三</span></span>文帝怒曰：「此人親驚吾馬，吾馬賴柔和，令他馬，固不敗傷我乎？而廷尉乃當之罰金！」釋之曰：「法者天子所與天下公共也。<span class="inlay1"><span class="number">四</span></span>今法如此而更重之，是法不信於民也。且方其時，上使立誅之則已。今旣下廷尉，廷尉，天下之平也，一傾而天下用法皆為輕重，民安所措其手足？唯陛下察之。」良久，上曰：「廷尉當是也。」</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《集解》：張晏曰：「在渭橋中路。」瓚曰：「中渭橋兩岸之中。」《索隱》：張晏、臣瓚之說皆非也。案今渭橋有三所：一所在城西北咸陽路，曰西渭橋；一所在東北高陵道，曰東渭橋；其中渭橋在古城之北也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：如淳曰：「長安縣人。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：如淳曰：「乙令『蹕先至而犯者罰金四兩』。蹕，止行人。」《索隱》：案：崔浩云「當謂處其罪也」。案：《百官志》云「廷尉平刑罰，奏當所應。郡國讞疑罪，皆處當以報之」也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《索隱》：小顏云：「公謂不私也。」</span></p><p>　　其後有人盜高廟坐前玉環，捕得，文帝怒，下廷尉治。釋之案律盜宗廟服御物者為奏，奏當弃市。上大怒曰：「人之無道，乃盜先帝廟器，吾屬廷尉者，欲致之族，而君以法奏之，<span class="inlay1"><span class="number">一</span></span>非吾所以共承宗廟意也。」釋之免冠頓首謝曰：「法如是足也。<span class="inlay1"><span class="number">二</span></span>且罪等<span class="inlay1"><span class="inlay1"><span class="number">一</span></span>《索隱》：案：法者，依律以斷也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《集解》：徐廣曰：「足，一作『止』也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">三</span></span>《集解》：如淳曰：「俱死罪也，盜玉環不若盜長陵土之逆也。」</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">四</span></span>《集解》：張晏曰：「不欲指言，故以取土譬也。」《索隱》：抔音步侯反。案：《禮運》云「污尊而抔飲」，鄭氏云「抔，手掬之，字從手」。字本或作「杯」，言一勺一杯，兩音並通。又音普回反。坯者，磚之未燒之名也。張晏云「不欲指言，故以取土譬」者，蓋不欲言盜開長陵及說傷迫近先帝故也。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">五</span></span>《集解》：徐廣曰：「一作『閒』。《漢書》作『啟』。啟者，景帝諱也，故或為『開』。」</span></p><p><br/>　　後文帝崩，景帝立，釋之恐，<span class="inlay1"><span class="number">一</span></span>稱病。欲免去，懼大誅至；欲見謝，<span class="inlay1">孫守真按：先抓動詞+找對主詞：見什麼、謝什麼？見帝而謝罪也。</span>則未知何如。用王生計，卒見謝，景帝不過也。</p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：謂帝為太子時，與梁王入朝，不下司馬門，釋之曾劾，故恐也。</span></p><p>　　王生者，善為黃老言，處士也。嘗召居廷中，三公九卿盡會立，王生老人，曰「吾韤解」，<span class="inlay1"><span class="number">一</span></span>顧謂張廷尉：「為我結韤！」<span class="inlay1"><span class="number">一</span></span>《正義》：上萬越反，下閒買反。</span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">二</span></span>《索隱》：結音如字，又音計。</span></p><p>　　張廷尉事景帝歲餘，為淮南王相，猶尚以前過也。<span class="inlay1">孫守真按：斷句：猶尚，同義複詞。以，以為；前，昔；過錯。仍前與景帝故。</span>久之，釋之卒。其子曰張摯，字長公，官至大夫，免。以不能取容當世，故終身不仕。<span class="inlay1"><span class="number">一</span></span></p><p><span class="inlay2">　<span class="inlay1"><span class="number">一</span></span>《索隱》：謂性公直，不能曲屈見容於當世，故至免官不仕也。</span></p><p><br/><br/>　　馮唐者，其大父趙人。父徙代。漢興徙安陵。唐以孝著，為中郎署長，<span class="inlay1"><span class="number">一</span></span>事文帝。文帝輦過，<span
